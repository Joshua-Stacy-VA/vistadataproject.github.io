description
This CLASS 1 file id designed to provide the system with a controlled list of CHCS Data Elements that can used withe rules processing capability. Data elements such as patient age, current test results, or previous test results, are exposed in this file. Each data element in this file can be evaluated within the context of a rules' IF CONDITION and based on the condition, further action can be specified in the rules' THEN portion.
