description
This file defines the Outpatient MEPRS Billing Rates used for the DD7A Outpatient billing. Each rate table entry includes the International Military Education and Training (IMET) Rate, the Interagency and Other Federal Agency Sponsored Patient (IOR) rate, and the Other rate category currently used for the Full Reimbursable Rate (FOR). Each Inpatient Division will have an Outpatient MEPRS Billing Rate table.
