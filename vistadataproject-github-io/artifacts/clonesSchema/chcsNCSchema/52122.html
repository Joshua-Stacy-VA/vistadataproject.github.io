description
This file contains a subset of ICD0 codes and the procedure text associated with the code. The entries are a DINUMed pointer relationship to the ^ICD0(D0, file and contains all available vaccine codes that are not maintained on the MARC by name. EX: There is an ICD0 code for YELLOW FEVER although it will not be included in this file as there is a specific field on the MARC for this immunization type. This file is needed due to the size of the ^ICD0( file and the length of time it takes to return a usable, filtered picklist.)
