description
This file contains types of blood products currently available from the Defense Blood Standard System. The DMIM organization is responsible for defining the blood product type data and maintaining that data definition. The site manager and lab supervisors may edit the SYNONYM, DBA INACTIVE, and DEACTIVATION DATE fields. The CODE, DESCRIPTION, and AUTOLOGOUS DONATION fields are uneditable. New codes may not be added and existing codes may not be deleted.
