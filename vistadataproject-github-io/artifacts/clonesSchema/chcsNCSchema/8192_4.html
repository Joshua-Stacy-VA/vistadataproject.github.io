.01carrier coverage type code0;1POINTER8096BThe Coverage Type Code is used to define the scope of coverage, both at the Health Insurance Carrier (HIC) level, and at the individual OHI policy level. Each Health Insurance Carrier record must have, at least one Coverage Type Code. Each OHI policy must have, at least, one Coverage Type Code. Examples of Coverage Type Codes are: MD (Medical) RX (Pharmacy), DN (Dental), etc. The system will not allow an OHI policy record to include a Coverage Type unless that Coverage Type is an attribute of the Health Insurance Carrier associated with the policy.
.02coverage payer type code0;2SET OF CODESB:BOTH INSTITUTIONAL AND PROFESSIONAL
I:INSTITUTIONAL ONLY
P:PROFESSIONAL ONLY
N:NON-BILLABLE
CThe Carrier Payer Type, in combination with the Coverage Type Code, provides the information necessary to direct bills to the appropriate address. Claims will be directed as follows: If a Coverage Type has a "B" Payer Type, that means that both Institutional and Professional health care services are billed to a single address. If a Coverage Type has an "I" Payer Type, that means that only health care services defined as "institutional" services will be billed to the associated address. If a Coverage Type has a "P" Payer Type, that means that only health care services defined as "professional" services will be billed to the associated address.
.03coverage status code0;3POINTER8095Code that defines the status of a Coverage Type in terms of where the Coverage Type stands in the verification process. Newly entered Coverage Types will always have a Coverage Status of "Temporary" until they are verified by the Verification Point of Contact (VPOC) for the DEERS HIC database. The Coverage Status Codes are as follows: "Standard", "Temporary", "Deactivated", "Placeholder", "Cancelled", "Rejected."
