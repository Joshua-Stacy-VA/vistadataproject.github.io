description
This file contains all the functions related to the Desktop, and other Clinical Interface functionality. It is linked to the Action file, in that certain actions require the user to select an entry from the file to complete the action. This file contains MUMPS code that controls program flow. Each function has help text associated with it, that can be reviewed from within the Desktop via the Help action. ***** THIS FILE SHOULD ONLY BE MODIFIED BY AUTHORIZED PERSONNEL *****
