description
This file contains the Other Health Insurance (OHI) profile for each enrolled patient who has OHI policies. The file is organized by "Patient IEN." Each patient's record includes the OHI policies under which the patient is insured. The minimum data set for each policy includes the following elements: the "Carrier Identifier" which represents the Insurance Carrier, the "Policy Identifier." and the "Policy Effective Date." Policyholder information may also be included, as well as Group Policy information, when applicable. The scope of coverage and claims filing information are defined in a sub-file, by the Coverage Type and Payer Type.
