.01code(+)0;1FREE TEXTBThe Coverage Type Code is used to define the scope of coverage, both at the Health Insurance Carrier (HIC) level, and at the individual OHI policy level. Each Health Insurance Carrier record must have, at least one Coverage Type Code. Each OHI policy must have, at least, one Coverage Type Code. Examples of Coverage Type Codes are: MD (Medical) RX (Pharmacy), DN (Dental), etc. The system will not allow an OHI policy record to include a Coverage Type unless that Coverage Type is an attribute of the Health Insurance Carrier associated with the policy.
