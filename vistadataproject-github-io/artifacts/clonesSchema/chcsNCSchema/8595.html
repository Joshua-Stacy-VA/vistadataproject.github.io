description
This file is used for storing historical records of SIDR Outpatient Data transmissions by month/year. It has a DINUMED month/year .01 field. The file contains all of patient/info sent on the last transmission run. NOTE: It is possible to do a retransmit, but the OLD entries will be deleted for the (re-transmit) month selected and all current entries in the "ASADR" xref from the patient appointment file will be rebuilt into the new transmission for the selected month.
