description
Upon selection of an order to print the future order, this file will be stuffed and the batch number will be passed to a taskman job and the taskman will create or update an entry in the Pharmacy file and will print a label. The entries in this file will be purged in a specified time (two weeks), regardless of whether the order is activated or not. the BATCH^PSSFSTUF routine will be used to stuff the entries in the batch file and this ^PSM(8215,"*","ID") will be used to keep track of the last batch number.
