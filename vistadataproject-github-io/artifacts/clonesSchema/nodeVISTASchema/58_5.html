This file contains the data necessary to generate the AMIS statistics for AR/WS. This data is accumulated automatically by a queued nightly job. *** NOTE *** There are two cross-references that exist under this file that are created in the Pharmacy AOU Stock file (#58.1). The xref names are "AMIS" and "AMISERR", if you create any local xrefs for this file (#58.5) DO NOT use these names as it will overwrite the existing xrefs.
