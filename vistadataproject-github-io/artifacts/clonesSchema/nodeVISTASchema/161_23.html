description
In order to keep track of rate changes for an authorization in the CNH program, this file was developed. It stores the rate that is paid for an authorization time frame. When new contracts are negotiated for a CNH new entries must be created in this file. Per VHA Directive 10-93-142, this file definition should not be modified.
