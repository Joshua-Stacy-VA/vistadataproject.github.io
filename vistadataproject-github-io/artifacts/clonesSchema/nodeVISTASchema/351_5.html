description
This file is used to store data related to each Pharmacy billing transaction with the Tricare fiscal intermediary. Each transaction is submitted to the the RNA/Triad Pharmacy ClaimGuard System, which is a commercial point of sale pharmacy billing software package, where it is forwarded to the intermediary through an electronic switch company. All of the information which is returned from the intermediary is stored in this file.
