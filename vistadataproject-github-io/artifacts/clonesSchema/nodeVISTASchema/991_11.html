description
This file holds the CIRN HL7 exception types. It is pointed to by the TYPE field (#2) for the EXCEPTION subfield (#991.12) of the CIRN HL7 EXCEPTION LOG file (#991.1).
Fields
#NameLocationTypeDetailsIndexDescription
.01number(+)0;1NUMERICBThis is the index number of this exception.
1severity0;2SET OF CODES0:INFORMATIONAL
1:MINOR
2:SEVERE
This is the severity level of this exception, '0' for Informational, '1' for Minor or '2' for Severe.
2action0;3SET OF CODES0:NONE
1:MAIL
2:ALERT
3:MPI EXCEPTION HANDLER
This is the action to be taken upon logging this exception, '0' for None, '1' for Mail, '2' for Alert, or '3' for MPI Exception Handler. Exceptions whose action is MPI Exception Handler go to the exception handler located at the Austin MPI.
3count0;5NUMERICThis is a count of all exceptions logged of this type.

10text10;E1,100FREE TEXTCThis is a free text field (up to 100 characters) which holds the message for this exception.
20security key20;0MULTIPLE991.112This is used to control access to a particular exception type. A user must possess at least one of the listed security keys to access the associated exception. If no security keys are listed, access is unrestricted.
