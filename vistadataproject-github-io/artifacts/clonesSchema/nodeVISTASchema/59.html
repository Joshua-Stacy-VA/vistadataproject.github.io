59

description
This file holds the set of parameters which modify the operation of the outpatient pharmacy package to suit the needs of your site. The address information appears on the labels. Most of the other fields hold parameters which enable or disable a particular feature of the package. These parameters are loaded into a local variable when entering the package. Any changes made to these parameters will not be effective until the users exit from the package and re-enters it.
