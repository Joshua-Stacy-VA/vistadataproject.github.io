5accession0;5SET OF CODESL:Long Accession Number
S:Short Accession Number
The value of this field is a code that specifies whether the equipment described in the current entry uses 'short' or 'long' accession numbers. A complete accession number consists of two groups of digits separated by a dash. The first group is 6 digits, and is derived from the date. The second group is an integer number that uniquely identifies the study for that date. When the value of this field stars with an 'L', 'long' accession numbers are assumed (both parts are to be provided). When the value of this field starts with an 'S', 'short' accession numbers are assumed (only the second part is to be provided).
6description0;6FREE TEXTThe value of this field is a description that identifies the equipment that is specified by this entry. Such a description typically includes the name of a building or the number of a room.
