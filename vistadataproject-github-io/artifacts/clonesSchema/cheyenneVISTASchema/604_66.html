description
Data and executable code for administering the Addiction Severity Index. Branching, defaults, screen reminders and question wording are here. This is a static file and local modifications are not permitted.

This field contains a value if the entry is required for scoring the Addiction Severity Index.
12id0;11FREE TEXTThe ID field in the ASI QUICK file is the shortest unique humanly readable text to identify each question and sometimes sub-question on the ASI.It is used in the full screen administration mode at the top of each question, as a terse listing of missing required fields and used to determine the section a question belongs to. These IDs match the question ID on the printed ASI forms. An ID is the combination of the section letter ie M for medical, L for legal and the question number for that section eg E2, L17. Sometimes a question has multiple parts and this is reflected in the ID eg E6 Yrs, E6 Mons.
