description
This Inpatient Evaluation Center file contains the study parameters for upcoming IPEC data acquisition rounds through FY 2011. This uneditable file resides both in the central IPEC VISTA database and in the database of each remote study site. Routines called by server options will first ascertain the study identifier passed in the server request message, then look up the needed parameters in this file, and then set variables that will be used by the IPEC data acquisition software in identifing the inpatient episodes of care for which data will be required. cvamc-st;3.2.2007 --------------------------------------------------------------- Updated: 9.5.2007 --------------------------------------------------------------- Inpatient Evaluation Center (IPEC) - VISN 10 VAMC, Cincinnati, OH - Director: Marta L. Render, M.D. ---------------------------------------------------------------
