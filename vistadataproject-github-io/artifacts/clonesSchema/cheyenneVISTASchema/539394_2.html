description
The ICU Outcome Improvement Project relies heavily on this table of study windows in the data acquisition and files population processes. Files reliant on this table for their formation include IPEC LABORATORY RESULTS File (#539394.2), IPEC COMORBIDITY File (#539394.6), and the IPEC OUTCOMES File (#539394.9). For the purpose of the project, EVERY critical care episode encountered within each admission occurring during the study period is selected for scrutiny. Selected laboratory values, in particular, are grouped and evaluated as they occur within the defined windows. The window formation starts with each Critical Care Entry Date/Time. The first window spans the the time width of 192 hrs (8 days), starting exactly 168 hours (7 days) prior to unit entry and extending to 24 hours (1 day) following unit entry. Subsequent days of critical care are tracked in contiguous 24 hours windows. The Unit Exit Date is included so that one last, overlapping 24 hours window is included in the model. * Window Model * ========================================================================= Unit 1st 2nd 3rd Unit Entry Unit Unit Unit Exit D/T Window Window Win D/T V V V V V {-------- 192h -------}*{--- 24h ----}*{-- 24h --}*{--- 24h --} ========================================================================= NOTE: The exceptionally wide first window is an accommodation for the study of critical care occurrences involving surgical procedures occurring in the operating room setting. All other non-OR cases will use a survey window 48 hours in width, 24 hours before and after unit entry during the home site processing of data.
