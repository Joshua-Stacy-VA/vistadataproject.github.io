description
This file contains a list of expressions (pointers) from the expression file #757.01 and shortcuts by context. The expression(s) pointed to will be returned immediately if any of the context sensitive shortcuts associated with that expression are detected (exact match) in the users input. The context must be a default value stored in the user defaults in the Subset Definition file #757.2 to permit the use of the shortcuts.
