description
This is the holding file for the daily control point totals sent from all sites in Region 1 for fund 0167BO. Data is gathered nightly via a queued option and is sent to this file via Mailman and server option ARMK CONTROL POINT COLLECTOR. Sites should not modify the data via fileman.
