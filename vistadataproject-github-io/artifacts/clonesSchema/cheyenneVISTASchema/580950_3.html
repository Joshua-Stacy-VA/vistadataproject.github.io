description
This file is somewhat like the AJK1UB TRANSMITAL file except that the records contained in here are all status updates associated with an initial transmission in the AJK1UB TRANSMITAL file. As the daily processor searches the database for activity to transmit it will, as it finds status changes to initial transmissions, store those change in this file. Routines, options, etc., associated with this process are all in the AJK1UB namespace.
