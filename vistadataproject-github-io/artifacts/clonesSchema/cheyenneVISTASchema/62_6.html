.04lab collect only0;4BOOLEAN1:YES
If this accession test group is to be used for routine lab collects, enter YES in this field, otherwise leave blank.
.05display order0;5SET OF CODES0:CONTINUOUS
1:INTERNAL NUMBER
This field determines if test numbers on list are numbered "CONTINUOUS", starting at 1,2,3,... or by "INTERNAL NUMBER" when the test list is displayed. "INTERNAL NUMBER" will assign the test number on list based on the internal entry number (field #.001 NUMBER) of test in the "TEST" multiple. This will print the list with gaps if tests have been removed. It allows removal of tests without subsequent tests being renumbered on the list and tests being added at designated positions. This may assist site when modifying test lists and users have memorized existing number of tests, by maintaining existing number.
