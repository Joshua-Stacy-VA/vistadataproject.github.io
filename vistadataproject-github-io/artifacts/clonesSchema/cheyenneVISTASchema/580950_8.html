description
This file contains the codes to be sent on a status change to the collection agency. These codes are associated with accounts receivable transaction types which the facility want posted with the collection agency. For example, a bill with a transaction type of paid in part would be pointed to the AJK1UB STATUS CODE of "paid in part" and transmitted to the collection agency. For any accounts receivable transaction type which should not be posted to the collection agency this status code would be left blank. All options, routines, etc., associated with this process are in the AJK1UB namespace.
