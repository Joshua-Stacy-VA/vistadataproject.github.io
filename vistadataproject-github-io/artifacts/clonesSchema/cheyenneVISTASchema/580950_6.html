description
This file is where the format for collection transmission strings is stored. It should not be edited directly. There should be only two entries in this file. Entry 1 contains the transmission string format for new (initial) transmissions to the collection agency. Entry 2 contains the same data, except for status changes to the initial transmission. All options, routines, etc., associated with this process are in the AJK1UB namespace.
