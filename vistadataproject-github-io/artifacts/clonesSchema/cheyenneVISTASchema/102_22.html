CPRS QUERY CONSTRAINT
number
102.22
location
^ORD(102.22,
description
This file contains the list of constraints that are available when defining a query of CPRS items. Each constraint is associated with a particular editor. This is a graphical interface based editor. The constraint name is also used to build the name=value pairs that make up the query.
Fields
#NameLocationTypeDetailsIndexDescription
.01name(+)0;1FREE TEXTBThis is the name of the constraint. The graphical interface references values for the constraint using this name. The constraint is passed into the query processer as a name=value pair, also using this name.
