description
This file contains by calendar year all relative value units associated with a geographic area. Geographic area is defined with Zip Codes. This is a table file that should not be altered at the station unless instructed to do so. The relative values are set by the Health Care Finance Administration. Per VHA Directive 10-93-142, this file definition should not be modified.
