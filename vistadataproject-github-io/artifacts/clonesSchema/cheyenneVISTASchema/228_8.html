description
This file contains coding compliance rules that are evaluated as users enter ADA (CPT) codes into DRM Plus. Rules may generate errors which prevent the use of the code, or informational warnings. For example, users may not use code D2950 (Core build-up incl any pins) in the same visit, on the same tooth, as code D2951 (Tooth pin retention).
