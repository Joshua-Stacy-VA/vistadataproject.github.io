description
This file contains the standard report text that the radiologist or transcriptionist can choose from when entering text for a Radiology Exam Report. Standard text may be phrases or complete report text. At the time of Exam Report Entry, one or more of these standard text(s) may be selected and modified as needed for the specific exam report.
