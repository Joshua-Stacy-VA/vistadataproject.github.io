description
This file is used to store the Defense Medical Information System Identification Numbers for each facility. The information for this file is obtained from the government. The DMIS Information Center is the only office authorized to assign DMIS ID codes. The Master File Primary Key (Unique ID) for this file: CODE(.01)\DESCRIPTION(1)\99DMI
