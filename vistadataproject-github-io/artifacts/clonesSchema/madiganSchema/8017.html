description
This file contains the matching criteria used by the duplicate patient search to identify potential duplicates. This string of characters is used to control the level of investigation when searching for duplicates. The STANDARD search will use the string "3:111~2:11". The first portion of this string (3:111) controls the soundex comparsion. The last portion (2:11) controls the social security number comparsion. The EXTENDED search will use the string "2:111~1:11". The first portion of this string (2:111) controls the soundex comparsion. The last portion (1:11) controls the social security number comparsion.
