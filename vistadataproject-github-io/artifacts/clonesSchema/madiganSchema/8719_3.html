description
An entry must be in this file for every CHCS performing laboratory to which panel tests will be sent for testing. For each panel test sent to another laboratory, the performing laboratory test name and panel code must be in the file entry for that laboratory. Authorized users may add new entries or delete existing entries via the LTE option.
