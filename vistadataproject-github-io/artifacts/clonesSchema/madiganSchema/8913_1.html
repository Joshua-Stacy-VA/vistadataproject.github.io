description
This file contains the audit trail of UCAPERS transmissions. Only the last 500 transmitted records are retained in the file. The audit trail is turned on or off by the Activate/Deactivate Audit Trail option. The file can be purged using the Print Audit Trail Report option on the monitor screen.
