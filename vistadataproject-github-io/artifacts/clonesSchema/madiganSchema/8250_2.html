description
DACDESCTBL - RDACD0 The Drug Allergy Code Description Table uses the 2-byte Drug Allergy Code DAC*TBL as a key to provide a 50-byte description of the drug or drug family responsible for the allergy. The DACDESCTBL is associated with the HICTBL (ingredient) through the DAC*TBL value contained in the HICTBL. Each HICTBL record can contain zero or one DACDESCTBL reference. The Internal entry number (D0) is the Individual Drug Allergy Code Table (DACTBL) with leading zeros removed.
