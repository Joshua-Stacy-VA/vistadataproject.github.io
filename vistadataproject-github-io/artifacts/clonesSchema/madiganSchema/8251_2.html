DOSETBL - RDOSED0 The Dosage Form Description Table uses the 2-byte Dosage Form Code (GCDF) as a key and provides both the 10-byte dosage form and a 40-byte description of the dosage form. The DOSETBL is associated with the GCNSEQNOTBL using the GCDF. The Internal entry number (D0) is the SKEY_DOSE described below: The smart key dosage code is a three digit number (with leading zeros removed) representing the nearly 200 different dosage forms. Like all Smart Key components, the SKEY_DOSE values are the same across all other SKEY components. The SKEY_DOSE is compatible with NDDF* current dosage form code (GCDF), forming a one to one relationship, and providing downward support for existing applications. SKEY_DOSE can be used for selecting dosage forms reqardless of strength, it can be used for sorting products into unique dosage form groups, or for encoding drug formularies. The SKEY_DOSE has the added benefit in the selected ranges of values are reserved for common dosage forms. The common dosage form types using special ranges are aerosol's, capsules, and tablets. Aerosols use the range values 400-499, capsules 500-599, and tablets, 600-699. These ranges are very selective, however, certain applications may require the specification of all capsules, or all tablets, or all aerosols, and these special SKEY_DOSE ranges provide this support. If a SKEY_DOSE for an NDC is assigned zero, CHCS will assign it a numeric value by concatenating each ascii character of the GCDF field as its Numeric equivalent $A(). All NDCs using the same 2 character dosage form description will point to the same value. As First DataBank assigns a SKEY_DOSE to that dosage description CHCS will change SKEY_DOSE to reflect the assignment. *NDDF is a Trade Mark of the Hearst Corporation. Description of the FDB Smart Key. The Smart Key is a 24 numeric digit field assigned to each NDC as follows: Col Col Description MNEMONIC # of digits 1 - 2 Generic Therapeutic Class SKEY_GTC 2 3 - 6 Specific Therapeutic Class SKEY_STC 4 7 - 11 Generic Name SKEY_HICL 5 12 - 15 Drug Strength SKEY_STR 4 16 - 18 *Dosage Form SKEY_DOSE 3 19 - 20 Route of Administration SKEY_RT 2 21 - 23 Package Size SKEY_PS 3 24 - 24 Unit Dose - Unit of Use SKEY_UDUU 1 (0=Neither,1=Unit Dose 2=Unit of Use)
