description
Topography is a detailed, structured nomenclature of anatomic terms used to identify all parts of the body. There are over 8000 presupplied entries. Probably there will be little editing of this file. Since it is pointed to by a number of other files it is considered a "Day One File". This term means an entry in this file must exist here before it can be entered in fields of other files. Entries may be added or modified, but none of the suplied entries should be deleted. The file defines specific information on each site/specimen, such as SNOMED code, Sex Secific criteria, and abbreviation.
