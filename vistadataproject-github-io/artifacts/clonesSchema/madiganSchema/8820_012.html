A012AThis field will determine who has Access to the View Text. All user created text is set up as Private. All System created text (reports) are set up with System Access. If the site has limited the number of reports that a given user can send to the Desktop, then any Text marked as System Access will be counted, and possibly overwritten. The user may protect any System report sent to the Desktop by changing the Access from System to Private or Public. If a View Text is marked as Public, then other users may View (but not edit) the Text.
