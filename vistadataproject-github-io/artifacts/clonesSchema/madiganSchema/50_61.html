.01robot-rx drug0;1POINTER50BEnter a drug name, synonym, or therapeutic classification. The system returns a first picklist of entries matching your input that are already stocked in this robot. Entries in this picklist are identified with the letters "ROB" in parentheses to the right of the drug name. After selecting a drug from the picklist, you may delete the robot drug entry. If you press ENTER or F10 to reject the first picklist, the system displays a second picklist that does not include any of the drugs already stocked in the current robot device. This list of drugs is NOT identified by the letters "ROB" following the drug name. After you select a drug, you may add the drug to the ROBOT drug list. The system does not allow duplicate drug entries. If your input results in an empty first picklist, the system will only display the second picklist. If nothing matches your input, no list will be generated.
