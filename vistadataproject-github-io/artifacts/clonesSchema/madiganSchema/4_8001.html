name
FAIL-SOFT TRACKING
number
4.8001
location
^DIZ(4.8001,
description
This file was designed to track changes in the Fail-Soft Priority. When the Fail-Soft flag is changed or deleted, an entry is made to this file. All fields are uneditable for secutity. This file is used by the routine XQF. The Fail-Soft flag is stored as the first '^' $PIECE in ^%ZOSF("FAILSOFT"). The second '^' $PIECE is the date and time that the flag was set, the third is the name of the person who set the flag.
