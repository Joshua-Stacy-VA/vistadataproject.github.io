description
The PROVIDER PRE-MERGE file is used to define certain criteria before Provider Merge is run. A check of the PAS schedules and any data base structures in conflict with the Provider Merge Status File Statistics multiple is performed. A "skip pre-merge flag" is used to bypass the current screening.
