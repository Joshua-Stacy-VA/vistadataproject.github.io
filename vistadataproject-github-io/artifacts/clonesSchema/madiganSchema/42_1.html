description
This file contains a table of codes and descriptions for the various admission types possible at a Medical Treatment Facility. Only those codes used at this MTF are "activated". SOURCE: DMSSC, DATA ADMINISTRATION CONTRACT, MDA903-86-C-0215, FINAL ADDENDUM TO THE AQCESS/CHCS FUNCTIONAL DESCRIPTION FOR THE STANDARD REQUIREMENTS FOR THE INPATIENT DATA RECORD FOR TASK ORDER NO. DAT2, 12 AUGUST 1988.
