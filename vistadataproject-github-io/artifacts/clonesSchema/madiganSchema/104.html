description
This file is a utility file for storing the Actions which are possible in the Order Entry Screens. It's use is similar to the Option file in that every action record is associated with a routine reference or line of Mumps code which will be executed when the Action is chosen by the user. In the Action prompt at the bottom of the Order Entry Screens, a DIC lookup is done on this file.
