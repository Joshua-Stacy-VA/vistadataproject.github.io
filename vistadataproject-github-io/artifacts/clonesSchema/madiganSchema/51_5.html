description
This file contains Drug Routes and abbreviations which are used to create the drug name when using the PSDRUG input template. The internal reference numbers of this file are the same as those used by MEDIPHOR in coding its drugs for its Interaction/Adverse Reaction data base. The main purpose of this file is to make the DRUG file compatible with MEDIPHOR codes.
