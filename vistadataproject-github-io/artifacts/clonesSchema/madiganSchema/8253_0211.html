.01pemtxti0;1FREE TEXTPatient Education Text Ident. (Short) is a one byte Set of Codes as follows: T:Title,W:Why am I taking this drug?,H:How should I take it? S:Are there any side effects?,R:How do I store this?,D:If I should miss a Dose? This field enables the user to locate specific types of records within the monograph. For example, a user may wish to substitute Brand Name for the Generic Name of the drug which is listed in the monograph. This can be done by using the "T" character appearing in the field for TITLE records.
