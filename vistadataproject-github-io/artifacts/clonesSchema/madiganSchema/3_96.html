Files > SMTP TRANSMISSION QUEUE

name
SMTP TRANSMISSION QUEUE
number
3.96
location
^XMTP(3.96,
description
This file holds message entries waiting to be transmitted to a remote site. Once the message is successfully transmitted to a remote user, the message entry is removed from the file. This file should not be edited directly (ie, using FileMan). MailMan will add and delete entries as messages are sent and received from users at remote sites.
