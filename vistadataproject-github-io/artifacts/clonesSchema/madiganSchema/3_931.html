SMTP HARDWARE CONFIGURATION
number
3.931

^XMTP(3.931,
description
This file contains the hardware specific code for Network MailMan. It tracks which MUMPS routines to execute when the MailMan SMTP interface must read, write, open, or close a socket.
Fields
#NameLocationTypeDetailsIndexDescription
.01type(+)0;1FREE TEXTBThe name of the hardware configuration.
.02platform0;2POINTER.7The platform on which this configuration can be used (for example, DSM or MSM).
.03server timeout0;3NUMERICThis field indicates how long a server should wait to reconnect after an error has been received.
.04dynamic receivers0;4BOOLEAN1:YES
This field indicates if the SMTP receiver should remain active. If the hardware type if marked as having dynamic receivers, then it indicates that the SMTP MUMPS process that receives mail messages should shut down after the mail transfer is complete. Operating system services will automatically start MUMPS processes when more mail needs to be processed.
