# SC CHANGE HISTORY OK

This call returns a 1 if changing an exisiting date for a Historyentry is ok, or a 0 if the date cannot be changed.  Primary historyfiles are 404.52, 404.58, 404.59

--- | ---
Label | CHGDTC
Routine | [SCUTBK10](http://code.osehra.org/dox/Routine_SCUTBK10_source.html)
Return Type | SINGLE VALUE
