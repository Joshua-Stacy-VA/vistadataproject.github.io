# ORRHCR USRRPT

Returns the IEN of an entry in CPRS QUERY DEFINITION given the displaytext of the entry.  The entry must be a user-owned report.

## Properties

Property | Value
--- | ---
Label | USRRPT
MUMPS Implementation | [ORRHCR](http://code.osehra.org/dox/Routine_ORRHCR_source.html)
Return Type | SINGLE VALUE

--- | --- | --- | --- | ---
