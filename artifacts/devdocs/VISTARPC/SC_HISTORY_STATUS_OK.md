# SC HISTORY STATUS OK

This call returns a 1 if the status of a new history entry is ok, a 0if it is not.  Currently restricted to files, 404.52, 404.58, 404.59

Property | Value
--- | ---
Label | NEWSTC
Routine | [SCUTBK10](http://code.osehra.org/dox/Routine_SCUTBK10_source.html)
Return Type | SINGLE VALUE
