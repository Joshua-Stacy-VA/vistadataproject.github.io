# XUS SET VISITOR

This controlled-subscription RPC is used by the Broker SecurityEnhancement on an authenticating VistA system to obtain a BSE TOKEN for anauthenticated active user. The TOKEN is used to identify and authenticatea visiting user on a remote VistA system, which calls back to theauthenticating system to obtain the authenticated user's demographics.
