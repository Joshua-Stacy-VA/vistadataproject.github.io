# YTQ CHOICETYPE ADD

Use this API to enter a new choicetype (MH CHOICETYPE 601.751). A choicetype is a collection of choices (MH CHOICE 601.75) andtheir sequence.         input: list of choice iens in numbered sequence ex YS(1)=3,YS(2)=22        output NEW choice type number
