# YTQ COPY TEST

Copies a test to a new name. Used when design for one test is based on another. Questions, Intros, Content, Sections, Scales, Rule and Keys are all duplicated. Input: ORIGINAL: As name of test to be copied. ie 'MH Survey'        New: Name of new test ie 'MH Survey Copy'        NATIIONAL : optional if test is to be distributed nationally.                     Available only to MH developers. Output: DATA vs Error
