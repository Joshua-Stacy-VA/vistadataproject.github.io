# MAGQB QUEDEL

This procedure call deletes a single IMAGE BACKGROUND QUEUE file (#2006.03) entry.

Property | Value
--- | ---
Label | DQUE1
Routine | [MAGQBUT2](http://code.osehra.org/dox/Routine_MAGQBUT2_source.html)

Name | Type | Maximum Data Length | Required | Description
