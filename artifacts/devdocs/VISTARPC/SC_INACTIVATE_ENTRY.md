# SC INACTIVATE ENTRY

This call returns a 1 if the entry can be inactivated, or a 0 if theentry cannot.  Used primarily for history files.

Property | Value
--- | ---
Label | INACTC
Routine | [SCUTBK10](http://code.osehra.org/dox/Routine_SCUTBK10_source.html)
Return Type | SINGLE VALUE
