# YTAPI ASI SIGNER

Allows interviewer to sign ASI, transcriber to "sign off" and send abulletin. Input Required:                CODE: electronic signature code                IEN: ien of ASI in file 604 Output: reports on status of the operation only                YSDATA(1)=[DATA]                YSDATA(2)=1^ASI SIGNED
