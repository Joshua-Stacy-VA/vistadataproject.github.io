MAGQBP ALL SHARES

The output of this RPC is a list of all online magnetic shares(non-routing). This list establishes a set of roots for the purge processto downwardly traverse and evaluate the age and VistA attributes of eachimage file for the purpose of purging archaic and archived files.

Properties

Property Value
Label ALLSHR
MUMPS Implementation MAGQBUT1
Return Type ARRAY

Input Parameters

Name Type Maximum Data Length Required Description
TYPE LITERAL 8 true This value is used to determine if there is a screen intended for members of the Next RAID Group."ALL" performs no screen."AUTO" will screen for members of the Next RAID group.
