# DSIF GET PRINTERS

 Returns a list of exisiting VistA printers. Using routine: DEVICE^XUDHGUI

## Properties

Property | Value
--- | ---
Label | GETPRINT
Routine | [DSIFENA5](http://code.osehra.org/dox/Routine_DSIFENA5_source.html)

Name | Type | Maximum Data Length | Required | Description
--- | --- | --- | --- | ---
