# XU REBUILD MENU TREE

This API rebuilds the menu trees and display nodes for a single user (DUZ). It returns 0 if unsuccessful, 1 if successful.

Routine | [XQ84](http://code.osehra.org/dox/Routine_XQ84_source.html)
