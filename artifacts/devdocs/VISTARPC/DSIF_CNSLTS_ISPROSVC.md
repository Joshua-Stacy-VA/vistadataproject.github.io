# DSIF CNSLTS ISPROSVC

This RPC will return whether the specified Consult Service is part of the Consults-Prosthetics Interface. Cloned from ORQQCN ISPROSVC

Routine | [DSIFCNS1](http://code.osehra.org/dox/Routine_DSIFCNS1_source.html)
