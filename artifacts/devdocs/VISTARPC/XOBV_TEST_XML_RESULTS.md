# XOBV TEST XML RESULTS

This simple RPC takes no input parameters and returns a 'chunk' of data that is in XML format. This RPC is used to test whether VistALink properly transports and parses application results in XML format. This RPC is used as part of the test and learning application distributedas part of VistALink.

Property | Value
--- | ---
Label | XMLRP
Routine | [XOBVLT](http://code.osehra.org/dox/Routine_XOBVLT_source.html)
Return Type | SINGLE VALUE
