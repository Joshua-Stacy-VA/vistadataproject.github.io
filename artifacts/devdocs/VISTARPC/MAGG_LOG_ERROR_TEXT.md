# MAGG LOG ERROR TEXT

This call inserts text into the Error field of the IMAGING WINDOWSSESSION FILE.

## Properties

Property | Value
--- | ---
Label | LOGERROR
Routine | [MAGGTU3](http://code.osehra.org/dox/Routine_MAGGTU3_source.html)

## Input Parameters

Name | Type | Maximum Data Length | Required | Description
--- | --- | --- | --- | ---
