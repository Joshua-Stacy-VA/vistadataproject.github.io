# YTQ ADMIN DELETE

Users who have the MHS Manager menu can delete a specified MH Administration and its MH Answers and MH Results. Please use caution as deletions cannot be restored. Only ADPACs should usethis feature and in consultation with medical records policy.  Input: YS("AD") as internal entry number of MH ADMINISTRATIONS (#601.84) file.Output: Data vs ERROR

--- | ---

Routine | [YTQAPI5](http://code.osehra.org/dox/Routine_YTQAPI5_source.html)
