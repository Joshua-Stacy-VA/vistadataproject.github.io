# PSB WARDLIST

Returns a list of active wards that are availablefor the definition of IV parameters in the BCMA IV PARAMETERS file 53.66.Wards already in file 53.66 are returned with the type of IV PARAMETERSdefined.

Property | Value
--- | ---
Label | WLIST
Routine | [PSBPARIV](http://code.osehra.org/dox/Routine_PSBPARIV_source.html)
Return Type | GLOBAL ARRAY
