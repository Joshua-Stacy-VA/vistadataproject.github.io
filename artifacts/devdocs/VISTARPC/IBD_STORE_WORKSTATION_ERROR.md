# IBD STORE WORKSTATION ERROR

This call back is used by AICS to log entries in the AICS ERROR AND WARNINGFILE when the recognition of a form is canceled or otherwise interrupted.

Property | Value
--- | ---
Label | WSERR
Routine | [IBDFBK1](http://code.osehra.org/dox/Routine_IBDFBK1_source.html)
Return Type | SINGLE VALUE


### Input Parameters

Name | Type | Maximum Data Length | Required | Description
--- | --- | --- | --- | ---
FORMID | LIST | 40 | true |
