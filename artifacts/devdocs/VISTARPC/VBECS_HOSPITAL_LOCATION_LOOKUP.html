VBECS HOSPITAL LOCATION LOOKUP

This RPC returns XML containing a list of Hospital Locations from the HOSPITAL LOCATION file (#44) of either Ward or Clinic Type for use in VBECS when issuing units to a location other than the Blood Bank. XML Mapping:LocationName element = HOSPITAL LOCATION file (#44) / NAME field (#.01)LocationIEN element = HOSPITAL LOCATION file (#44) / NUMBER field (#.001)LocationType element = HOSPITAL LOCATION file (#44) / TYPE field (#2) XML Example: 6W REHAB-KC 503 W 7E PCUS-KC 504 W KC-GI RESEARCH 511 C
