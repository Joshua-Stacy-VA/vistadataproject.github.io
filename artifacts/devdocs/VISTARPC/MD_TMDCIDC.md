# MD TMDCIDC

This RPC will do the following:  Input Parameter: RESULTS - (Both Input/Output) Passed in as the array to                              return the results.                   OPTION  - (Input) PROC - obtain a list of Procedures                                            defined for a clinic.                                     DIAG - obtain a list of diagnosis                                            defined for a clinic.                                     SCDISP - Obtain the patient's service                                              connection and rated disability.                   DFN     - (Input) Patient internal entry number                   MDSTUD   - (Input) CP Study internal entry number
