# MAGV DELETE DICOM FAILED IMAGE

This routine will delete a single entry in the DICOM FAILED IMAGES File(#2006.575).

--- | ---
Label | DCRCTDEL
MUMPS Implementation | [MAGVRS51](http://code.osehra.org/dox/Routine_MAGVRS51_source.html)
