ROR TASK VALIDATE RESCHEDULING

The ROR TASK VALIDATE RESCHEDULING remote procedure validates the rescheduling code and returns the next date/time to run.

Properties

Property Value
Label VALIDSCH
MUMPS Implementation RORRP012
Return Type ARRAY

Input Parameters

Name Type Maximum Data Length Required Description
SCHCODE LITERAL   true Rescheduling code ("1D", "1M", "D@12:00", etc).
SCHDT LITERAL     Date when a task is scheduled to run for the first time (FileMan). By default (if $G(SCHDT)'>0), the current date/time is used.
