# ROR TASK DELETE

The ROR TASK DELETE remote procedure completely deletes the task.

## Properties

Property | Value
--- | ---
Label | DELTASK
Routine | [RORRP010](http://code.osehra.org/dox/Routine_RORRP010_source.html)
Return Type | SINGLE VALUE
