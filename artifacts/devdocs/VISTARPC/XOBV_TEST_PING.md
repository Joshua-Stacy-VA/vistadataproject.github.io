# XOBV TEST PING

This simple RPC takes no input and returns a success message. This is the primary mechanism to test whether the client successfully connects toan M server using VistALink.                                      This RPC is used as part of the test and learning application distributedas part of VistALink.

Property | Value
--- | ---
Label | PINGRP
Routine | [XOBVLT](http://code.osehra.org/dox/Routine_XOBVLT_source.html)
Return Type | SINGLE VALUE
