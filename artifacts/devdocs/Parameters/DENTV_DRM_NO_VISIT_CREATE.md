# DENTV DRM NO VISIT CREATE

This parameter controls whether or not users may add PCE visit &#x27;on the fly&#x27; within DRM Plus.  If set to yes, then no visits can be created and the user must select an existing PCE visit for the dental encounter.

## Properties

Property | Value
--- | ---
Summary | Don&#x27;t allow creation of PCE visits
Value Type | yes
Value Domain | 
Multiple Instances | false
