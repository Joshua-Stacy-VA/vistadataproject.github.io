# LR MAPPING ASK DOMAIN

This parameter holds the default when asking the user if theywant to specify a specific domain when selecting a facility tosend a STS file mapping. It allows the user to override thenormal domain associated with an Institution. This is useful when sending the mapping to a facility&#x27;s test system which is usually anotherMailMan domain.
