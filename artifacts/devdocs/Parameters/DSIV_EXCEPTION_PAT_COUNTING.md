# DSIV EXCEPTION PAT COUNTING

If the parameter &#x3D; Yes, then the Exception Report will contain only 1 patient per day.  If the parameter &#x3D; No, then the Exception Report will count clerks (or appts) and could have the same patient on the report multiple times for different clinics (appts).
