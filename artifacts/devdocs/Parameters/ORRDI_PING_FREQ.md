# ORRDI PING FREQ

This parameter represents the number of seconds between the initiation of attempts to ping the HDR for a successful read.  This pinging takes place when an OUTAGE state is encountered for RDI.

--- | ---
ID | 463
Summary | Frequency of HDR ping attempts
Value Type | numeric
Value Domain | N/A
