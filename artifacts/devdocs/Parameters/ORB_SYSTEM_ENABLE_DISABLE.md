# ORB SYSTEM ENABLE/DISABLE

Parameter determines if any notification processing will occur.  &#x27;E&#x27; or&#x27;Enable&#x27; indicates the notifications system is enabled and running.  &#x27;D&#x27;or &#x27;Disabled&#x27; indicates the notifications system is disabled and notrunning.  Can be set at the Institution, System or Package level.

## Properties

Property | Value
--- | ---
Summary | Enable or disable notifications.
Value Type | set of codes
Value Domain | E:Enable;D:Disable
Multiple Instances | false
