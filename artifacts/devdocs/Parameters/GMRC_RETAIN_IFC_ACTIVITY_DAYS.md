# GMRC RETAIN IFC ACTIVITY DAYS

This parameter controls the number of days that completed Inter-facility Consult transactions are maintained in the IFC MESSAGE LOG (#123.6) file. Completed transactions represent Consult activities transmitted to aremote facility via HL7 which have been responded to with no error condition.

Property | Value
--- | ---
ID | 4735
Summary | Days to retain complete IFC transactions
Value Type | numeric
Value Domain | 7:180
Multiple Instances | false
