# DENTV DRM DUPCHECK OFF

This parameter controls whether or not the users see the Potential Duplicates screen which attempts to synchronize planned and completed care.  If set to yes, then no duplicate processing is done.  If not defined, or set to no, (the default) then potential dupes will be shown to the user for selection/processing.

## Properties

Property | Value
--- | ---
Summary | Turn off Potential Duplicate Processing
Value Type | yes
Value Domain | 
Multiple Instances | false
