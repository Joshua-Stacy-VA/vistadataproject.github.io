# LA7UTILA PARSE

Allows the User to select the default setting for &quot;Parse HL7 Message&quot; prompt when using OPTION [LA7 PRINT LAB UI MESSAGE]. &quot;LAST&quot; means the User wants the system to keep track of their lastresponse to this prompt and use that as their default. 

## Properties

Property | Value
--- | ---
Summary | Lab Messaging - Parse HL7 Messages
Value Type | set of codes
Value Domain | Y:YES;N:NO;L:LAST
Multiple Instances | false
