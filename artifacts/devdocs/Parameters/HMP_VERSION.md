# HMP VERSION

This parameter holds the current version number of the eHMP PatientRecord (HMP) data extract RPC&#x27;s, in the form V.PP where V is thepackage version number and PP is the latest patch number.

## Properties

Property | Value
--- | ---
Summary | HMP Version
Value Type | numeric
Value Domain | 1:99:2
Multiple Instances | false
