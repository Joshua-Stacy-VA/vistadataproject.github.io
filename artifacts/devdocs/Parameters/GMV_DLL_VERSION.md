# GMV DLL VERSION

This parameter is used to store the DLL versions that are compatible withthe current server version of the Vitals/Measurements application. TheInstance format is DLL:VERSION. Example:  GMV_VITALSVIEWENTER.DLL:v. 07/21/05 10:34

Summary | Vitals DLL version check
Value Type | yes
