# PSB DEFAULT BARCODE SITE

Used to flag whether or not to print the Site Number on the labelsgenerated with the PSBO BL Option.  If Yes then the barcode will includethe station number - Drug IEN (i.e. 677-263 for drug 263 at station 677)If the answer is No then only the Drug IEN will print (i.e. 263).
