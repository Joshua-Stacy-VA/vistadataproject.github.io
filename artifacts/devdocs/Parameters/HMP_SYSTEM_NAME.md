# HMP SYSTEM NAME

This parameter holds the name of this VistA system, as a hashed base 16value.  It is calculated by a HMP patch post-init and stuffed into theSYStem level value, and should NOT be modified.

## Properties

Property | Value
--- | ---
Summary | Hashed System Name
Value Type | free text
Value Domain | 1:10
Multiple Instances | false
