# GMRC IFC BACKGROUND FINISH

This parameter holds the date/time that the Inter-facility Consults background job last finished running. The parameter is set by the background job.  This parameter is not intended to be modified usingparameter tools. If this parameter becomes delinquent for an extended period, IRM personnelwill be alerted that the job is overdue.
