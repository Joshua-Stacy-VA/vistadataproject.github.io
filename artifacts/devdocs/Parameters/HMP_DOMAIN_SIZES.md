# HMP DOMAIN SIZES

 This parameter contains a list of VPR extract domains and their average sizes.  The sizes are used in computing the approximate size of an item with limiting the freshness calls by size.

## Properties

Property | Value
--- | ---
Summary | HMP Average Domain Sizes
Value Type | numeric
Value Domain | 0:9999999999:0
Multiple Instances | true
