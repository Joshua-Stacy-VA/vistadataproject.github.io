# RCDPE APAR

The AUTO-POST - AWAITING RESOLUTION screen that is generated whenexecuting the RCDPE AUTO-POST AWAITING RESOLUTION option provides thecapability to filter the data displayed via the Change View action. TheChange View action saves the filter preferences by user.
