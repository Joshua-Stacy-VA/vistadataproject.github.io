# ORWRP HEALTH SUMMARY LIST ALL

 This parameter overrides the ORWRP HEALTH SUMMARY TYPE LIST by making all health summary types available, in alphabetic order.

## Properties

Property | Value
--- | ---
Summary | List All Health Summary Types
Value Type | yes
Value Domain |
