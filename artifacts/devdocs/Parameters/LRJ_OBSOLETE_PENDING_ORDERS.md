# LRJ OBSOLETE PENDING ORDERS

The purpose of this parameter is to store the number of days before pending orders will be considered obsolete. When this number of days haspassed, the order will be canceled in legacy lab and a lapse order statusupdate will be sent to CPRS.

Value Type | numeric
Value Domain | 1:365
