# DVBAB CAPRI VICAP URL

This parameter contains the URL used to connect to the VA Informatics for Compensation and Pension (VICAP) system.  VICAP is a replacement for the AMIS 290 reporting in the AMIE package. Placing any value in this parameter has the effect of disabling the AMIS290 for the Regional Office [DVBA C RO AMIS 290] and AMIS 290 Report forC&amp;P [DVBA C AMIS REPORT] options in VistA, the corresponding AMIS 290 andAMIS 290 by Division reports on the CAPRI Reports dialog box, and the AMIS290 report on the Consolidated Remote Reports dialog box in the CAPRI GUIapplication.
