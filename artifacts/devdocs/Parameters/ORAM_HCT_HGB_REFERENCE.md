# ORAM HCT/HGB REFERENCE

This is the reference to the Lab Test for Hematocrit (HCT) or Hemoglobin (Hgb). It will determine whether the Demographics Tab will display the most recent Hematocrit or the last Hemoglobin.

## Properties

--- | ---
Summary | Hematocrit (HCT) or Hemoglobin (Hgb)
Value Type | pointer
Value Domain | 60:AEMQ
