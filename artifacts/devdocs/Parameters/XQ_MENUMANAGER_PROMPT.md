# XQ MENUMANAGER PROMPT

On non-production VistA systems, the text defined by this parameter is inserted in the MenuManager prompts. If no text is defined, the hard-coded default is &quot; &lt;TEST ACCOUNT&gt;&quot;. Suggested alternatives include&quot; &lt;LEGACY SYSTEM&gt;&quot;,&quot; &lt;CONTINGENCY&gt;&quot;, or &quot; &lt;READ ONLY&gt;&quot;.

## Properties

Property | Value
--- | ---
Summary | Non-production VistA Warning Prompt
Value Type | free text
Value Domain | 3:20
Multiple Instances | false
