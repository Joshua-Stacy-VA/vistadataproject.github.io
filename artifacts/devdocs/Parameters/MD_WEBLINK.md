# MD WEBLINK

This parameter contains the web address for the Clinical Procedures homepage.  This can be modified to a local address in the event that the pagesare downloaded to be displayed from a local server location.

## Properties

Property | Value
--- | ---
Summary | Clinical Procedures Home Page
Value Type | free text
Value Domain | 1:250
Multiple Instances | false
