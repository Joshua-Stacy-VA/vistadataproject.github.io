# ORQQPX COVER SHEET REM CLASSES

This parameter works in conjunction with the ORQQPX COVER SHEET REMINDERS parameter, acting as another level tothe cumulative parameter of User Class, inserted between the User andLocation Levels.  Each line of the Value word processing field represents a different reminder or category, and is in the formatSequence;FlagTypeIEN, where Flag indicates L:Lock, R:Remove or N:Normal, Type is a &#x27;C&#x27; or &#x27;R&#x27;, indicating that # is the internal entry number of a Category or a Reminder.
