# LR CH VERIFY CPRS ALERT

Used to allow the user to determine if they want to be promptedto send a CPRS alert after editing a Chem/Heme accession. The user can indicate: 1. They do not want to be asked 2. They want to be asked with a default of YES/NO to send an alert.

ID | 720
Summary | Prompt CPRS Alert in CH Result Entry
Value Type | set of codes
Value Domain | 0:Don&#x27;t Ask;1:Ask - Default NO;2:Ask - Default YES
