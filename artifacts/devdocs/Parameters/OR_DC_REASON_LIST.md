# OR DC REASON LIST

This parameter determines the sequence sites want the order DC reasons to  appear. Sites do not need to set a sequence for every DC Reasons.For example if a site wants to set the &quot;Requesting Physician Cancelled&quot; DC Reasons to the top of the list. Sites will set a value  of 1 for thatentry and CPRS will display the rest of the DC Reasons after the&quot;Requesting Physician Cancelled&quot;  DC reason.
