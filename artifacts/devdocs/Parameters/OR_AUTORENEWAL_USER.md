# OR AUTORENEWAL USER

This parameter points to a proxy user in the NEW PERSON (#200) file created for the auto-renewal process. This value is used to populate the &#x27;entered by&#x27; field in the ORDERS (#100) file, and the &#x27;Entry by&#x27; field in the PRESCRIPTION (#52) file, and is the proxy user IEN placed in the AudioRenewal module User Parameters field &#x27;Host User ID#&#x27;.
