# MDK APPLICATION INSTALL

This parameter is used to store the Hemodialysis applicationdistribution information.  The information includes the following:1) Date/Time when application first launched.2) User Name3) System Option Loaded (Y/N)4) Workstation of where the application was launched.

Value Type | free text
Value Domain | 1:250
Multiple Instances | true
