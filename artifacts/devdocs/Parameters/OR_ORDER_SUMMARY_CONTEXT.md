# OR ORDER SUMMARY CONTEXT

A value of 0 will print all orders with ORDER dates within the selected date range. A value of 1 will print all orders with START dates within the selecteddate range. A value of 2 will print all orders with START dates and orders with anyACTIVITY within the selected date range.

## Properties

--- | ---
Summary | Order Summary Context
Value Type | set of codes
Value Domain | 0:ORDER DATE;1:START DATE;2:START DATE PLUS ACTIVITY
Multiple Instances | false
