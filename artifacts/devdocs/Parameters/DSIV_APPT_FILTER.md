# DSIV APPT FILTER

This parameter determines how appointments are filtered out in the patient update list for ICB.  0&#x3D;not filtered  1&#x3D;filter out checked-in status (this also filters checked-out status)  2&#x3D;filter out checked-out status

Summary | ICB PATIENT UPDATE APPT FILTER
Value Type | numeric
Value Domain | 0:99
Multiple Instances | false
