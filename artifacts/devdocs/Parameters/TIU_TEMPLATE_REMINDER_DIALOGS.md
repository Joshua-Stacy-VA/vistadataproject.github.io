# TIU TEMPLATE REMINDER DIALOGS

Contains Reminder Dialogs that are allowed to be used as TIU Templates.This parameter is different than most others in that each level iscumulative, so all Reminder Dialogs at the System, Division, Serviceand User levels can be used in Templates.

## Properties

Property | Value
--- | ---
Summary | Reminder Dialogs allowed as Templates
Value Type | pointer
Value Domain | 801.41
Multiple Instances | true

## Allowable Entities

Precedence | Entity
--- | ---
1 | USER
3 | SERVICE
4 | DIVISION
5 | SYSTEM
