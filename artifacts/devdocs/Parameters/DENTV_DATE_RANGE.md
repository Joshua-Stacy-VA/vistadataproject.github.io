# DENTV DATE RANGE

This parameter contains the values for the various date range settings within DRM.  The personal preference setup window allows the user to set two values:  Number of days back that data will be retrieved from Today  Maximum number of records to retrieve The instance for this parameter is the name of the packageThe value is the numeric value for those two mentioned above The instance has a format of &lt;package&gt; NUMBER OF DAYS                             &lt;package&gt; MAX  where &lt;package&gt; : &#x3D; DENT                      LAB                      NOTES                      VISIT
