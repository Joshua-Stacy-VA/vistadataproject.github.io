# ORRDI SUCCEED THRESH

This parameter is a threshold value that holds the minimum number of successfull HDR reads for RDI before RDI will come out of the OUTAGE state.  The successfull reads are peformed in a task that is spawned when the OUTAGE state is initiated.

Property | Value
--- | ---
ID | 462
Summary | Threshold for HDR read successes
Value Type | numeric
Value Domain | N/A
Multiple Instances | false
