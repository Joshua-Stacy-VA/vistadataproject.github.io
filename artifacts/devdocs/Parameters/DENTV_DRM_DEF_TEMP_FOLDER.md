# DENTV DRM DEF TEMP FOLDER

The DRM GUI application has need to occasionally create temporary files.  This parameter stores the location for those temporary files.  The default location is c:\temp\.  The value must be a valid Windows NT path designation.

## Properties

Property | Value
--- | ---
Summary | Temporary Folder Location for PC
Value Type | free text
Value Domain | 3:100
Multiple Instances | false
