# GMRC IFC ALERT IMMED ON PT ERR

This parameter controls whether the IFC CLIN ERRORS GROUP at the requesting site is notified immediately of patient information errors at the consulting site.  If set to yes, this group will be alerted every time an Unknown Patient error is returned from the consulting site. If set to No, this group is only notified if the problem is unresolved after5 re-transmissions.
