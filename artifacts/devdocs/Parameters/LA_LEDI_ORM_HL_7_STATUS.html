LA LEDI ORM HL7 STATUS

This parameter is used to control the building of LEDI HL7 order messageson an interface basis for specific messaging partners based on the AGENCYassociated with the interface. It allows for the selectiveenabling/disabling of either all order messaging or just order messagingin support of specific areas of VistA Laboratory. It currently allows for control of the main laboratory areas, also referred to as subscripts or a combination. The areas are: 'CH' = laboratory tests within the "CH" subscript of LAB DATA file (#63). 'MI' = laboratory tests within the "MI" subscript of LAB DATA file (#63). 'AP' = laboratory tests within the "SP", "CY", and "EM" subscript of LAB DATA file (#63). Selecting "Disabled" disables all sections of laboratory.Selecting "Enabled" enables all three main sections ("CH", "MI", and "AP")
