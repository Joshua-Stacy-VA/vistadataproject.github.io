# ORQQAP SEARCH RANGE STOP

Returns the relative date to end listing appointments for a patient atall clinics.  For example, &#x27;T+30&#x27; will not list appointments at theclinics after 30 days from now. &#x27;T&#x27; will not list appointments later thantoday.
