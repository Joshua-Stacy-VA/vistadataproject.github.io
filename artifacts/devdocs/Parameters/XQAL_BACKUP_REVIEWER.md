# XQAL BACKUP REVIEWER

This parameter contains information about the Backup Reviewer for unprocessed alerts.  This person will be sent the alerts for thespecified entity that remain unprocessed by the original recipients.

--- | ---
Summary | Backup Reviewer for Unprocessed Alerts
Value Type | pointer
Value Domain | 200
