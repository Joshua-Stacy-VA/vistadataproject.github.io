# ORB REMOVE

Value indicates if the notification/alert can be deleted without processing.  A &quot;yes&quot; value indicates the notification/alert can be deletedwithout processing.  In the CPRS GUI a&quot;yes&quot; value signifies a selectedalert can be deleted when the Remove button is clicked.  If anotification/alert has a blank value or a &quot;no&quot; value, thenotification/alert cannot be deleted without processing.
