# XU645

This parameter determines if a terminated user information should be deleted. A &#x27;No&#x27; value means you do not want to purge the terminated user information. This was requested by the OIG when they want all user information preserved. A &#x27;Yes&#x27; value means to purge the information, which is normal operating procedure.
