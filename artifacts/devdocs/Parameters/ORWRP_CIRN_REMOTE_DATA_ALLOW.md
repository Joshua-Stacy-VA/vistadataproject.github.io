# ORWRP CIRN REMOTE DATA ALLOW

This parameter determines who is allowed access to patient data atremote sites.  The values for this parameter can be controlled down to theUSER entity.

Property | Value
--- | ---
ID | 33
Summary | Allow remote data access
Value Type | yes
Value Domain | N/A
Multiple Instances | false
