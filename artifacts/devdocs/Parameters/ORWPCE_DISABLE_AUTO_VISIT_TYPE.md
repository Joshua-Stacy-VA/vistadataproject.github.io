# ORWPCE DISABLE AUTO VISIT TYPE

When set to Yes, this parameter prevents the automatic selectionof a Type of Visit on the Visit Tab of the encounter form.

## Properties

Property | Value
--- | ---
Summary | Disable Auto Visit Type
Value Type | set of codes
Value Domain | 0:NO;1:YES
Multiple Instances | false
