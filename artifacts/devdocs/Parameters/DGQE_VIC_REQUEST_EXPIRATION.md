# DGQE VIC REQUEST EXPIRATION

This parameter defines the number of days that a VIC Request with a CardPrint Release Status of &quot;H&quot;old is allowed to remain pending before being cancelled.

ID | 365
Summary | VIC Request Expiration
Value Type | numeric
