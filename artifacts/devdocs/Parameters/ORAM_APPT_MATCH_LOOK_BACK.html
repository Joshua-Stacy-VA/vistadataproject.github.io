ORAM APPT MATCH LOOK-BACK

This is the whole number of days to look-back when attempting to match theAnticoagulation Encounter to a scheduled appointment in the VistA Scheduling Package. Please enter a number between 0 and 7, no decimal digits. The default will be one (1) day (i.e., yesterday). A value of zero (0)will restrict the search to begin with today's appointments (i.e.,T@00:00).
