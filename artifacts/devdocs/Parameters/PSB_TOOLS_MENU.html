PSB TOOLS MENU

This parameter may be used to identify which items should appear on the tools menu which is displayed by the BCMA GUI. Each item should contain a name that should be displayed on the menu, followed by an equal sign, followed by the command string used to invoke the executable. This string may also include parameters that are passed to the executable.Some example entries are: Hospital Policy=C:\WINNT\SYSTEM32\VIEWERS\QUIKVIEW.EXE LOCPLCY.DOCVISTA Terminal=C:\PROGRA~1\KEA\KEAVT.EXE VISTA.KTC An ampersand may be used in the name portion to identify a letter thatshould be underlined on the menu for quick keyboard access. For example,to underscore the letter H in Hospital Policy, enter &Hospital Policy asthe name part.
