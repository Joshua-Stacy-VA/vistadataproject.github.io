# LR MAPPING MESSAGE MAX RECORDS

This parameter holds the number of records to transport per mail messagewhich contains the STS mapping of standard code sets to VistALaboratory system files. It allows the message size to be adjustedto accommodate the various system settings through which the mailmessages containing the STS mappings are routed.

--- | ---
ID | 704
Summary | Lab STS Number of Records per Message
Value Type | numeric
Value Domain | N/A
Multiple Instances | false
