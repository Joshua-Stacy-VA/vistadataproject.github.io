# PSB COVERSHEET V1 COL WIDTHS

USER specific setting for the BCMA Coversheet&#x27;s &quot;first view&quot; . This parameter contains the width of each column viewed via the Bar Code Medication Administration Coversheet&#x27;s &quot;first view&quot; delimited by the slash (/).  Each &quot;/&quot;-piece of data is per respective column on the Coversheet&#x27;s &quot;first view&quot;, numbered from left to right.  An example parameter setting is:       170/20/100/30/11/22/31/44/50 Setting numbers are in units of centimeters.    THESE NUMBERS MUST BE WHOLE NUMBERS.
