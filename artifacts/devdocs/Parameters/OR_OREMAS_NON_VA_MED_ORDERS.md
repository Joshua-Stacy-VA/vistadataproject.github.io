# OR OREMAS NON-VA MED ORDERS

This parameter determines if clerks (i.e. users holding the OREMAS key)are allowed to act on non-VA med orders.  Enter YES to permit a clerk toenter new or DC non-VA med orders and send them to Pharmacy for reports and order checks. To prohibit clerks from handling non-VA med ordersentirely, select NO.

--- | ---
ID | 360
Summary | Allow Clerks to act on Non-VA Med Orders
Value Type | set of codes
Value Domain | 0:NO;2:YES
Multiple Instances | false
