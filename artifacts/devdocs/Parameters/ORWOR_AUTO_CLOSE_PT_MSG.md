# ORWOR AUTO CLOSE PT MSG

This parameter controls how long the patient messages window displaysbefore automatically closing.  The default is 5.  If the number of secondsis set to 0, the window will remain open until the user clicks it closed.

## Properties

--- | ---
Summary | Auto-Close Patient Messages
Value Type | numeric
Value Domain | 0:99
Multiple Instances | false
