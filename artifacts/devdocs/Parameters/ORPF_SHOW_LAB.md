# ORPF SHOW LAB #

This field controls the listing of lab orders for holders of the ORES key,after the electronic signature has been entered when entering new orders.Only after the order is released to Lab service is the number assigned;if physicians want to see the lab order # with the order after enteringand signing the orders, this parameter must be set to YES.  All otherusers get the listing regardless of what this parameter is set to.
