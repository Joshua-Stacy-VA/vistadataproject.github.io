# PSB COVERSHEET VIEWS COL SORT

This is a USER SPECIFIC PARAMETER. This parameter contains 4 column number used to sort each of the 4 different Coversheet Views respectively, delimited by the slash (/)                                                                character.  First (Medication Overview),  Second (PRN Overview), Third (IVOverview), and Forth (Expired/Expiring Orders), are the coversheet views in sequence.  Each view&#x27;s columns are numbered from left to right.  An example parameter setting is:             7/3/1/3
