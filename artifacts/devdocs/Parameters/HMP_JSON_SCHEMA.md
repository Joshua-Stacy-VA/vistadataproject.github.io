# HMP JSON SCHEMA

This contains the version number of the HMP schema which describes theJSON objects produced by the HMP extracts.  The whole number incrementswhenever the schema changes in a way that requires a conversion orre-extraction of objects.  The fractional piece increments whenever theschema is extended.
