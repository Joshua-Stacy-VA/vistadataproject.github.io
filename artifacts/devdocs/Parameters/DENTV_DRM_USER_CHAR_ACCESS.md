# DENTV DRM USER CHAR ACCESS

The DRM GUI allowed the user to lookup NEW PERSON information similar to the Kernel&#x27;s Toolbox option.  It also allowed the user to edit certain fields in the NEW PERSON file for their own record.  This parameter will allow the site to configure which of these options are available.

## Properties

Property | Value
--- | ---
Summary | 
Value Type | set of codes
Value Domain | N:no access;D:display only;E:edit/display allowed
Multiple Instances | false
