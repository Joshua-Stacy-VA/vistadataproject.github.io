# MD DAYS TO RETAIN COM STUDY

The number of days after check-in date/time to display the studythat has been completed in the CPUser application.  Studies that haveprocedures with multiple or cumulative results are NOT included.Cumulative and multiple results studies will have a default value of365.

## Properties

--- | ---
Summary | Days to Retain Completed Study
Value Type | numeric
Value Domain | 1:365
Multiple Instances | false
