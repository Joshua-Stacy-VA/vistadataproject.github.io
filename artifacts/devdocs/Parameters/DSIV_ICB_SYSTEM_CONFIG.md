# DSIV ICB SYSTEM CONFIG

This parameter contains settings for the ICB GUI.  If installed on an existing system, this parameter will contain the contents of the DSS.ini file for ICB.  If DSS.ini did not exist prior to installation, the pertinent data will be created and stored within this parameter.
