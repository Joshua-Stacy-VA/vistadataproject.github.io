# ORWDXVB VBECS TNS CHECK

This is used in the VBECS Order Dialog to check for recent orders for aType &amp; Screen order.  The default is 3 days, but a site can override thisnumber by setting this parameter to something different.

--- | ---
ID | 500
Summary | Days to check for Type &amp; Screen
Value Type | numeric
Value Domain | N/A
