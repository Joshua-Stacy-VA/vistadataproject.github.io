# ORPF CONFIRM PROVIDER

This field will allow an additional prompt to confirm the providerselection when adding new orders.Entering 2 in this field will exclude holders of the ORES key(providers) from this check.Notice that these parameters also control the default presented to the user:&#x27;Are you sure? &lt;no -or- yes&gt;&#x27;

Property | Value
--- | ---
ID | 176
Summary | Confirm Provider
Value Type | set of codes
Value Domain | 0:NO;1:YES (Default NO);2:YES (Exclude ORES);3:YES (Default YES)
Multiple Instances | false
