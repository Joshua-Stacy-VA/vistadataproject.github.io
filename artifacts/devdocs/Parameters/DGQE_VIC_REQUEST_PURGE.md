# DGQE VIC REQUEST PURGE

This value is used by the VIC Background Processing [DGQE BACKGROUND PROCESSING] option to determine how many days to retain completed VIC request transactions in the VIC REQUEST (#39.6) and VIC HL7 TRANSMISSIONLOG (#39.7) files.
