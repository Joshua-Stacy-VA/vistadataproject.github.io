# ORB FLAGGED ORDERS BULLETIN

&#x27;Yes&#x27; indicates a MailMan bulletin will be sent to the order&#x27;s Current Provider (usually the Ordering Provider) when the order is flagged forclarification.  This parameter has no effect on the Flagged Ordersnotification which is also triggered when an order is flagged forclarification.
