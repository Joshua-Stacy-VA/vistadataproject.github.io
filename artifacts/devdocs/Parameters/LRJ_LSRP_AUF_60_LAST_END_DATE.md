# LRJ LSRP AUF60 LAST END DATE

This field indicates the end date used when the File 60 Audit Report messages were previously generated by TaskMan. The purpose of this data is to record the previous end date for the report generated by the TaskMan job.  This date will be the start date used when File 60 audit messages are next generated by the TaskMan option. CHANGING THIS DATE WILL AFFECT THE INFORMATION REPORTED THE NEXT TIME TASKMAN RUNS THE File 60 Audit Report.
