# DG PT DIRECTORY CALL CENTER

Enabling this parameter will allow for the transmission of the privacy indicator to be included in HL7 messages generated by the Registration application in the PV2 segment. The privacy indicator allows patient to opt-out form being listed from facility directories. By including this information in the HL7 message the information can be communicated to COTSapplications doing patient directories.

## Properties

Property | Value
--- | ---
Summary | Enable Patient Directory Call Center
Value Type | numeric
Value Domain | 0:1
Multiple Instances | false
