# LRJ HL LAST END DATE

This field indicates the end date used when the Hospital Location Change Report messages were previously generated by TaskMan. The purpose of this data is to record the previous end date for the report generated by the TaskMan job.  It is used to prevent the TaskManjob from reporting a Hospital Location change or addition more than onceand differently than was previously reported. This date will be the start date used when the Hospital Location ChangeReport messages are next generated by the TaskMan option. CHANGING THIS DATE WILL AFFECT THE INFORMATION REPORTED THE NEXT TIMETASKMAN RUNS THE LRJ SYS MAP HL TASKMAN RPT.
