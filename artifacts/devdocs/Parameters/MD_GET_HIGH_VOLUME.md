# MD GET HIGH VOLUME

This parameter will contain a free text string that contains two piecesof data delimited by a semicolon &#x27;;&#x27;.  The two pieces of data are: 1)1/0 (Yes/No) to indicate whether or not the text of the result should beadded to the note, 2) 1/0 (Yes/No) to enter the text of the result as thesignificant finding of the Consult. (If you enter a 0, the note will beauto closed with the text inside.)Example string: 1;0
