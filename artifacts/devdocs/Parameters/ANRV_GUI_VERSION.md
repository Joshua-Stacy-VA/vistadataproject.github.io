# ANRV GUI VERSION

This parameter is used to store the application:version(s) that are compatible with the current server version of ANRV Patient Reviews.  Instance format of APPLICATION:VERSION  (example:  ANRV.EXE:0.0.0.0).

--- | ---
ID | 307
Summary | Active ANRV Executable
Value Type | yes
Value Domain | N/A
