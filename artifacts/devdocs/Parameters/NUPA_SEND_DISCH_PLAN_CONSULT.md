# NUPA SEND DISCH PLAN CONSULT

Allows each site to determine if they want the nurses to be able to send Discharge Planning consults.  If you say NO here, the &quot;Discharge Planning Consult&quot; button on page 2 of the Discharge Planning tab will be grayedout.
