# ORWD NONVA REASON

This parameter lists the reasons and statements for ordering/documenting a non-VA medication.  Non-VA meds include herbals, OTCs (over-the-counter medications) and prescriptions not obtained at VA pharmacies or from VAmail delivery services.

## Properties

Property | Value
--- | ---
Summary | Non-VA Meds Statement/Reason
Value Type | free text
Value Domain | 1:60
Multiple Instances | true
