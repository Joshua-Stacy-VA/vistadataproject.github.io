# DENTV DRM MONITORED DATA

This field changes the background color on the cover page for the Monitored Exam field.  If the date is older than the number of days in the parameter, then the default color will change to white to highlight the older date.

Property | Value
--- | ---
ID | 506
Summary | HIGHLIGHT MONITORED DATE ON COVER PAGE
Value Type | numeric
Value Domain | 1:999
Multiple Instances | false
