# LA7UTILA USE BROWSER

Allows the User to select the default setting for &quot;Use Browser to display HL7 Message&quot; prompt when using OPTION [LA7 PRINT LAB UI MESSAGE]. &quot;LAST&quot; means the User wants the system to keep track of their lastresponse to this prompt and use that as their default.

Property | Value
--- | ---
ID | 693
Summary | Lab Messaging - Display using Browser
Value Type | set of codes
Value Domain | Y:YES;N:NO;L:LAST
Multiple Instances | false
