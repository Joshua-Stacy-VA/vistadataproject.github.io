# DENTV DRM DAS TURNOFF

This parameter controls the DRM user interface to not allow transactions to be filed to DAS (file 221).  The default date will be 10/31/2004, but if the HL7 interface, or the AAC or VA-DSS determines that they are not ready to turn off DAS, we can change this date.

## Properties

Property | Value
--- | ---
Summary | Turn off sending data to DAS (file 221)
Value Type | date
Value Domain | 
Multiple Instances | false
