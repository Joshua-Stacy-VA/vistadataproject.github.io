# ORK EDITABLE BY USER

System and Division indicate if the order check can be edited bya user.  Valid values include &quot;Yes&quot; and &quot;No&quot;.  If the value is &quot;Yes&quot;, theorder check can be &quot;Enabled&quot; or &quot;Disabled&quot; by a user.  Users do haveaccess to change the order check&#x27;s ORK PROCESSING FLAG parameter value.If the value is &quot;No&quot;, the order check CANNOT be &quot;Enabled&quot; or &quot;Disabled&quot; bya user. Users do NOT have access to change the order check&#x27;s ORKPROCESSING FLAG parameter value.
