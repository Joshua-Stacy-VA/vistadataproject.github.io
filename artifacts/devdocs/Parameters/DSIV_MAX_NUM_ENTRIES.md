# DSIV MAX NUM ENTRIES

Contains the maximum number of records to return from the buffer in one rpc call.  If there are more than the max records, then multiplay rpc calls need to be made.

--- | ---
ID | 540
Summary | MAXIMUM BUFFER ENTRIES TO RETURN
Value Type | numeric
Value Domain | 1:99999
Multiple Instances | false
