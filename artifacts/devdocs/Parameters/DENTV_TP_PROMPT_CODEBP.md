# DENTV TP PROMPT CODEBP

This parameter determines whether the user will be prompted to reimport the Code Boilerplate when going back to the note after clicking &#x27;Back&#x27; and &#x27;Next&#x27;.  If NO (or not defined) then the Code BP will be automatically imported, if YES, then the user will be prompted and can have the option of not importing the Code Boilerplate into the note.

Property | Value
--- | ---
ID | 505
Summary | PROMPT FOR CODE BOILERPLATE
Value Type | set of codes
Value Domain | 0:NO;1:YES
Multiple Instances | false
