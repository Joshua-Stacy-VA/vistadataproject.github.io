# LRJ HL LAST START DATE

This field indicates the start date used when the Hospital Location Change Report messages were previously generated by TaskMan. The purpose of this data is to record the previous start date for the report generated by the TaskMan job.

## Properties

--- | ---
Summary | HL Report Last Start Date/Time
Value Type | date
Value Domain | ::T
Multiple Instances | false
