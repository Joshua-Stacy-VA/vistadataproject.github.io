DENTV DATE RANGE

This parameter contains the values for the various date range settings within DRM. The personal preference setup window allows the user to set two values: Number of days back that data will be retrieved from Today Maximum number of records to retrieve The instance for this parameter is the name of the packageThe value is the numeric value for those two mentioned above The instance has a format of <package> NUMBER OF DAYS <package> MAX where <package> : = DENT LAB NOTES VISIT
