# ORWOR BROADCAST MESSAGES

This parameter may be used to disable the use of windows messaging tonotify other applications of CPRS events.  Normally, this parameter shouldbe set to &#x27;Yes&#x27;.  If other applications do not respond appropriately tobroadcast messages, this parameter may be set to &#x27;No&#x27; to debug theseapplications.
