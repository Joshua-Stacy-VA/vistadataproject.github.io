# GMRC CONSULT LIST DAYS

This parameter is used to allow a site to set the number of days that will be searched backwards for a Consult to be associated with a NOTE.  A default parameter of 365 days will be set at the Package Level.

--- | ---
Summary | CONSULT LIST DAYS
Value Type | numeric
Value Domain | 0:10000:0
Multiple Instances | false
