# PSB SERVER CLOCK VARIANCE

Used by the client at application startup to ensure that the client clock is within a predetermined number of minutes of the server time before allowing the client to proceed.

## Properties

Property | Value
--- | ---
Summary | Allowable client-server time variance
Value Type | numeric
Value Domain | 0:30:0
Multiple Instances | false
