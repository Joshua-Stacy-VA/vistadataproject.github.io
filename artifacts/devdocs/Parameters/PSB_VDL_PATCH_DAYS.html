PSB VDL PATCH DAYS

PSB VDL PATCH DAYS - The value of this parameter should be between 7 and 14. PSB VDL PATCH DAYS is the parameter which determines the number of days a patient's GIVEN medication patch will display on the respective BarCode Medication Administration (BCMA) Virtual Due List (VDL) and BCMA Coversheet. If this parameter is equal to the "null" value, a GIVEN medication patch will always display on the respective BCMA VDL and BCMA Coversheet.
