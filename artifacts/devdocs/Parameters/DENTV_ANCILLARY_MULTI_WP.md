# DENTV ANCILLARY MULTI WP

This parameter is multi-instance for the purpose of knowing whichancillary products can be launched from within DRM Plus.  It enables, orhides, the appropriate buttons for the user.  For example, DRM Plus canlaunch MiPACS to view dental radiographs, and iMedConsent to get requiredconsent forms.
