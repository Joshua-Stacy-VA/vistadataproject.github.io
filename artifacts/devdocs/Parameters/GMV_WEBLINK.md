# GMV WEBLINK

This parameter contains the web address for the Vitals Measurments homepage.  This can be modified to a local address in the event that the pagesare downloaded to be displayed from a local server location or a site wishes to direct user to a local page instead of the national site.

## Properties

Property | Value
--- | ---
Summary | Vitals Measurments Home Page
Value Type | free text
Value Domain | 1:250
Multiple Instances | false
