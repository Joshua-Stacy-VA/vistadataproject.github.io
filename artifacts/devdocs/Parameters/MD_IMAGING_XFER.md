# MD IMAGING XFER

This parameter contains the name of a network server, share, and path(UNC) to a location where Clinical Procedures can put files for pick-up bythe Imaging background processor for archiving.

## Properties

Property | Value
--- | ---
Summary | Imaging Network Share
Value Type | free text
Value Domain | 1:250
Multiple Instances | false
