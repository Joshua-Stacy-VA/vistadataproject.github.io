description
This file contains all the actions related to the Desktop, and other Clinical Interface funtionality. The action selection the user sees is dependent on the application that is being run, and is controlled by data within it. This file contains MUMPS code that controls the program flow, in a similar way to the Option file. Each action has help text associated with it, that can be reviewed from within the Desktop via the Help action. **** THIS FILE SHOULD ONLY BE MODIFIED BY AUTHORIZED PERSONNEL *****
