description
The WAM Data Set ID File contains Data Set ID (DSI) data set requirements that are used by the CHCS-WAM subsystem. This is a Class 2 (user modifiable) file. Data for this file is generated via the DA "Data Set IDs Create" option based upon the business rules for the following CHCS files: WAM Core, WAM Core Business Rules, and MEPRS Codes file. The WAM Data Set ID File defines the workload data that is collected under the WAM option and transmitted to the EAS systems. The data is defined in one main entity and two sub-entities. The main entity contains information about the DSI data and the associated organization that reviews the data. The main entity is uniquely identified by the combination DSI and DMIS ID. The two sub-entities defined the organizations that are involved in reporting the workload information. The two sub-entities are identified by the Performing FCC(s) and corresponding Requesting FCC(s). Information stored in the Data Set ID File is maintained for 12 months after inactivation date for historical reporting purposes.
