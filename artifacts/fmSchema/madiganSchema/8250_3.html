description
The HIC1TBL HIC2TBL HIC3TBL file is a consolidation of three files: 1) The HIC1TBL (Hierarchical Ingredient Code Organ Systems) 2) The HIC2TBL (HierarchicaL Ingredient Code Pharmacological Classes) 3) The HIC3TBL (Hierarchical Ingredient Code Specific Therapeutic Class) The Internal entry number (D0) is either one of the following made into a numeric value by concatenating each ascii character Numeric equivalent $A(). 1) The first byte of the Hierarchical Ingredient Code is the Hierarchical Ingredient Code Organ System (HIC1). 2) The first two bytes of the Hierarchical Ingredient Code is the Hierarchical Ingredient Code Pharmacological Class (HIC2). 3) The first three bytes of the Hierarchical Ingredient Code is the Hierarchical Ingredient Code Specific Therapeutic Class (HIC3). A description of each file follows: HIC1TBL - RHIC1D0 The Hierarchical Ingredient Code Organ System Table uses the first byte of the Hierarchical Ingredient Code as a key to generate a 50-byte description of the organ system involved. HIC2TBL - RHIC2D0 The Hierarchical Ingredient Code Pharmacological Class Table uses the first two bytes of the Hierarchical Ingredient Code as a key to generate a 50-byte description of the pharmacological class involved. HIC3TBL - RHIC3D0 The Hierarchical Ingredient Code Specific Therapeutic Class Table uses the first three bytes of the Hierarchical Ingredient Code as a key to generate a 50-byte description of the specific therapeutic class involved. Each ingredient has an associated specific therapeutic class, related to either the HICTBL. Each product entity (GCNSEQNOTBL) has an associated specific therapeutic class.

.01HIC(+)0;1FREE TEXTB The Hierarchical Ingredient Code Organ System (HIC1) is the first byte of the Hierarchical Ingredient Code. Example: "A" identifies the Cardiovascular System. The Hierarchical Ingredient Code Pharmacological Class (HIC2) is the first two bytes of the Hierarchical Ingredient Code. Example: "A1" identifies the Cardiac Stimulants (A1). The Hierarchical Ingredient Code Specific Therapeutic Class (HIC3) is the first three bytes of the Hierarchical Ingredient Code. Example: "A1A" identifies the Specific Therapeutic Class, Digitalis Glycosides.
.02HICDESC0;2FREE TEXTD If HIC is a one byte variable then HICDESC is a 50-byte description of the organ system involved. If HIC is a two byte variable then HICDESC is a 50-byte description of the pharmacological class involved. If HIC is a three byte variable then HICDESC is a 50-byte description of the specific therapeutic class involved.
