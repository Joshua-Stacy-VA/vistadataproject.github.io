.01FIELD0;1POINTER78.7BEnter the name of a print field that you want to appear on this flash card or label. This field is a pointer to the Label Print Fields file (#78.7) where the allowable print fields are stored. If you want to include the same print field twice on the flash card or label, then enter the print field name once without quotes and then again in quotes. If you enter FREE TEXT in this field, then you should enter text in the LITERAL TEXT field.
2ROW0;2NUMERICEnter the number (from 1 to 80) of the row on which the CHCS should start printing this field on the flash card or label. (Rows are horizontal.) Although the CHCS accepts a number up to 80, enter a number that corresponds with the size of the flash card or label.
3COLUMN0;3NUMERICEnter the number (from 1 to 80) of the column on which the CHCS should start printing this field on the flash card or label. (Columns are vertical.) Although the CHCS accepts a number up to 80, enter a number that corresponds with the size of the flash card or label. If you do not enter data in the TITLE(OPTIONAL) field, and if no default title is defined in the Label Print Fields file (#78.7), then the CHCS starts printing the actual field value at the column number you enter here.
4TITLE (OPTIONAL)0;4FREE TEXTEnter a 1- to 15-character title that should precede this print field (for example, SSN: or NAME:). You may include blank spaces in or after the title. If you leave this field blank, then the CHCS prints the default title (if one is defined) for this print field in the Label Print Fields file (#78.7). Enter NONE if no title is desired.
5LITERAL TEXT0;5FREE TEXTEnter 1- to 80-character text (for example, **Department of Radiology**) that should appear on the flash card or label. The number of characters in the text must not exceed the number of columns allowed on the flash card or label.
6SIZE0;6NUMERICEnter the font size (that is, the print size). Enter 1 for very small (generally not used), 2 for mediaum, and 3 for largest. This field is applicable to Intermec printers only.
7BARCODE THIS FIELD0;712Enter y for YES if the CHCS should print this field in barcode format. Enter n for NO if it should not.
