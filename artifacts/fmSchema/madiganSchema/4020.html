description
This file allows a message (via a transaction type) to be replicated to multiple transaction type/destinations. The "AC" cross-reference (on the ORIGINATION TRANSACTION TYPE, .02 field) is used to identify all of the transaction types to which the originating transaction type is to be replicated. If fields in node 2 are defined, or user-defined formatting is specified (node 1), a new MSH segment will be created and attached to the base message being replicated for the destination. Otherwise, the base MSH segment will be used for the destination.
