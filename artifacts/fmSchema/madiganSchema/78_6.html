3IS THIS A ROOM GROUP?(+)0;312Enter Y for YES if this room is a room group. Enter N for NO if this room is not a room group. A room group is defined as a group of rooms that are used for the same or similar purposes when scheduling procedures. For example, if there are three rooms in which the same procedures are performed, and the scheduling of these procedures could take place in any of the three rooms, these rooms could be classified as a group. When the system searches for a room in which to schedule the procedure, it searches all three rooms for open times and schedules accordingly. An example of a room group would be RAD RM GRP. This room group could include RAD RM 1, RAD RM 2, and RAD RM 3.

5CROSS-DIVISION SCHED0;612This field determines whether one division can schedule appointments in another division at your MTF. Data should be entered in this field if your MTF is multidivisional. If this field is set to YES, then you must enter data in the following fields: CROSS-SCHED DIVISION, CROSS-SCHED DAYS, CROSS-SCHED START TIME, and CROSS-SCHED END TIME. Enter N (for NO) if you do not want the CHCS to allow cross-divisional scheduling at your MTF. Set this field to NO if your MTF is not multidivisional. If you leave this field blank, then the CHCS defaults to the NO setting.
