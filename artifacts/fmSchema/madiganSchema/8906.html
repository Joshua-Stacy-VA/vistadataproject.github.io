description
This is a list of files that have restricted queueing times associated with them. If a file is listed here, any print or search of it will be forced to queue, and the time of queueing may be restricted by day of week and time of day. This can be used to force large file searching jobs to run during non prime hours when an ad hoc report is run against them. Auditing should be turned on for this file.
