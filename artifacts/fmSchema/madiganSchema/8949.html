All opto files in the archive system have a multiple field named ARCHVEAPPENDEDTEXT. File 8949 and its subfile 8949.01 serve as a template for the creation of the ARCHVEAPPENDEDTEXT multiple in each of the opto files. This file is not intended for the storage of data. It is only used during the creation of new opto files.
