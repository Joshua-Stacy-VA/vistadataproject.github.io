description
The DG ICU HOURS REPORT File holds data for the Hours of Service In Intensive Care Units By Referrring MEPRS Clinical Service Report, commonly referred to as the ICU Hours Report. Variations of this report can be run from the PAD>IRM>ICU and PAD>IRM>WLR options. The amount of data stored in the file will be limited for two reasons. First, the ICU option will only permit a single "ICU" record to be written to the file at a time. Second, the Worldwide Workload functionality will ensure that only the last 18 months of data for a given division is retained in the file. As a result, a given division will have a maximum of 18 monthly "WW" records within the file at any given time.
