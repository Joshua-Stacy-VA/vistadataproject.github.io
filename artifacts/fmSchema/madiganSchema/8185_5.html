description
The WAM Data Set Exclusions file contains the list of DMIS IDs to be excluded for a DSI/Performing FCC/DMIS combination during the WAM Data Set ID creation process. Prior to the creation of the Data Set ID file for the new fiscal year, the authorized user must first access the option Data Set Exclusion Enter/Edit to exclude any DMIS IDs as Requesting Functional Cost Codes (FCC). CHCS will screen out standard Data Set Ids (i.e., one with no Performing FCC). The WAM Data Set Exclusions file is a Class 2c file structure that will be shipped to the sites. This file will not be populated with data as it is site defined.
