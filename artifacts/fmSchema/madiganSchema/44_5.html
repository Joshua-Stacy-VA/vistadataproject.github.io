.01NAME(+)0;1FREE TEXTBYou may only enter non-standard non-medical appointment type. The 2-5 character abbreviation/code/name is linked to the appointment type description. These appointment type codes are what normally appear on the booking screens. Examples include XRAY, MRI and PERIO.
1DESCRIPTION(+)0;2FREE TEXTCThe 3-30 character complete description of the appointment type and what it is used for. Examples include: New Patients Only, Procedures 30 minutes, and Follow-up appointment.
2ARE APPT SLOTS SEARCHABLE?0;312A Yes/No flag that indicates whether an appointment type will be displayed when searching for open appointment slots. The system appends an asterisk to appointment type name. Non-searchable appointment types such at MTG*, Lunch*, Class* are often utilized to indicate non direct patient care activities such as Meetings, Lunch or Classes. Non-searchable appointment are displayed in the (B)rowse booking action.
3STATUS(+)0;4SET OF CODES1:ACTIVE
0:INACTIVE
A flag that indicates whether an appointment type is active or inactive. Values include: 1 for Active; 0 for Inactive. If the status field is inactive at the Facility Profile level, the appointment type is inactive for all clinics and providers regardless whether the STATUS: field is active in the clinic or provider profiles. The order of precedence of the status field is Facility is greater than Clinic which is greater than Provider.
4CATEGORY(+)0;5SET OF CODESI:INITIAL
F:FOLLOW-UP
A flag that indicates whether the appointment type is for a new or existing patient. Values include I for Initial/New; F for Follow-up/Return.
5MEDICAL APPOINTMENT TYPE0;612Set this field to YES for all medical clinic standardized appointment types to be used when creating/editing templates and schedules and booking appointments. Set this field to NO for all nonstandard Dental appointment types in dental clinics that have a Location Type of 'D' (Dental) and where the DDSA software is being used for appointment booking.
6NON-MEDICAL APPOINTMENT TYPE0;712Non-Medical Appointment type is a flag indicating that the appointment type can be used by the clinic with MEPRS code starting with 'C' or 'DCA'.
