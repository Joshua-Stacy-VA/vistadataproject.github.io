.05ACCESSION STATUS(+)0;5SET OF CODESI:INTERMEDIATE
F:FINAL
A:AMENDED/UNIT STATUS UPDATED
This field contains the actual status for this accession when DBSS sends an update to the unit status information. This allows CHCS to appropriately note that the unit status data was updated by DBSS while the accession was either intermediate or finalized. Each unit status update that is transmitted after the accession is finalized is considered an amendment by CHCS.
