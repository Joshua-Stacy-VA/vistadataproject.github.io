5REPORT STATUS(+)0;5SET OF CODESV:VERIFIED
A:AMENDED DRAFT
PD:PROBLEM DRAFT
D:DRAFT
This field defines the status of this report. The CHCS automatically enters the code in this field depending upon actions taken by the transcriptionist and approving radiologist. NOTE: the exam status of PRELIMINARY can apply to all report statuses depending upon the report being returned to the resident for signature. V = Verified: The report has been signed, and the radiology exam status equals COMPLETE or PRELIMINARY. AD = Amended Draft: The report has been amended and is awaiting re-verification. The radiology exam status equals AMENDED or PRELIMINARY. PD = Problem Draft: The report has been sent back to the transcriptionist or resident for re-editing. The radiology exam status equals DICTATED or PRELIMINARY. D = Draft: The report has been transcribed and/or verified by a resident. The radiology exam status equals TRANSCRIBED or PRELIMINARY. If this field is set to VERIFIED or DRAFT, then the report may be available for display outside the radiology department, depending upon additional radiology-specific parameters located in the Radiology Overall System Parameters. If this field is set to AMENDED DRAFT or PROBLEM DRAFT status, the report can only be displayed in the radiology department.
