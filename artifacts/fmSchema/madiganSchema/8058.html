description
This file contains Ancillary (Lab, Rad and PHR) transactions triggered by the certification (Lab), verification (Rad), and label print (PHR) events. These transactions are held for 3 days and, on the 3rd day, the transaction is either a) placed in an ASCII file with other identifying information and sent to TPOCS if the associated patient is TPC billable, b) used to create an MSA itemized outpatient account,if the associated patient is agency billable, or c) used to create an I&R account. All entries are purged from this file on the 3rd day following the date of service. This file is maintained by Lab, Rad and Pharmacy triggers. DO NOT EDIT THIS FILE!!
