description
Enter or edit the pharmacy information that will be transmitted to the VA CMOP with authorized refills. The VA CMOP interface can be activated at the Group DMIS ID level. . Each Group DMIS ID can define a single refill pharmacy to interface to the VA CMOP. Each pharmacy will be referred to as a CMOP Group Pharmacy (i.e., there could be several CMOP Group pharmacies defined at each host). Some of this information - such as the division pharmacy name and address - will appear on the label that the VA CMOP will use for dispensing the prescriptions.
