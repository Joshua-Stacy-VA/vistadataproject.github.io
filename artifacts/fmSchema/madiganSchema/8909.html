description
This file contains the eligibility and registration responses from DEERS. Eligibility responses can be for a single person or a family response if DEERS finds a duplicate sponsor SSN, multiple patient DOBs, or no patient DOB matches. For each individual returned on a family response, a single entry is written to this file. Registration responses are always only for a single person.
