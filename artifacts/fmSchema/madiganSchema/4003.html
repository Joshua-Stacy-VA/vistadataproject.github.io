name
INTERFACE ERROR
number
4003
location
^INTHER(
description
This file is used to store errors that occur during the operation of the interface. Errors that occur in any component of the GIS are stored here. These errors can also be stored in the Universal Interface File if the error occurred while processing a message. If the error was a MUMPS interpreter error it will also be stored in the error trap.
