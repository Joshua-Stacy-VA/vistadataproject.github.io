name
SPOOL DOCUMENT
number
3.51
location
^XUSPLDSM(
description
This file holds the name of "Spool Documents" created by ^%ZIS for DSM systems. It does NOT hold the text of the documents themselves. That text is spooled out to the spool space. File 3.51, however, "holds" spool space so that it can only be deleted by deleting the entry in this file.
