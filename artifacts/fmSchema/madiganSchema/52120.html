description
This file contains the exception patients from the CHCS database where the MPL Extractor was run. This file is created by the Extractor routine and data will not be inputted by the user.

The gender of the patient and is either M for Male and F for Female and Z for unknown. This is a required field on CHCS and will place a patient's record on the Exception report if blank. This is not a currently required field for this file.
.03DOB0;3DATE-TIMEThis field contains the Date of Birth of the Patient. It is a required field on CHCS. If it is blank, it is placed on the Exception Report. Imprecise DOB on CHCS will be Blanked out and sent to the ASCII file and will not appear on the Exception Report. This field is not required for this file.
.09SPONSOR SSN0;9FREE TEXTThe Social Security Number (SSN) of the patient's sponsor. This is a required field on CHCS and if blank for either a sponsor or dependent will place that patient on the Exception Report. This is not a required field for this file.
.17PATIENT SSN0;17FREE TEXTThe Social Security number of the Patient. If the patient is his own Sponsor it will be the same as the Sponsor SSN. This is a newly required field in CHCS so many patients on the database will not have a patient SSN. The Extractor routine will not place patients without Patient SSN onto the exception report. This is not a required field for this file.
