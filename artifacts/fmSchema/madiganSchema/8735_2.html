description
This CLASS 1 file will be used to store various mathematical, relational, and boolean operators to be used during Lab Rules building and processing. Entries in this file can only be added by software update. This is to ensure adequate analysis and evaluation as been done to make sure the operator makes sense and performs correctly when used within a rule.
