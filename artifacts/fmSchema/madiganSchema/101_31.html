The DAR code is the unique identifier for the delayed action. It is used asa reference in program in-line documentation and as a parameter in DAR function calls.
.02STORED VARIABLES1;0WORD-PROCESSINGThe variables needed for future activation are stored here in the format: ^ORB(ORNUM,2,DAR LOG#,1,n,0)=varnam_"^"_data There is a possibility that the length of the variable name plus its data would be greater than 255. The format for such a case: ^ORB(ORNUM,2,DAR LOG#,1,n,0)="+"_varnam_"^"_$E(data,1,200) ,n+1,0)=$E(data,201,999) The plus (+) in line n indicates to the unstacking function that line n+1 should be concatenated to line n data.
