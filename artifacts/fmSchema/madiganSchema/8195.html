description
This file contains the patient identification code and related description used to differentiate between newborns, delivery cases and all other cases. It is required for all cases at Air Force MTFs. The first liveborn is denoted with an A, the second liveborn with a B, and so on up to the letter E. SOURCE: DMSSC, DATA ADMINISTRATION CONTRACT, MDA903-86-C-0215, FINAL ADDENDUM TO THE AQCESS/CHCS FUNCTIONAL DESCRIPTION FOR THE STANDARD REQUIREMENTS FOR THE INPATIENT DATA RECORD FOR TASK ORDER NO. DAT2, 12 AUGUST 1988.
