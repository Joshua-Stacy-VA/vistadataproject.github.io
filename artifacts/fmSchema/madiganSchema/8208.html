description
This file contains the Pharmacy MEPRS information. ********************************************** **** DO NOT UPDATE THIS FILE WITH FILEMAN **** ********************************************** To create/update the records of this file use only the PUTMEPR^PSMPRU subroutine. In each record of this file only one of the filling site (OUTPATIENT SITE, INPATIENT SITE, IV ROOM OR VAULT) fields is set. For any combination of DATE, REQUESTING DIVISION, MEPRS, TRANSACTION and filling site, there will be one and only one record. The file contains the following cross reference: ^PSM(8208,"AC",DATE, site field#, filling site, REQUESTING DIVISION, MEPRS, TRANSACTION, DA)="" where: if site field# is 6 then filling site is OUTPATIENT SITE if site field# is 7 then filling site is INPATIENT SITE if site field# is 8 then filling site is IV ROOM if site field# is 9 then filling site is VAULT
