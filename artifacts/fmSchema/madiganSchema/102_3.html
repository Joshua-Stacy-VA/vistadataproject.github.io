.01NAME(+)0;1FREE TEXTBEvery order requires a priority. The priority determines the processing importance of an order. The priority comes from a standard file and is not editable by the MTF. All priorities, except ROUTINE, will appear as part of the order display on the Patient Order List and Due Lists. Not all priorities are available for all order types. The NOTIFY AND ROUTINE priority is only available for the RAD and LAB order types and a message is triggered when results are available for viewing. The STAT priority triggers a QA event if there is no task response within the time range defined in the site parameter. The priority names available are: STAT ASAP PREOP NOTIFY AND ROUTINE ROUTINE
.02SHORT DESCRIPTION0;2FREE TEXTThis field is used to enter a brief description of this priority.
.03ORDER TYPE SCREEN0;3FREE TEXTThis field is used to identify which order type will utilize this priority. By leaving this field blank, the order priority can be used by all of the order types. If an entry is made, only the order types entered will utilize this priority. Separate and surround order types with a "," as follows: Enter ,5, when only the Radiology order type utilizes this priority, or Enter ,4,5, when both the Laboratory and Radiology order types apply.
