number
3.98

description
This file contains domain names or mailboxes that are prevented from accessing network mail services. An entire domain or mailbox can be blocked out, or they can be blocked out by specified users. In addition, a domain can be locked with a security key. If this capability is used, then a user must hold that security key in order to send to that domain.
