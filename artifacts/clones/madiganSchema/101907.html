description
This is the parameter file for FastFill. FastFill is an outpatient pharmacy pill counter, similiar in function to Baker Cells. This file is referenced by the KIFASTF routine which is called by custom print templates for the Prescription file.
Fields
#NameLocationTypeDetailsIndexDescription

.02active0;2SET OF CODES0:INACTIVE
1:ACTIVE
Defaults to Active. Set to Inactive if the FastFill becomes unavailable. Setting to Inactive will speed up batch prints. The FastFill interface will try for ten seconds to open a FastFill device if it is not marked as Inactive.
