description
This file contains the parameters for customizing the operation of the Workload Assignment Module (WAM) for a given division. No WAM functional parameters will be in effect unless the WAM SWITCH field within the file is set to on for the given division.
Fields
#NameLocationTypeDetailsIndexDescription
.01DIVISION(+)0;1POINTER40.8BDivision for which the WAM software configuration information is being established.
.02WAM SWITCH0;212Indicator as to whether the WAM Workload Assignment Module (WAM) is available to the referenced division.
.03WORKLOAD DEVIATION FLAG0;312Indicator for whether the value of the WORKLOAD DEVIATION field is set and should be used.
.04WORKLOAD DEVIATION0;4NUMERICNumeric field which tracks the site-specified acceptable percentage deviation of workload amount between the current and previous reporting month. A value between the range of 15 (%) and 25 (%) can be entered into the field.
.05DATA GENERATION FREQ FLAG0;512Indicator as to whether the value of the DATA GENERATION FREQ field is set and should be used.
.06DATA GENERATION FREQ0;6NUMERICNumeric field which indicates the frequency in days of workload processing utilized to provide the default data in the WAM editing template. Only a value between 7 to 15 (days) can be entered into the field.
.07DELINQUENCY ALLOW DAYS FLAG0;712Indicator as to whether the value of the DELINQUENCY ALLOW DAYS field is set and should be used.
.08DELINQUENCY ALLOW DAYS0;8NUMERICNumeric field which tracks the number of days after the initialization of the WAM workload data that the editing of the workload data should be considered delinquent.
.09DELINQUENCY BULLETIN FLAG0;912Indicator for whether the workload reporting delinquency bulletin should be automatically generated.
