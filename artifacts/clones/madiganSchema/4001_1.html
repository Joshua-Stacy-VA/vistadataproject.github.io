name
INTERFACE CRITERIA
number
4001.1
location
^DIZ(4001.1,
description
This file is used to store the criteria used when running searches of the Universal Interface file and the Interface Error file. The Interface Test Utility stores criteria for running interface testing here also. Interface Criteria entries can be saved by the user and standard entries can be made for all to use.
