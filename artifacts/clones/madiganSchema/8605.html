description
This file contains the workload data associated with each radiology exam. The CHCS automatically enters the workload data into the fields in this file and later outputs the data on workload reports generated via the Workload/MEPRS Reports Menu (in the RAD subsystem). The RAD Workload Data file was created as part of the CPT development effort (CHCS Version 4.31/MU2).
