description
This file is a fixed file of the operations and check code which may be specified by the developer of an action bar. It should not be changed by anyone. The internal entry numbers for an operation must be less than one hundred, and the internal entry numbers for a check must be one hundred and greater. The action bar routines do not depend on this file for the presentation of an action bar. This file is used when defining a new action bar.
