description
This file contains one entry for each dining hall collection posted through the dining hall collections function of cashier processing. Date Collected is the date of the actual collection. Date posted is the date the entry is/was made. The surcharge amount is divided in half and stored in Surcharge Amount and Surcharge 2 Amount. If the amount is not an even penny, the extra penny is accrued to the Surcharge Amount.
