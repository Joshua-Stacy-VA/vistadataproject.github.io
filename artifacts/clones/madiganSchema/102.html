name
ORDER TYPE
number
102
location
^ORD(1,
description
This file holds the names and programming information about each of the ancillaries with which Order Entry is currently communicating. This file is NOT editable by the MTF and is really a part of the parameterization of the system. The full text of the order type may be changed without adversely affecting the system.
