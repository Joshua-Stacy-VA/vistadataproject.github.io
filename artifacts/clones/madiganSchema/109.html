description
This file contains information concerning what orders are on each page of the Patient Care Plan. The `AAPVPAST' cross-reference contains APV pages with Page Status = PAST Patient # , Date/time page status changed to PAST , Page # ) = "" The `ACURPAGE' cross-reference gives the current page for each inpatient: Patient # , Page # of the Current Page ) = "" The `ADEFPAGE' cross-reference is the default page to sort on or store to. The variable ORPAG("PAG") will be set to this when the patient is first accessed for an order entry session. Patient # , Page # of the current Default Page ) = "" *** THIS X-REF WILL BE SET ONLY UNDER PROGRAM CONTROL, NOT THRU FM! *** The `AEPI' cros-reference gives the orders in an episode: Patient # , Episode # , Order # ) = Order Type NOTE: The Outpatient orders are under episode 0. The `AORDER' cross-reference gives each page that an order appears on: ORDER # , Page # ) = "" The `AOUTPAGE' cross-reference give the outpatient page for each patient: Patient # , Page # of his/her Outpatient Page ) = "" The `APAGE' cross-reference gives the orders for a page referenced by: Patient # , Episode # , Page # , Order Type , Order # ) = Order Status The `AZUTL' cross-reference is a temporary global: Page# of the current Default Page (ADEFPAGE) ) = Order Information The 'AUSERPAGE' cross reference is used as page locking mechanism: Current Page #, number users have current page locked)=""
