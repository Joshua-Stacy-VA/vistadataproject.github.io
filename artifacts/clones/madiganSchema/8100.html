description
This file contains codes and descriptions of the specialties that a provider of patient services can have. For example: 110 - UROLOGIST 207 - PATHOLOGY RESIDENT. A provider can have more than one specialty. The primary specialty must be entered first. This is a standard file (classified as Group 1) and cannot be modified by the site. SOURCE: DMSSC, DATA ADMINISTRATION CONTRACT, MDA903-86-C-0215, FINAL ADDENDUM TO THE AQCESS/CHCS FUNCTIONAL DESCRIPTION FOR THE STANDARD REQUIREMENTS FOR THE INPATIENT DATA RECORD FOR TASK ORDER NO.
