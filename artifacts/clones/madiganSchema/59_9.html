description
This file contains the pharmacy clinical screening parameters. Clinical screening parameters are used to determine the level of checking of Rxs, Meds (Unit Dose) , and IVs orders for drug warnings. The clinical screening parameters are system wide. There is one and only one record in the file. Its D0 value is 1.

.04SEVERITY0;4SET OF CODES1:SEVERE
2:MODERATE
 This parameter allows you to limit which interactions are generated by the system. Enter a '1' to include only severe interactions or enter a '2' to include all interactions, both severe and moderate. The severe and moderate interactions are related to how First DataBank ranks interactions. The following table describes how the parameter relates to First DataBank's ranking of interactions. Severity Parameter First DataBank's Ranking --------------- ------------------------ Severe Severe interactions, both well documented and not well documented. Moderate Moderate interactions that are well documented. If the parameter is set to moderate (2) ALL interactions will be generated. The system will not allow the generation of only moderate interactions.
.05OVERLAP-DEA0;5SET OF CODES1:SCHEDULE 1
2:SCHEDULE 2
3:SCHEDULE 3
4:SCHEDULE 4
5:SCHEDULE 5
9:*** DISABLE CHECK ***
This parameter sets at what level the system will perform DEA overlap checking. For example, if this parameter is set to SCHEDULE 3, the system will generate DEA overlap warnings for drugs defined in the system as DEA schedule 1, 2, or 3. If set to ***DISABLE CHECK***, the system will not perform any DEA overlap checking. No entry will default to SCHEDULE 5, the maximum setting.
.06OVERLAP-THERAPEUTIC CLASS0;6SET OF CODES0:ENABLE
1:*** DISABLE ***
This parameter enables or disables all therapeutic classification overlap checking. No entry in this field will default to ENABLE. The system will check First DataBank's therapeutic classification of all the drug's active ingredients as well as the therapeutic classification of the entire drug product. For example, Actifed with Codeine has the following three ingredients and their respective classifications: Ingredients Classifications --------------- --------------- Codeine Analgesics, Narcotic Tripolidine Antihistamines Pseudoephedrine Sympathomimetic agents Any drug that contains ingredients with the above classifications has the potential to cause an overlap warning. In addition, Actifed with Codeine is classified as a Cough and Cold preparation. Any other drug product with an identical classification would also produce an overlap warning. Overlap checking will take into account the drug product's route when performing overlap checking. Both drugs must have either systemic routes or identical non-systemic routes for overlap checking to be performed.
.07DUPLICATION CHECK0;7SET OF CODES0:ENABLE
1:*** DISABLE ***
This parameter enables or disables all duplication checking. No entry will default to ENABLE. A duplicate warning occurs whenever identical drugs or drugs that reference the same drug product are prescribed for a patient. For example, if an HCP prescribed identical drugs to a patient a duplicate warning would be generated. If two different drugs referencing the same drug product (same generic name, route, strength and form) via the Primary NDC Number were prescribed to a patient, a duplicate warning would also be generated. If a duplicate warning is generated, no overlap checking will be performed. However, if the duplicate checking parameter is disabled, the system will continue to perform overlap checking and generate overlap overlap warnings against identical drugs or drugs belonging to identical drug products.
.08ALLERGY CHECK0;8SET OF CODES0:ENABLE
1:*** DISABLE ***
This parameter enables or disables all allergy checking. No entry will default to ENABLE. There are three ways an allergy warning can be generated. First, a patient is allergic to a drug and an ingredient of that drug is ordered for the patient. Second, a patient is allergic to a drug that belongs to an allergy class and a drug belonging to the same allergy class is ordered for that patient. Finally, the patient is allergic to an allergy class and a drug belonging to the same allergy class is ordered for that patient.
.09DOSAGE CHECK0;9SET OF CODES0:ENABLE
1:*** DISABLE ***
This parameter enables or disables dosage checking. A blank parameter is the same as ENABLE.
