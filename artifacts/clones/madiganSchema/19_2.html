description
This file contains the different groups of security keys for the MTF. Groupings of these keys may be listed in the security key file (#19.1) and used for bulk key assignments when a site or specific functionality is bein implemented. For example, if a Scheduling supervisor should hold both a manager key and all keys that a scheduling clerk, keys normally assigned to scheduling clerks would be flagged as both a supervisor and clerk key in the security key file.
