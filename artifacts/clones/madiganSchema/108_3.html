description
This file holds the relatively small number of ADT procedures the HCP can order for a patient. The routine ORADT will process the order with reference to field #3 in the file, which provides further reference to specific code for order processing.

This field contains a set of codes (required). Enter one of the abbreviations for the types of ADT orders, i.e. "ADM" for ADMISSION,"TRNI" for TRANSFER (INTERWARD), "TRNA" for TRANSFER (ABSENCE), "DIS" for DISPOSITION, and "TRNP" for TRANSFER (PASS).
