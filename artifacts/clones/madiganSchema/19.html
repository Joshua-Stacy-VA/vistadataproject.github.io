1920requires user input1916;212This field is used to indicate whether or not this option requires a user's input interaction. The purpose of the 'REQUIRES USER INPUT' Option field is to prevent this class of user input required options from being accidently tasked to run, when the particular option should only be used interactively. For example: A Menu Path Choice option type should not be tasked off to run. The value of 'YES' in this field indicates that the option requires some user interaction for it to be completed. The value of 'NO' in this field indicates that a user's input is not required, and therefore that this option can be tasked off to be run without any further user input or monitoring.
