1UNIT STATUS HISTORY1;0MULTIPLE63.8This multiple field contains an audit trail of updates made to any of the unit status fields. When the first unit status is filed on CHCS, it will be filed into the Unit Status field. DBSS will transmit any change in unit status to CHCS. The current unit status will file in the Unit Status field, and the previous unit status result will be moved into the Unit Status History fields. Both the current Unit Status and the Unit Status History will display in CHCS inquiry and reporting options.
