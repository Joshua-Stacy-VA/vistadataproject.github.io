description
The Lab Host Platform Parameters file is used to identify the single CHCS platform used by all users in all divisions on the single box which hosts CHCS for the sites. There will only ever be a single entry in this file. The file will contain data which applies to all divisions, work elements, laboratories, and users for this CHCS platform.

4lab test std mail group0;4POINTER3.8The Lab Test Standardization Mail Group allows qualified users to be notified when a Miscellaneous Lab Test is re-defined in the CHCS Lab Test file. Miscellaneous Tests are non-standard tests and should be evaluated for subsequent addition to the DOD Standard Lab Test file. At the time a Miscellaneous Test is re-defined and filed, a bulletin will be generated to members of the Lab Test Standardization mail group. The bulletin will indicate that a Miscellaneous Test has been re-defined and how many Miscellaneous Tests remain which may be re-defined for the Lab Host Platform. When the mail bulletin is generated, a user who is qualified to evaluate the test for standardization will be made aware that a test has been re-defined in the CHCS Lab Test file, is now a non-standard test, and should be submitted for evaluation. Members of the Lab Test Standardization Mail Group are entered in the Mail Enter/Edit options.
