.01result0;1FREE TEXTThis field contains the answer to the test performed. The result field is free text and contains the result transmitted from the DBSS. All test results transmitted from DBSS to CHCS are certified. Results for only certain single tests are stored in this multiple. Results for positive Antibody Screen which includes Antibody ID and/or Antibody Titer and results for unit status are stored in separate multiples.
