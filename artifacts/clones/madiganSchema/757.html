description
This file is a map of Major Concepts within the Clinical Lexicon to both the expression file (#757.01) and the Unified Medical Language System (UMLS). While the primary purpose of this file is for file maintenance, it also supports various other functions such as the display of classification codes by linking concepts to codes and the ability to filter out unwanted search responses by linking concepts to semantic classes and types. NOTE: Data Entries in this file should not be altered by the site.

.01expression(+)0;1POINTER757.01BPointer to the clinical expression in the Expression file (#757.01) which represents the preferred term for the Major Concept. The expression for the Major Concept is either taken from the Unified Medical Language System (UMLS) Metathesaurus or one of several DHCP controlled vocabularies distributed by an authoritative software package produced at a VA Information System Center (ISC) and will represent a unique clinical concept.
1usage(+)0;2NUMERICThis number represents the frequency of usage of a concept. Each time a Major Concept is selected during look-up, the usage value will be incremented by 1. In future releases, the look-up routine will use this value to order the selection list, placing more frequently used expressions at the top of the list.
