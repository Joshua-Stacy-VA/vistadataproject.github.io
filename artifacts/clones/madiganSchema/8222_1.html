6.01bar code scanner interface6;1SET OF CODES0:DISABLED
1:ENABLED
This field determines whether or not a prescription label bar code scanner is interfaced with the TransLux DataWall. If this field is set to ENABLED, then the patient name for new prescriptions will display on the TransLux DataWall when the prescription label bar code is scanned using the TransLux Scan In Rx Label option. For refills, the patient name for the prescription will display if the DISPLAY REFILLS INDICATOR field is set to YES. The patient name will be removed from the TransLux DataWall when the prescription label bar code is scanned using the Scan Out Rx Label option. Also, when this field is set to ENABLED, the printing of a label no longer interacts with the TransLux DataWall. The TransLux DataWall can only be updated via the TransLux Scan In Rx Label option. If the field is set to DISABLED, then the patient name will display on the TransLux DataWall when the prescription label is printed.
6.02display refills indicator6;212This field indicates whether or not the patient name for prescription refills should display on the TransLux DataWall. If the field is set to YES, and the BAR CODE SCANNER INTERFACE field is set to ENABLED, then the patient name for a refill will be displayed on the TransLux DataWall. If the field is set to NO, then the patient name for a refill will not be displayed on the TransLux DataWall.

This field determines whether or not a record is created of prescriptions not picked up by patients.
