.01RECORD INDEX FILENAME0;1FREE TEXTB This field contains the name of the record index filename which the shipment data file is associated with. The name represents the ASCII filename of the record index and is displayed in the following format: XXXX999999999999YYZZZZZZZZZZZZ.R01 for the Record Index XXXX = The four (4) character DMIS ID Code of the facility retiring the records. 999999999999 = The twelve (12) digit internal entry number of the fileroom in the Borrowers file (includes leading zeros). YY = The two (2) character abbreviation of the record type being retired. This abbreviation is derived from file 8393 - STANDARD RECORD TYPES. ZZZZZZZZZZZZ = The twelve (12) digit of the current year,month date/time (when the Shipment Data File and Record Index File were created, Example: 200506241000 for 24 June 2005@1000). R = Indicates the record index S = Indicates the Shipment Data file 01 = An incremental number associated which acts as a counter for the index or SDF. This number will turn over after it reaches 99 and start with 01 again. As generated on CHCS the filename is prefixed with MRRS, e.g.: MRRS0124000000000003OP200506241327.R01 The SY-ETU utility strips the MRRS during the transmission to the NPRC.
