description
This file contains information for the I&Rs to be printed as a result of the nightly fun. This will be I&Rs on newly dispositioned accounts that have not had a current I&R printed, and month end I&Rs on accounts with status 'S'. These I&Rs are printed using the I&R batch print option. The file is cleared out using a special option.
