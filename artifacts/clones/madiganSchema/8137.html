description
This file contains codes and descriptions for injuries. It defines the type of injury sustained by the patient. The category field points to another file - CAUSE OF INJURY CATEGORY (8145). This field categorizes an injury by the method in which an injury occurred. The codes contained in this file are Standard NATO Agreement (STANAG) codes. This file is a CHCS standard file (classified as Group 1) and cannot be modified by the site. SOURCE: DMSSC, DATA ADMINISTRATION CONTRACT, MDA903-86-C-0215, FINAL ADDENDUM TO THE AQCESS/CHCS FUNCTIONAL DESCRIPTION FOR THE STANDARD REQUIREMENTS FOR THE INPATIENT DATA RECORD FOR TASK ORDER NO. DAT2, 12 AUGUST 1988.
