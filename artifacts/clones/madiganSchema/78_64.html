This field determines the days on which appointments for this room are available for booking by other divisions. Enter the number corresponding to the day that cross-divisional scheduling should be allowed for this room. If you leave this field blank, then the CHCS does not permit cross-divisional scheduling for this room. NOTE: This field is used in conjunction with the CROSS-SCHED START TIME and CROSS-SCHED END TIME fields, which define the times during which appointments are available for cross-divisional scheduling on the days designated in this field.
.02CROSS-SCHED START TIME0;2FREE TEXTEnter the earliest time that appointments for this room are available for booking by other divisions. Enter the time in four- digit military format (for example, 0800, 1300). If you leave this field blank, then the CHCS does not allow cross-divisional scheduling for this room on this day.
.03CROSS-SCHED END TIME0;3FREE TEXTEnter the latest time that appointments for this room are available for booking by other divisions. Enter the time in four-digit military format (for example, 0900, 2300). If you leave this field blank, then the CHCS does not allow cross-divisional scheduling for this room on this day.
