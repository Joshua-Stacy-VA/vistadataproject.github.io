description
This file contains the codes and descriptions for categorizing the major trauma types. Codes 0-1 represent battle wound or injury 2-4 represent intentionally inflicted nonbattle injury 5-9 represent accidental injury This is a standard file (classified as Group 1) and cannot be modified by the sites. SOURCE: DMSSC, DATA ADMINISTRATION CONTRACT, MDA903-86-C-0215, FINAL ADDENDUM TO THE AQCESS/CHCS FUNCTIONAL DESCRIPTION FOR THE STANDARD REQUIREMENTS FOR THE INPATIENT DATA RECORD FOR TASK ORDER NO. DAT2, 12 AUGUST 1988.
