description
PEMMSTR - RPEMMA0 The Patient Education Master Table uses the 6-byte Patient Education Code as a key to provide the drug or drug class involved. There is a two-line educational message to print out for the patient on the prescription label. Pharmex, and ASHP patient education modules are all referenced. The Master Table contains 4-byte monograph code fields that reference the First DataBank Patient Education Monograph Tables. The Internal entry number (D0) is the Patient Education Code (PEC) with leading zeros removed. If Zero the Internal entry number becomes 999999.

.01pec(+)0;1NUMERICPatient Education Codes (PEC) are used to support the creation of educational information to be used to help the patient to better understand the medications being taken. The Patient Education Code is used in conjunction with the master table and monograph file which exist in separate files. This data element and related files have application in outpatient and community pharmacy systems. Each drug record carries a maximum of one advisory, which is six characters in length. The patient education code has been assigned by First DataBank on the basis of the generic entity, varying by drug product, dosage, route and/or strength. If users want to maintain the patient education data at a more generic level than the government-regulated code (NDC or Canadian DIN), they are advised to relate the patient education code to the generic code number GCN Sequence Number (GCNSEQNO).
