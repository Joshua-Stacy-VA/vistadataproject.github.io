description
The 8037 WORLDWIDE WORKLOAD PROBLEM File will be used to hold information about "problem patients" that will not appear on the monthly Worldwide Workload Report because of various problems such as an invalid MEPRS code or a missing register number or patient category code. A Worldwide Workload Exception report will be able to be generated on demand from the 8037 file information retained in the ^DG global. Each time the Worldwide Workload Report is recalculated, the problem information recorded for the associated month will be updated.
