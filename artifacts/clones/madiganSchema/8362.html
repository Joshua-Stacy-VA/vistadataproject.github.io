description
This file is uneditable via FileMan. This file contains the monthly historical workload data for the facility. The "ASITE" x-ref contains the 3-letter abbreviation for the site. This abbreviation is used as part of the VMS file name when the report is created. E.G. ^SYWL(8362,"ASITE","KNX")=""
