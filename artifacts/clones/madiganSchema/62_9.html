description
The Lab Category file contains the category the Lab Sections are in for the MEPRS/UCA reporting. The categories are defined so the lab can define the three main categories of Clinical Pathology, Anatomic Pathology and Blood Bank. If certain lab tests are performed by departments other than the laboratory, the other categories (X-RAY, Blood Gas, RIA, etc.) can be added to this file and assigned to the Lab Section.
