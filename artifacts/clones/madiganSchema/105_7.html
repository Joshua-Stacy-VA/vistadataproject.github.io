description
This file is used as the menu selection driver for determining the appropriate Nursing Care Hour and Personnel Requirement Tables. When a user selects the 'Acuity Selection' option a program does a 'DIC' lookup on the entries in this file. The user then selects an entry in this file and the program notes which table to use in its final calculations. The .01 field is used as the menu synonym. The .02 field as the menu description.
