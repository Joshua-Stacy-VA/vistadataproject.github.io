description
This file stores the discrepancy records for the EWPD-CHCS interface. This file has been renamed from CCQAS to EWPD to accurately reflect the interface.

.01PROVIDER EDI PN(+)0;1FREE TEXTBThis field stores the EDI_PN (Electronic Data Interchange_Person Number) value that uniquely identifies each provider. This field is assigned to each provider by DEERS and DEERS is the only system authorized to change this value for a provider. An update may occur if providers are merged or otherwise re-assigned an EDI_PN value. This field should never be modified on CHCS. Doing so will cause problems with properly identifying and linking other data fields to a provider.
