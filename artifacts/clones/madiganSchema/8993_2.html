description
This file is used to store a record of each print job sent to a device. Each record specifies a device and the attribute settings for the device at the time of the print job. The attribute settings are stored in the ATTRIBUTE multiple file (#8993.21). The cross-references are "AC" the USER file (#3), "AD" the DEVICE file (#3.5), and in the PRINT JOB SETTING file, "ADEFAULT" the SAVE AS DEFAULT field (#13) and "AE" the STATUS field (#8).
