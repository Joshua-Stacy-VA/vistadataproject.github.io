.04MICRO ANTIBIOTIC LOINC CODE0;4FREE TEXTThis field is the LOINC Code assigned by the system for this test result. If the system can determine the LOINC code, it is automatically assigned. For Bacteriology accessions, a LOINC Code is captured for the GRAM STAIN field (if this field is populated during result entry), for each microorganism identified, and for each antibiotic tested. Data in this field is the actual LOINC code value determined for this test. The data is the same as the .01 field of the LOINC Codes file (file #8199). Note this field is NOT a pointer field. This is to allow test results and LOINC codes to be assigned that may not be in the LOINC codes file. A LOINC code which is not in CHCS may be captured from an off-board system via an HL-7 message when the off-board system is using a different version of the LOINC tables. Data in this field should not be modified by the user and is generally hidden from most data entry mechanisms.
