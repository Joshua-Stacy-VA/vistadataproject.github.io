4seconds before system halt0;5NUMERICThis field is used in a VAX DSM environment to delay the system level logoff process from MUMPS (HALT). The value of this field is the number of seconds the user process will ist in an idle state after the user exits the CHCS system. This will cut overhead caused by the startup of a new process if a new user should use the saem terminal within x amount of seconds.
