description
This file contains standardized entries identifying DOD units and ships: the data for these entries is uneditable, but can be marked as Inactivated. Sites can enter/edit additional entries as desired. The Master File Primary Key (Unique ID) for this file: CODE(1)\NAME(.01)\99USI
