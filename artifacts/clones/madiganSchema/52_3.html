description
This file contains the Reason Codes utilized (pointed to) by the Reason Code field (8) for the Activity Log multiple (4) of the Prescription file (52). IENs of this file have originally been incremented by 10 to allow future Reason Codes to be inserted while maintained the alphabetic order of the file entries. New entries to this file must be coordinated with the personnel responsible for Pharmacy maintenance.
