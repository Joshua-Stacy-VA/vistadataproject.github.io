MSA OIB NOTIFY MESSAGES
number
8058.2
location
^MSAIB(8058.2,
description
This file contains messages generated for exceptions encountered during OIB processing that are not specific to an account. This also includes patient specific issues which cause an account to not be generated. The specific messages are included in the MSA Notify Message file (8059).
