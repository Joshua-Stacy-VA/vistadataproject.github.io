description
This file contains the errors that were encountered during mini-registration via PACMEDNET software. If the errors are critical, then the mini-registration process will not be able to occur, such as a SSN missing or if the data was in an invalid format. The raw MARC interface data is then stored in this file until the correction is made and the data reprocessed. The MARC data will then be moved to file #52111 and deleted from this file.
