name
SMTP MAIL STATISTICS
number
3.99
location
^XMTP(3.99,
description
This file contains mail statistics which may be useful to determine which users are abusing mail. Whenever a mail message is sent or received, this file is updated. The SMTP UNAUTHORIZED DOMAIN File (file #3.98) can be used to disallow users access to certain domains.
