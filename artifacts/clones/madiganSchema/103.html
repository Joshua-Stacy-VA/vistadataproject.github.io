description
This file will hold "sets" of predefined orders which can be used as defaults when entering orders for a patient. The MTF will be able to define certain sets as being "standard of care" for a particular problem. The HCP will be able to build on these to create customized sets applicable to his/her style of practice. Secondary orders will be stored as sets of orders as well. For instance, a Barium Enema might require a number of orders such as diet and medi- cations prior to doing the procedure. In the Radiology procedure file there will be a pointer to a particular Order Set record which will list those orders to be generated as a result of order the BE. The file itself contains many of the same fields as in the Order file. The process of creating an order set looks very much the same as writing orders on a real patient, so the fields are similar.
