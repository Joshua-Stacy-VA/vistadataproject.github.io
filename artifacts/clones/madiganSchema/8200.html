1drug key1;0MULTIPLE8200.01This field indicates any key or keys that the user wants to group under this new key in the NAME field. Keys entered here must already be created in this file. When data is entered in this field, the authorization key entered in the NAME field automatically becomes a "master" key, encompassing all keys entered in this field. An HCP who has been assigned a master key will have access to all keys grouped under the master. If no data is entered in this field, the authorization key entered in the NAME field does not become a master key, but is used on an individual basis.
