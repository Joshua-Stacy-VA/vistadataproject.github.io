description
This CLASS 1 file is used to define various processing activities that can be defined in the THEN part of a Lab Rule. When processing results, the rules are evaluated for a TRUE/FALSE condition. For each condition that is TRUE, one or more ACTIONS can be performed. Those actions that are available are defined in this file. Creating new entries in this file requires programmer assistance and the data in this file should never be manipulated by a site, as irreparable damage may cause the action to be create unnecessary system error or other adverse issues.
