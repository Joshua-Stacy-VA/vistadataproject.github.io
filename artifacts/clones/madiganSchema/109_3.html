Let Episode-Counter = $P(^ORP(109.3,PID,1,0),"^",3). The Toggle is used to calculate the next epsiode, SEE '$$NEXTEPIS^ORF1' 3 CASES: 1. Patient is Out Patient ; NO Future Admission Page ; TOGGLE = 0 Next Episode = Episode-Counter + 1. 2. Patient is Out Patient ; IS Future Admission Page ; TOGGLE = 1 Next Episode = Episode-Counter. 3. Patient is In Patient ; Admisson Page Activated ; TOGGLE = 0 Next Episode = Episode-Counter + 1. The Toggle identifies Case 2. . General Algorithm: S:TOGGLE=1 Next-Episode = Episode-Counter S:TOGGLE=0 Next-Episode = Episode-Counter + 1. ------------------ TOGGLE is Set when Admission page created. TOGGLE is Cleared when Admission Page Activated -OR- Cancelled.
