description
This file contains the Fileman Path Links that specify a single branch of a Path Tree. A Link consists of a series of Fileman file and field numbers that specify a navigation pathway from a starting file to a target file. A Link is functionally equivalent to a SQL join specification. An entry is created automatically when a Path Tree is saved, and should not be edited manually.
