description
This file contains the Fileman Path Masks that specify which fields in a specific file should be referenced for data retrieval. A Mask consists of a root file number and a series of field numbers. A Mask is functionally equivalent to column selection in a SQL Select statement. Mask entries are created automatically from the Path Builder when a Mask or Tree is saved and can also be created/edited manually.
