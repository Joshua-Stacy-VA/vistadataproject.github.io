description
This file contains user-specific data related to various functionalities. It points to the USER File (#3) and can be thought of as an extension of it. Each functionality manages the selection of available choices for its users. The USER PREFERENCE File currently supports the following functionalities: EUI - Enhanced User Interface PLI - Patient Lab Inquiry RNR - Review New Results TC - Telephone Consult
