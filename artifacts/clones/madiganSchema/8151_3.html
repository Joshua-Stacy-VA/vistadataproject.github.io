This file lists CPT codes as Ambulatory Procedure Visits (APV).The APV acronym will be displayed next to a matching CPT code in the APV MOS Industry Average Comparison Report. This information is updated annually from CMS which defines procedures using CPT codes as ambulatory care surgeries in the Ambulatory Surgical Center (ASC) table.
