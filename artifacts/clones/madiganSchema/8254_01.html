description
The patient allergies selections file (8254.01) consist of: D DACLD ;Load all entries from 8250.2 Rules Store all DACs into 8254.011 as DAC Store .01 or simple names into 8254.01 as (DAC*10000) IEN1 D DAC571 ;Load 998,999 from 8250.2 Store 998 and 999 from 57,1 into 8254.011 as DAC Store GNNs into 8254.01 as (DAC*10000) IEN1 Store its Allergy name NDC into 8254.01 for appl. Lookup D HICL ;Load All HICLs from 8250 Rules Store all HICLs into 8254.011 as HICL+10000 (IEN011) Store GNNs into 8254.01 as (IEN011*10000) IEN01 Store all BNs into 8254.01 as IEN01+CNT (Cnt increment by 1) Store its GNN & all unique BNs from NDC into 8254.01 for appl. Lookup Exceptions: Rule 1. If HICL Is a device or supplies(All HICs start with X,Y Ignore) Rule 2. If HICL has one HIC and one DAC use the DAC entry in 8254.011 Rule 3. If HICL has one HIC and no DAC and a previous HICL has the same Chemical Abstract number (CASTBL) Rule 4. Store four character HICS (disregard that its a salt) Rule 5. If Bn is duplicate With different form store in AS

.04GNN0;4FREE TEXTThis field contains the drug ingredient names adopted by the USAN. The chemical name is used when the USAN name is not available. For multi-ingredient products, abbreviations may be used; for example, HCTZ (Hydrochlorothiazide), PP (Phenylporpanolamine), or CPM (Chlorpheniramine Maleate). This field is free format. The sequence of generic names in multi-ingredient products is determined by therapeutic priority.

.06COMMENT0;6FREE TEXTEnter other comments about the patient's allergy.
.07OBSOLETE DATE0;7DATE-TIMEThis field contains the date that this National Drug Code (NDC) has been identified by the manufacturer as an obsolete (discontinued) NDC number. When an NDC has been discontinued, First DataBank assigns an obsolete date to the NDC record. First DataBank will stop updating this NDC after a cutoff obsolete period agreed upon by the customer. This cutoff obsolete period is typically 3 years, so that if, for example, an NDC contained an obsolete value of 890101, January 1, 1989, then on any update or full file tape created on or after January 1, 1992 (3 years after 890101), the NDC would no longer be sent. Customers need to know that the NDC is no longer being supported on their update tapes, so that the customer system can either anticipate when the NDC will no longer be sent, based on the cutoff date the customer provides to First DataBank, or by receiving a full file replacement formulary on some regularly scheduled basis, or through ad hoc requests. This field can be used in conjunction with the Previous National Drug Code (PNDC) and the Replacement National Drug Code (REPNDC).
.08DUPLICATE BRAND NAME0;8SET OF CODESD:Duplicate
This field is a flag indicating whether an entry is a duplicate brand names. over-the-counter items such as Bayer Select may have multiple formulations but identical brand names.
1.01EXPANDED GNN1;1FREE TEXTThis field contains the drug ingredient names adopted by the USAN. The chemical name is used when the USAN name is not available. Under FDB, because of space considerations, abbreviations were used in multi-ingredient products; for example, HCTZ (Hydrochlorothiazide), PP (Phenylpropanolamine), or CPM (Chlorpheniramine Maleate). This field will NOT use these abbreviations, since the generic name is more explicit and easier to read.
