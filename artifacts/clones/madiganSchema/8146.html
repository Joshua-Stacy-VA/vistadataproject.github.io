The LANGUAGE IDENTIFY file is a file containing DOD-Standard languages. Note that this file was submitted to CHCS (6/92) by Bill Didier. For languages which were identified as a "dead language", thes entries were added into the file and inactivated. The file was created primarily for MCP design to search MCP Providers who speak/communicate in a particular language. The Master File Primary Key (Unique ID) for this file: Code(.01)\Description(1)\99LNG
