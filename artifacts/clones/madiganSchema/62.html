description
The Collection Sample file is a dictionary for collection samples and describes the collection sample requirements specific to a laboratory. This is anothe "Day One File" since entries included here are used when building the LAB TEST file. The file includes specific information about each collection sample, such as the default specimen, tube top color, synonyms, and CAP code units for blood collection by laboratory.
