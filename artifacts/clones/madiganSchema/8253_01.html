description
The ADIMMSTR ADIMTBL ADIMMONO file consist of three files: 1) The Main file is the ADIMMSTR 2) Subfile 1 8253.011 is the ADIMTBL 3) Subfile 2 8253.012 is the ADIMMONO A description of each file follows: ADIMMSTR - RADIMMA1 The Drug/Drug Interaction Master Table uses the 3-byte drug interaction code ADI*CODE as a key to provide an interaction description, clinical effect, and references to Hansten's and Facts and Comparisons drug interaction information. The Internal entry number (D0) is the American Drug Interaction Code (ADICODE) with leading zeros removed. ADIMTBL - RADIMGC0 The GCN Seq No / Drug Drug Interaction Code Relation Table is an association between the Generic Code Number Sequence Number and an individual interaction code (ADI*CODE). The association resolves the many to many relation that exists between the GCNSEQNOTBL and an individual interaction code, through the use of a concatenated key. If a GCNSEQNOTBL record does not have any drug drug interaction codes, then there will be no reference to the GCN Seq No in the ADIMTBL. If there are one or more drug drug interaction codes for a specific GCN Seq NO, then there will be one or more records in the ADIMTBL, each record containing the specific GCN Seq NO concatenated with an individual interaction code. The NDDFTM provides for a maximum of 39 ADIMTBL records for each GCN Seq No. The Internal entry number (D1) is the Generic Code Number Sequence Number (GCNSEQNO) with leading zeros removed. ADIMMONO - RADIMMO0 The Drug/Drug Interaction Monograph Table uses the 3-byte ADI*MONO monograph key from the Drug/Drug Interaction Master Table to provide significance level, mechanism of action, clinical effects, predisposing factors, discussion and references concerning the interaction. There is an index to provide selective retrieval of individual sections of the monograph. The Internal entry number (D1) is the Drug-Drug Interaction Monograph Text Sequence Number (ADIMONOSN) associated with a ADIM Monograph Number (ADIMONO) with leading zeros removed. The Master File Primary Key (Unique ID) for this file: IEN\(.01)Name\99FDA
