description
This file contains the electronic medical records and electronic image tracking folders defined on the CHCS system. Electronic records are linked to file rooms defined in the Borrower's File and can be requested to be pulled, charged out and checked in. Electronic medical records can be retired using the record retirement functionality within the MRT application.
