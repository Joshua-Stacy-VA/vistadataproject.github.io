description
This file is used to store site-specific lab result processing rules. An authorized user can build a number of rules, each with an IF CONDITION that must be evaluated to a TRUE condition, and based on that condition, tell the system to automatically perform some action or actions. User will enter the rule conditions using an intuitive, English-like system, and once the condition has been entered, the user will instruct the system to save the condition, at which time a number of basic syntax checks and validations will be performed on the condition to make sure there are no major issues with the construction of the rule. The system will not be able to validate the logical view of the rule, so the rule may not behave correctly when executed. The user should further design a number of formal testing scenarios to allow the user to more fully validate the condition.
