description
The SIDR ICD PLACEHOLDER SET file contains ICD DOD Unique codes that are for use by CHCS only. These ICD DOD Unique codes are system generated and must not be used by end-users or sent to off-board systems. CHCS uses these codes as placeholder codes that are system generated and are not part of any ICD code set. This is a standard file (classified as Group 1) and cannot be modified by the site.
