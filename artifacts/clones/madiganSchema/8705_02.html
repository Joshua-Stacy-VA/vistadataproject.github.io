.01lab result field0;1FREE TEXTBThis is the actual field number in the designated file/subfile that will store a LOINC code. The combination of subscript, file/subfile, and field allows the system to find other fixed attributes for determining the LOINC code during result entry. If an attribute is not defined, that information is to be inherited from the test and/or accession being resulted, whichever is appropriate.
.02loinc component0;2POINTER8188.1The LOINC Component Name is the name of the component or analyte measured. Examples are "GLUCOSE" for a Chemistry test, "MICROORGANISM IDENTIFIED" for a Microbiology test, "GENTAMICIN" for an Antibiotic Susceptibility test, or "REPORT.TOTAL" for an Anatomic Pathology test. A LOINC Component Name must be entered to capture a LOINC Code for a test result. This field holds a default LOINC Component to be used whenever this field is populated during result entry. If this field is null, the LOINC component will be inherited from either the test being resulted or other appropriate file linked to the accession or result being entered. This field should not be edited by site personnel.

.06loinc scale0;6POINTER8188.5This is the LOINC Scale which must be entered to capture a LOINC Code for the test result. The LOINC Scale is the scale of measure of a test. It may be Quantitative, Ordinal, Quantitative or Ordinal, Nominal, Narrative, or "Multi". Quantitative is when the result is a numeric value that relates to a continuous numeric scale. Ordinal refers to results with ordered categoric responses (1+,2+,3+, positive, negative). Quantitative or Ordinal means the test can be reported as either Quantitative or Ordinal. Nominal refers to results that have categories of appearance that do not have a natural ordering (yellow, clear, bloody). Narrative is text, such as the description of a microscopic part of a surgical test. "Multi" is many separate results structured as one piece of text. This field holds a default LOINC Scale to be used whenever this field is populated during result entry. If this field is null, the LOINC Scale will be inherited from either the test being resulted or other appropriate file linked to the accession or result being entered. This field should not be edited by site personnel.
