description
A logical record is a linkage of multiple physical records. It is defined by the type of data it contains, not by the physical location of that data. This file identifies the logical records in CHCS: by CATEGORY (Patient Demographics, Clinical, Administrative) by RECORD TYPE (an abbreviated name) by PHYSICAL LOCATION (the files/subfiles and fields which contain data for the referenced record) The physical location data is used as a "map" for creating the archive DDs.
