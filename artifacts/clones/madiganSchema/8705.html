description
This file is used during result entry to provide LOINC attributes that are the same for all cases. This prevents the programmer from having to hard-code LOINC attributes. When a Micro or AP subscript test result is being filed, the system uses the subscript, file/subfile number and field number to look into this file and populate fixed LOINC attributes. If an attribute is null in this file, that attribute is then either test or accession specific, and is therefore obtained from that reference, prior to deriving the LOINC code. The data in this file is used internally within the lab system, and should not be modified by the site users. Doing so may cause the system to assign an incorrect LOINC code or none at all.
