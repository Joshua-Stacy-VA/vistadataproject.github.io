description
This Class 4 static file contains all the possible Accession Number ID formats that can be used with an Auto Instrument. Each format entry will have a Name, Description, a flag field indicating whether a Work Document is required, a line-tag and routine reference, and an Accession Identifier Field number. The line-tag and routine reference fields will be used during the TAR option later during results Auto Validation and Certification to determine if the sample ID returned with the results is a valid ID that directly points to a valid, open accession with pending tests. The Accession Identifier field number will be used during the BID option and during the Auto Download phase to determine which identifier field will be included with the test orders being sent to the instrument. This value should always be printed on the label, either in human-readable and/or encoded in a barcode.
