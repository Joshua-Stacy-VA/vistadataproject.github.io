The Lab Description file is a dictionary of "canned" descriptions the laboratory uses repeatedly. They are called by using a defined abbrevation. Microbiology uses this file extensively to describe comments on various cultures. Other laboratory sections use this file to add comments describing test results or the reason a test is being cancelled. Entries may be added or modified.
