description
The PEMASHP file consist of two files: 1) The Main file contains the PEM - ASHP Monograph Code (ASHPCDE) with leading zeros removed. 2) Subfile 1 8253.0221 is the PEMASHP A description of each file follows: The 3-byte ASHPCDE monograph code provide ASHP-licensed patient education monographs. These are the most detailed of the three types of First DataBank Patient Education Monographs. The Internal entry number (D0) is the Patient Education Monograph Code (PEMONO) with leading zeros removed. PEMASHP - RPEMMOA0 The Patient Education ASHP Monograph Table uses the 3-byte ASHPCDE monograph code from the Patient Education Master Table to provide ASHP-licensed patient education monographs. These are the most detailed of the three types of First DataBank Patient Education Monographs. The Internal entry number (D1) is the PEM - ASHP Monograph Text Sequence Number (ASHPCDESN) with leading zeros removed.
