description
This file is used to map CoPath Prefix and Case Types to CHCS Anatomic Pathology subscripts. The site may edit the standard entries, add new entries, and modify and delete existing entries. This file is used by sites interfaced to the CoPath Anatomic Pathology COTS system.

.01copath prefix(+)0;1FREE TEXTBThis file contains the name of CoPath Prefix entries and associated CoPath Case Types used with the CHCS/CoPath Anatomic Pathology Interface. CoPath Prefix and Case Types are configured by CoMed and can be found in the accessioning module for each CoPath department (Surgical Pathology, Cytology, and Autopsy). Entries will be used for mapping to CHCS accession areas and anatomic pathology lab tests and must be typed exactly as displayed on CoPath. Entries can be added, deleted, or edited.
