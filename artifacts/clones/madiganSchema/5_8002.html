description
The ZIP CODE file contains zip codes for cities in the fifty states and U.S. territories. The information in the file is extracted from a tape provided by the U.S. Post Office. This is a Standard Class 1 file, users cannot add or delete entries, and only the following fields can be editted: o PROV NETWORK AREA ZIP: o PT ENROLL AREA ZIP: o MODIFIED CITY The Master File Primary Key (Unique ID) for this file: IEN\CODE(.01)\99ZIP
