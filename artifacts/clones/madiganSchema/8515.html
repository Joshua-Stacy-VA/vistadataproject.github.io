description
The PAS MAILERS file can be modified by the site or File & Table build team to customize the form letters that patients are sent to notify them them of a Wait List Notification, No Show Notice, Strong No Show Notice, or Cancellation Letter. In a site with multiple Divisions each Division can set up their versions of the standardized letters to be sent out.
