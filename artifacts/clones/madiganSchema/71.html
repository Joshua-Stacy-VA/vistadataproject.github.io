description
This file contains all the procedures that may be associated with an imaging exam. The entries in this file are the allowable choices the user will be able to pick from at the 'RADIOLOGY PROCEDURE' prompt. Standard (defaults) procedure data such as film utilization is stored in this file.

This field assigns the procedure to one of three possible categories: radiology procedure, procedure group, or charge. Set this field to RADIOLOGY PROCEDURE if this is a single radiology diagnostic procedure. 12total weight0;5NUMERICEnter the RVS weighted workload value associated with this procedure. Enter a value in this field only if this procedure does not have a divided workload value (that is, a procedure weight and a reporting weight). For procedures with a divided workload value, the CHCS adds the values entered in the PROCEDURE WEIGHT and REPORTING WEIGHT fields and automatically enters the sum in this field. If no value is entered in the PROCEDURE WEIGHT and the REPORTING WEIGHT fields, then a value must be entered in this field. The CHCS assigns workload for this procedure when the exam reaches a status of COMPLETE. If you are changing this procedure from a divided workload value into a single workload value, then you must delete the values in both the PROCEDURE WEIGHT and REPORTING WEIGHT fields before you enter a value in this field.

Enter ALL if you want this procedure or procedure group to be suppressed from view for all users. Enter a NO or leave this field blank if you do not want to suppress this procedure or procedure group from view. Enter NO if a signed consent form does not have to be submitted before this procedure can be performed.
