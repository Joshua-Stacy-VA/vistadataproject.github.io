name
CLINICAL SITE PARAMETERS
number
101.1
location
^ORD(101.1,
description
This file holds the MTF specific data needed to customize Order Entry to the particular site. In setting up a site, this file should be edited. Certain fields are mandatory but others are not. For those non mandatory fields, when no data is entered various actions may occur which are documented in each field.
