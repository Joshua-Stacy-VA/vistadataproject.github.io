description
This file contains information for each schedule function type (make, modify or cancel). The fields included in this file are exam number (the default is the last exam entry for the function type), start date (the date the system will start its search for an appointment), end date (the date the system will end its search for an appointment), start and end times (the times of day set for the search parameters), radiology location (designated in Locations File #79.1), radiology room (designated in Room File #78.6), if overbooking is desired (allowed only by those with the RADOB security key), if viewing of pending appointments is desired, the date and time of the scheduled appointment, the exam duration, patient mobility status and category, the reason for modification or cancellation, the patient's name, the procedure name, the preliminary procedure room (can be changed by scheduler), and the preliminary appointment date and time (can be changed by scheduler). This information is set up in screen form and can be viewed in the scheduling, modify or cancel procedure options in the Procedure Scheduling Menu of Radiology Scheduling Menu.
