description
This file contains the result codes that can be associated with an exam. The code is attached to an exam at the time of results entry for the exam. The diagnostic code represents a quick overall summary of what the radiologist wrote in his/her report concerning the exam.

Enter 1 (for YES) in this field and the CHCS immediately notifies the requesting HCP when a radiology report is assigned this result code. Enter 0 (for NO) and the CHCS notifies the requesting HCP at log on when a radiology report is assigned this result code.
5verification priority0;5NUMERICEnter a number in this field that the CHCS can use to determine the priority placement of radiology reports in the radiologist's queue for reports assigned this result code. The highest priority is 1, the next highest is 2, and so forth. The lowest priorities are 0 and null (that is, leave this field blank). NOTE: With the exception of null and 0, do not assign the same verificaton priority to more than one result code. Null and 0 may be assigned to as many result codes as desired.
