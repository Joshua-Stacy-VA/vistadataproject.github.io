This field is pertinent to element type FILE only. It determines the method of loading CMS saved data into the environment. Saved data could be merged, or added. It could also overwrite matching existing data. Or the entire FILE could be deleted and the saved data added as new entries. For this field the following values apply: 0 = Merge entries - This means update the file without first deleting the old data so the old and the new both end up in the file. 1 = Overwrite entries - This means any entry being loaded that matches an existing entry. The existing entry will be deleted prior to loading. 2 = Add new entries only - This means that only those entries which do not already exist will be loaded. 3 = File Overwrite - This means that the entries existing in the file will be deleted then the new entries will be added.
