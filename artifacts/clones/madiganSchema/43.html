name
ADT PARAMETERS
number
43
location
^DG(43,
description
This file contains the site specific parameters which are used by the Admission, Discharge and Transfer (ADT) modules. The parameters are set by using the 'DGPAR' routine or the 'ADT Parameters' menu option. The ADT parameters must be set prior to running ADT.
