name
VA CMOP BATCH FILE
number
8220
location
^PSCMOP(8220,
description
This the VA cmop batch file. It contains the selected refills that will be send to the VA CMOP for processing and mailing out. This file manages all of the activity related to refill an rx at the VA CMOP.
