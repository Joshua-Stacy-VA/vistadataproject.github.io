description
This file contains all the facility ward locations and their related data, i.e., Operating Beds, Rooms, Beds within rooms, Bedsection, etc. The wards are created/edited using the WARD DEFINITION option of the ADT module. This file also contains all statistical data, by date, necessary for the computation of the daily Bed Status report.
