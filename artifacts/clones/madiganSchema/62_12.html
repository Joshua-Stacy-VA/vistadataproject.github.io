description
This file contains all the Antibiotics and their definitions needed for microbiology result entry. It contains approximately 40 pre-supplied entries. Each of the entries is defined according to the need of your MTF. Information which is MTF definable for each antibiotic includes sensitivity result, interpretation of each result, abbreviation, print order, and organism specific results and interpretations. DO NOT DELETE any of the existing entries.
