.01error code0;1FREE TEXTBThis is the exact string of characters that represents a specific error condition for this instrument. In the case where DII is using error codes from more than one location in the instrument's data stream, and there is an ambiguity between the codes used for those locations, DII will resolve one set of codes to be unique. New error codes are added to this file via HL-7 messages from the DII system. Error codes cannot be entered into this file via the AIE option and should NOT be manually entered via Fileman.
.02error text1;1FREE TEXTThis is the English translation for the particular error code. It may be up to 245 characters in length. Do not include any formatting characters in this string.
.03user defined error message2;1FREE TEXTThis is a site-defined English translation for the particular error code. This message text will be used in all places instead of the default message text. It may be up to 245 characters in length. Do not include any formatting characters in this string.
.04action code0;2SET OF CODES1:Display Error, File Result
2:Display Error, Do NOT File Result
3:Do NOT Display Error, File Result
4:Do NOT Display Error, Do NOT File Result
This field allows the user to specify the action that is to be taken when this error code is encountered while processing the data in the TAR option. The valid actions are: 1. Display the error message (or user-defined error message) and file the test result. 2. Display the error message (or user-defined error message) and do NOT file the test result. 3. Do NOT display the error message, and file the test result. 4. Do NOT display the error message, and do NOT file the test result.
