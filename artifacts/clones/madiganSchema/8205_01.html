Enter 'B' if this item is usually a Bulk Issue to this ward/clinic. Enter 'C' if it is usually a Clinic Issue. This value will be used as a default during the issue entry process (Enter Stock Issue) and can be accepted or changed.
4display order0;5NUMERICADThis OPTIONAL field allows you to control the order in which items display. If the Display Order is not defined the items will display in ALPHABETIC order in the Enter Stock Issue option and on the Issue Requisition Form. This may be useful if the requisition form has items listed in a format other than alphabetic. For example, if the form has different sections for topicals, injectibles, and orals, you could use the Display Order field to imitate this format. If the Display Order is not defined, the item will display in alphabetic order. It is recommended that Display Order Numbers increment by 10 (ie. 10,20,30...). The following list of items demonstrates the use of the Display Order field: AMINOPHYLLIN 10MG/ML 10ML VIAL (display order = 10) DIGOXIN 0.1MG/ML 2ML VIAL (display order = 20) FUROSEMIDE 10MG/ML 2ML AMP (display order = 30) ASPIRIN 325MG TAB 100/BTL (display order = 40) MILK OF MAGNESIA 30 ML (10 CUPS) (display order = 50) If you wish to add BENADRYL 50MG TAB between ASPIRIN and MILK OF MAGNESIA, the Display Order value must be between 40.1 and 49.9.
4.5suppress from cart list0;612Answer YES if MED orders for the Reference Drug of this item should be suppressed from printing on the Unit Dose Cart List. Answer NO to NOT suppress MED orders from the Cart List.
5min maxCOMPUTEDThis computed field contains the minimum and maximum quality of the stock item based on the parameters set in the MINIMUM and MAXIMUM fields. It displays on the Issue Requisition Form pertaining to the ward or clinic.
