description
The Rad Workload Exception File captures workload that is not yet eligible for reporting to WAM and other workload reporting systems. Exceptions captured in this file are a) procedures with invalid CPT codes, b) performing MEPRS/DMIS ID mismatches, and c) requesting MEPRS/DMIS ID mismatches. The end user must resolve the exception making it eligible for reporting to WAM and other working reporting systems. Once resolved, the exception is purged from this file and the valid workload is stored in the Rad Workload Data file.
