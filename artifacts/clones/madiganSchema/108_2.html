description
This file holds all nursing procedures which can be ordered by either the HCP or the nurse (for the nursing care plan). A nursing or intraward procedure is defined as a procedure carried out and resulted by ward personel, without any involvement with another ancillary. Examples would include Blood pressures, Dressing changes etc. Certain of these procedures may be ordered both by an HCP and a nurse. These procedures have the Order Type of "BOTH". Certain procedures can only be ordered by the Physician class users. These procedures have the Order Type of "NRS". And there are certain procedures that can only be ordered by the Nurse class users. These procedures have the Order Type of "NIO".
