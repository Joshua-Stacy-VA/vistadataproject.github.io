description
This file is used to support the monitor mapping option of the CHCS M/SQL Administration Utilities functionality. Information pertinent to the background job status of the mapping process will be stored in this file.

.06COMPLETED DATE/TIME0;6DATE-TIMEThis field contains the date and time of completion of the mapping job. The data is intended to be used in conjuction with the corresponding data stored in the ACTUAL START DATE/TIME and NUMBER OF FILES MAPPED fields to provide an estimate of the average length of time to map a FileMan file to an M/SQL table.
.07NUMBER OF FILES MAPPED0;7NUMERICThis field contains the number of root FileMan files attempted to be mapped during this particular run. .08ACTUAL START DATE/TIME0;8DATE-TIMEThis field contains the date and time of when the TaskMan task is actually started. The data is intended to be used in conjunction with the data stored in the COMPLETED DATE/TIME and the NUMBER OF FILES MAPPED fields to provided an estimate of the average length of time to map a FileMan file to an M/SQL table.
.09TASK NUMBER0;9NUMERICThis field contains the TaskMan task number associated with this run.
