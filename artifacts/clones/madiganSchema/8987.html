description
This file contains the definitions and directives required to generate a series of symbols by the System Throughput Analysis routines. These definitions are segregated into OS specific versions. They are prepositioned for accepting input from the Option/Audit Global (and a Storage Global used only at the Software Modeling and Algorithyms Group). The operating system type is used to seggregate the reports for the specific system information. This set of definitions are input to the XUSODRV* routines as invoked from the EN^XUSODRV or RUN^XUSODRV.
