description
This file holds workload data which is used for WAM reporting purposes. It is updated according to an amount of days specified by the user. The default timeframe for updating is every 30 days. The data in the file is stored to match the limit of the supporting WAM Data Set ID file, currently set at 2 years. Everytime the data in the file is updated, a check is made for any data older than 2 years or no supporting data in the WAM Data Set ID file. If any data older than 2 years or no supporting data in the WAM Data Set ID file, it is purged from the file prior to updating.
