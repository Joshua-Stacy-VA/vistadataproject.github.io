.07pull date0;7DATE-TIMEDEnter the date for which you want the CHCS to search for appointments. Medical Record Tracking: The CHCS looks at the PAS Clinic Profile, PATIENT PULL RECORD field to determine the number of days in advance of the appointment date to pull records. For example, if the profile indiates "1" day, the user would enter a pull date of TODAY for tomorrow's appointments. Image Tracking: When clinic appointments require films, the system looks at the PAS Clinic Profile, RADIOLOGY PULL RECORD field to determine the number of days in advance of the appointment date to pull films. The CHCS also checks radiology locations such as Ultrasound and MRI for appointments within radiology. For radiology locations, there are no parameters to pull folders days in advance. For example, users would enter TODAY for today's appointments.
