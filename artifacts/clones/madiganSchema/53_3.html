description
This file is simple a `table' of look-up for the ACTION field in the ACTIVITY LOG sub-file of the UNIT DOSE package. The ACTION field was orginally a set of codes, but the activity reasons became too many, thus creating the need for this file. The ACTION field is now a pointer to this file.
