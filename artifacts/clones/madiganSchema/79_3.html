This file contains radiology parameters that will apply to all radiology locations at this MTF. Once the parameters defined in this file have been set, then they should not be changed. If a parameter does need to be changed, then contact the Radiology Department Supervisor.

.02DAYS TO CHECK FOR BARIUM0;2NUMERICEnter the number of days before and after procedures are ordered and/or scheduled that you want the system to check the patient's Radiology Exam file entries for exams that use barium. If the system finds one or more radiology exams that use barium within this date range, then it will display a warning. For example, if you enter 7 in this field, then the system will check for duplicate barium exams that are scheduled up to 7 days before the proposed exam appointment date, on the proposed exam appointment date, and 7 days after, for a total of 15 days. If you enter 0 in this field or leave it blank, then no barium check will be performed.
.03DAYS TO CHECK FOR CONTRAST0;3NUMERICEnter the number of days before and after procedures are ordered and/or scheduled that you want the system to check the patient's Radiology Exam file entries for exams that use intravascular (IV) contrast media. For example, if you enter 7 in this field, then the system will check for duplicate IV contrast media exams that are scheduled up to 7 days before the proposed exam appointment date, on the proposed exam appointment date, and 7 days after, for a total of 15 days. If you enter 0 in this field or leave it blank, then no IV contrast media check will be performed.

.22# DAYS TO RETAIN REPORT BATCH0;22NUMERICEnter the number of days that a batch of transcribed radiology reports should remain in the system after all reports in the batch have been verified. 202REPORT RESULTS AVAILABILITYRES;2SET OF CODES0:NO APPROVAL REQUIRED
1:DEPENDENT RESIDENT SIGNATURE ONLY
2:DEPENDENT AND INDEPENDENT RESIDENT SIG. REQUIRED
3:RADIOLOGIST SIGNATURE REQUIRED
This field is used to determine what level of verification will be required on radiology reports before they will be made available to CHCS users. NOTE: Radiology reports that have been signed by both the interpreting and the supervising radiologists will be available to all CHCS users, regardless of the setting in this field. Enter 0 (for NO APPROVAL REQUIRED) if you want radiology reports to be available to CHCS users before the reports have been verified by anyone (i.e., the report status equals EXAMINED or TRANSCRIBED). Enter 1 (for DEPENDENT RESIDENT SIGNATURE ONLY) if you want radiology reports to be available to CHCS users when the reports have been verified by a dependent resident radiologist (as the interpreting radiologist) and the supervising radiologist. Reports that have been verified by an independent resident or a staff radiologist (as the interpreting radiologist), but not by the supervising radiologist, will be available to all CHCS users and will display/print with the disclaimer that the report is pending verification and should not be considered the final version. Enter 2 (for DEPENDENT AND INDEPENDENT RESIDENT SIGNATURE REQUIRED) if you want radiology reports to be available to CHCS users when the reports have been verified by a dependent or an independent resident (as the interpreting radiologist) and the supervising radiologist. Reports that have been verified by a staff radiologist (as the interpreting radiologist) will still be available to all CHCS users and will display/print with the disclaimer that the report is pending verification and should not be considered the final version. Enter 3 (for RADIOLOGIST SIGNATURE REQUIRED) if you want radiology reports to be available to CHCS users when the reports have been verified by a staff radiologist (as the supervising radiologist).
203DAYS IN RESIDENTS QUEUERES;3FREE TEXTEnter the number of the days a report may remain in a resident's or interpreting radiologist's queue awaiting verification. If the resident or the interpreting radiologist does not verify this report within the number of days defined in this field, then the report will be sent to the queue of the supervising radiologist (as identified on the report) for verification. NOTE: If a supervising radiologist has not been identified on the report, then the system will send the report to the queue of the supervising radiologist for the procedure, as defined the Procedure Class file. For example, if you enter the number 3 in this field, then reports for any given resident or interpreting radiologist will remain in that resident's or interpreting radiologist's queue for 3 days. On the fourth day, the system will remove the report from the resident's or interpreting radiologist's queue and send it to the queue of the supervising radiologist identified on that report. If a supervising radiologist has not been identified on the report, then the system will place the report in the queue of the radiologist who has been defined as the supervising radiologist for the procedure class associated with this procedure.
