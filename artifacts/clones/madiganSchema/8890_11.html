description
This file acts as an interface between Nursing and Facility QA by listing all the incident reports generated under nursing and forwarded to facility QA for review.

.01incident #(+)0;1FREE TEXTBThis field contains the division(s) which are using the selected Military Education/School (military educational program or school) when documenting provider's credentialing information. Valid entries may be found in the MEDICAL CENTER DIVISION file (file #40.8). Users may add or delete their allowable divisions from this field as desired. However, the selected Military Education/ School may only be used if the user's division is in the field and, if the provider whose credentials are being documented is associated with one of the divisions in the field.

This field contains the ancillary department with which the incident being reviewed is associated. This value is derived from a set of codes, currently including "NUR" for Nursing and "LAB" for Laboratory. At present, the QAN functionality is used primarily for tracking Department of Nursing incidents. It can be used by other ancillaries, if users are given appropriate security codes.
