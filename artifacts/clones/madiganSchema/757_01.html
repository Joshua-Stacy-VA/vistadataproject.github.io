description
This file contains all text pertaining to the Major Concepts, Concept Synonyms, Concept Lexical Variants, Synonymous Lexical Variants, displayable text, original displayable text (if the display text has been altered), and the term definitions (when available). Searches are conducted using the special look-up routine GMPTA1. NOTE: With the exception of the and the Definition (6), the data entries in this file should not be altered by the site. This file requires re-indexing to occur in a specified sequence. The complete re-indexing instructions are included in the Technical Manual.
