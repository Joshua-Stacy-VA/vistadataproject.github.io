description
The DMHRS ACUITY file receives data only via an HL7 message from DMHRS. That data is then stored in this file on CHCS. The Nursing Care Hours for each patient for each episode will be used by PAD functionality to be recorded on the SIDR tape in approximately monthly increments.
Fields
#NameLocationTypeDetailsIndexDescription
.01PATIENT(+)0;1POINTER2BThis field contains the patient IEN and is a pointer to the Patient (#2) file. Data is entered into this field only via an HL7 message from DMHRS.
.02REGISTER NUMBER(+)0;2NUMERICThis is the patient's register number that is associated with a specific episode of the patient. Data is entered into this field only via an HL7 message from DMHRS.
.03DATE/TIME OF CARE(+)0;3DATE-TIMEThis is the date/time associated with the patient's acuity level. Data is entered into this field only via an HL7 message from DMHRS.
.04PATIENT ACUITY(+)0;4NUMERICThis is the patient's acuity value associated with this date of care. It is a number ranging from 0 to 6. Data is entered into this field only via an HL7 message from DMHRS.
.05NURSING CARE HOURS(+)0;5NUMERICThis is the number of nursing care hours associated with the patient's acuity level. Data is entered into this field only via an HL7 message from DMHRS.
