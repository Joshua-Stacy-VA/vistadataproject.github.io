description
This file is a table of major command posts for the military. The file contains acronyms of different agencies and command centers for the military. It is a standard file (classified as Group 1) and cannot be modified by the site. SOURCE: DMSSC, DATA ADMINISTRATION CONTRACT, MDA903-86-C-0215, FINAL ADDENDUM TO THE AQCESS/CHCS FUNCTIONAL DESCRIPTION FOR THE STANDARD REQUIREMENTS FOR THE INPATIENT DATA RECORD FOR TASK ORDER NO. DAT2, 12 AUGUST 1988.
