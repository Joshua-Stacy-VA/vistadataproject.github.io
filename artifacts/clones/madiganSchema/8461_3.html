description
This file contains the Fileman Path Trees that specify a series of Path Links and Path Masks. A Tree consists of a series of paired Links and Masks that specify the navigation path from a root file to a target file, and the selected fields in that target file. A Tree is functionally equivalent to a SQL View, and the Path Builder utility can generate valid M/SQL 'Create View' DDL text that can be imported. An entry is created automatically by Path Builder Tree saving, and should not be edited manually. A saved Tree can be loaded for review and manipulation, and grafted onto a branch of another tree being built in the Path Builder.
