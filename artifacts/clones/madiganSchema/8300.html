description
Nurse "chart" the carrying out of order tasks in this file, which serves as part of the clinical chart of the individual patient. This file contains all the nursing results of order tasks. The nursing results are those that are carried out by nurses. Results from automated ancillaries, such as Lab and Rad, are stored in their respective result files. A result record is created when a result is entered for a task. A task may have multiple results (i.e. a PRN task).
