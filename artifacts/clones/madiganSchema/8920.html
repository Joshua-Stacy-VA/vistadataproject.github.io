.08VOLUME FOR ARCHIVE0;8FREE TEXTThis is the name of the volume set on the Archive System that is to be used for archive/purge tasks. It will usually be equal to "ARC". When the value of this field is empty, archive/purge tasks will be performed on the Production System.
.09VOLUME FOR RETRIEVE0;9FREE TEXTThis is the name of the volume set on the Archive System that is to be used for retrieve tasks. It will usually be equal to "ARC". When the value of this field is empty, retrieves will be performed on the Production System.
.1SOFTWARE SHADOWING0;1012This will turn on the archive implementation of software shadow. However, if the site is not using shadow sets, this implementation does need to be used.
