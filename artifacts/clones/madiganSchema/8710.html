description
The Lab Method file provides a means for changing normal ranges without impacting previously reported results. The file contains the site/specimen multiple which was formerly part of the Lab Test file. Rather than creating a new test to reflect a change in method that results in new reference ranges, multiple methods with their associated reference ranges can be designated for a test. The methods can be specified as a default method or can be changed at the time of result entry.

Use the variables created in the "LAB TEST USED IN CALCULATION" multiple. Example: T7=T4+T3 The variables are T4=C(1) and T3=C(2). The formula entered for this calculation would be C(1)+C(2). The system will read the calculations from left to right. If the formula requires a specific portion to be calculated separately, enclose that section in parentheses. In the example C(1)*100/(100+C(2)), if the parentheses were not included the system will calculate C(1)*100 then divide by 100 and then add C(2). With the parentheses the system will calculate 100+C(2) then divide it into C(1)*100.
