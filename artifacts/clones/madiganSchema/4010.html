#NameLocationTypeDetailsIndexDescription
.01segment name(+)0;1FREE TEXTThe full name of the segment.
.02segment id(+)0;2FREE TEXTCThis identifies the segment in a message. It should be a 2-8 character ID. It will be used to pattern match the segment as well as serve as the prefix for all script variables.
.03segment pattern0;3FREE TEXTThis field, if present, will be used for inbound messages to set the pattern match for the segment. A wild-card character '*' will be appended to the end of the value of this field. For example, if this field is set to ABC?12 the pattern match will be ABC?12* in the script. If no value is present, the segment ID will be used to identify the segment.
