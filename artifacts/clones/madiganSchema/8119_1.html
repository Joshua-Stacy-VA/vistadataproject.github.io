description
The MEPRS Replacement Table contains the inactive MEPRS codes and the associated replacement MEPRS code. This file is used for RX refills and Quality Control specimen updates when the associated MEPRS code is inactive. The PHR module will check this file to see if a replacement value is available for an inactive MEPRS code. The fill and workload will then be associated with the replacement MEPRS code. The LAB module will check this file to see if a replacement value is available for an inactive MEPRS code. The Quality Control specimen update will then be associated with the replacement MEPRS code. This is a class 2c file.
