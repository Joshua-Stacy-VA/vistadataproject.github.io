description
This file contains selective metadata about all the Fileman Data Dictionary files and subfiles at this site. This data is used by the AP SQL utilities in Path building, and also by ongoing M/SQL Client-Server development on PCs. It is created during the post-initialization phase of the AP SQL installation. It can be recreated as needed to accurately reflect any Fileman changes by running the 'Path file init' from the AP SQL Site menu, but should not be edited directly.
