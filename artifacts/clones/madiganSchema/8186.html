description
This file contains the old values, such as DMIS ID and GROUP ID, for a division prior to a DMIS ID Realignment being run for that division. After a realignment is run, this file contains the effective month of the last realignment, the old DMIS ID, and/or GROUP ID for a division. This is a Class 2C file. This file is populated by the software during a DMIS ID Realignment.
