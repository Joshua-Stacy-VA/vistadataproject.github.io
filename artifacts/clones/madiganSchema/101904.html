description
By default, it is assumed that all Lexmark printers include the PharMex SIMM chip and no other printer brands include the PharMex symbol set. This file was created to allow sites to override these assumptions.

Does this printer support the PharMex symbol set?
1pem0;3BOOLEAN0:NO
1:YES
Only devices with a subtype that starts with P-LEXMARK are allowed to print a PEM (patient education monograph) with each label. By default it is assumed that all Lexmark printers will print the PEM. This field provides the ability to override that assumption. The most common scenario for setting the PEM field to NO is for refill pharmacies that use Lexmark printers. It is possible to have a Lexmark printer with an 8 1/2 x 11 tray and a half-sheet tray. Just set up two different entries in the Device file to reference the same physical device with two different subtypes.
2flush each script0;4BOOLEAN0:NO
1:YES
To reduce paper jams in Lexmark printers, the paper path is cleared before printing the next label. The down side is that it is much slower.
3reverse orientation0;5BOOLEAN0:NO
1:YES
Send the escape sequence to a Lexmark printer for Reverse Portrait. This allows the site to put label stock in backwards from the normal feed direction. Some sites have heard this reduces paper jams. The experimentation at TMSSC shows that Reverse Portrait orientation dramatically increases paper jams.
