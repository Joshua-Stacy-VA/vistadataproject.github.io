description
This file is used to store the individual questions that can be asked during order entry. The file stores all the parameters necessary for the question, such as the prompt string to be displayed, the expected data type of the user's response, and various data input control information, such as lower/upper limits and maximum length.

Because the prompt presented during order entry will not appear on displays and reports with the user-entered answer, use this field to enter a label that identifies what the response refers to. Include appropriate punctuation. For example, the prompt for Dose Time during order entry may read "Enter the dose time?". This field may contain the phrase "Dose Time:" that will display on displays and reports in front of the response of the date/time entered (Dose Time: 10 Feb 2001@0800).

Include appropriate punctuation. For example, the prompt for Patient Weight may read "Enter the patient weight in pounds:" and prompt for a numeric response. This field may contain the label "lbs" that will display on reports/inquiries following the numeric response entered (125 lbs).
