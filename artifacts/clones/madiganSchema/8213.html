.09do not transmit flag0;9SET OF CODES1:USER REQUESTED THAT CART LIST NOT BE SENT TO ROBOT
Indicates whether or not to send an HL7 message to the ROBOTRX system when running the Cart List or Update Cart List and building the batch file. This field is set to 1 when a user answers "no" to the prompt "Send Cart List to ROBOT RX (Y/N)?" The batch is built, and may be viewed using the option SRX, where it is displayed with the text "SEND NOT REQUESTED" in the STATUS field. If a user selects that batch batch manually to be transmitted to the robot, the flag is set to 0 (zero) to indicate that the request is no longer in force. If, when the batch is built, it contains no drugs stocked in the robot system, the text will read, as usual, "NO ROBOT DRUGS".
