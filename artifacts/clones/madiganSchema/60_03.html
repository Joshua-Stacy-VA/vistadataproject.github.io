.01collection sample0;1POINTER62ABEnter the appropriate collection sample(s) for standard processing in the lab. These are the common names the phlebotomists and ward personnel use and would recognize. The first entry will be the "default" collection sample. This field points to the COLLECTION SAMPLE file so the entry must exist in the COLLECTION SAMPLE FILE. In the case of Microbiology the collection sample may or may not be defined. If the collection sample is not defined, the HCP will be prompted to enter the collection sample. The HCP will choose the collection sample from entries in the COLLECTION SAMPLE file.

3special lab handling0;4FREE TEXTCEnter a brief description of instructions for proper handling of this sample. (EXAMPLE: Place sample on ICE) This field will display on the Nursing DO list and on the collection list. The instructions will also display during order entry.
5ward remarks1;0WORD-PROCESSINGEnter ward related information about the test. For example, enter the time a drug is to be drawn after a dose or the preservative needed for a 24hr. urine collection. This field displays when the option "Lab Test Information" is selected which provides on-line dictionary of instructions for users needing information about lab tests. The users can also access the on-line dictionary during order entry by entering a "?" before the name of the lab test being ordered. The same Lab Test Information display will appear without leaving the order entry option.
6lab processing instructions 2;0WORD-PROCESSINGEnter lab-related test information important to lab users processing specimens for this test. For example, enter the name of the reference lab that performs this test or special handling instructions. Information from this field serves as an on-line dictionary of details relating to laboratory processing and is also available in option LTI (Lab Test Information).

The prompts presented can be found in the Order Required Data set option (LRS). Entries in the LRS option consist of prompts specified in the Order Required Data option (LRD).
