description
This file contains the allowable film sizes that the technician can choose from when s/he enters film data for an exam. If possible, it is a good idea to assign film types names that have unique characters at the beginning. If the first few characters are not unique, then the technician will always have to respond to another 'Select' prompt when entering only the first few characters. To avoid this second prompt, the technician will have to enter many characters. Entries in this file should not be deleted. Adding new entries is always allowed.
