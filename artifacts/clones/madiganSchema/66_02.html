2.08result optional0;812Enter 'NO' in this field or leave blank if this is a mandatory test. If no data is reported, the default status is "pending" and the test will appear on a Pending List until it is certified. When YES is entered in this field, the test will not appear on a Pending List. In addition, optional tests will not be available for result entry once all of the required tests on a panel have been certified and the nightly accession rollover task has run. An example is the test BASO on the Differential. The entry should be YES for this test because not Differential will have a result for BASO.
