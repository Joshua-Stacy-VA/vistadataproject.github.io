name
ANCILLARY PROCEDURE
number
108.1
location
^ORD(108.1,
description
This file holds those procedures which are not included in the files of the automated ancillaries. For example, respiratory therapy and cardiology orders (EG. EKG, Pulmonary Function tests, Consultations) These Orders may either be scheduled procedures or not. If scheduled, they will go through PAS and the procedure must be part of a clinic known to PAS.
