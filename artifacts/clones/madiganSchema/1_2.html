description
This file contains the data which facilitates the ability to create What You See Is What You Get (WYSIWYG) input screens, called Windows, which make up a Gallery. The essential data includes X and Y fields in order to store location coordinates, pre- and post-action code, which denotes MUMPS code that will be executed before and after a field is accessed, and additional parameters which are necessary in order to create a gallery.
