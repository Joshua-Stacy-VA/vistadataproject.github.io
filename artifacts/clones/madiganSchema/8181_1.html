PROVIDER MERGE STATUS
number
8181.1

^DODMR(8181.1,
description
The PROVIDER MERGE STATUS file is the most comprehensive file of the Provider Merge function. The name, start and stop data, and current status data are found in this file. This information is used to track and identify the progress of the Merge.
