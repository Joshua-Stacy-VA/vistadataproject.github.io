description
The GEOGRAPHIC LOCATION file contains a list of all fifty states and abbreviations, state codes, and county information. This file also contains certain continents and countries outside the continental United States. The file is pointed to by a number of files (i.e. PATIENT, MTF, PROVIDER ...). The GEOGRAPHIC LOCATION file is one of the common files shared among the functional areas for CHCS. It is a standard file (classified as Group 1) and cannot be modified by the site.
