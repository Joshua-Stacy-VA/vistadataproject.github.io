5PROCESS QUEUE0;512Answer 'NO' to capture triggered events in the Event Queue but not create data records. Answer 'YES' to capture triggered events in the Event Queue and to create the appropriate data records for transfer to the MCSC System. This parameter should be set to 'NO' during the initialization phase and set to 'YES' after the initialization records have been created and successfully transferred to the MCSC System. When set to 'YES' all events captured in the Event Queue will be processed immediately for batch transfer to the MCSC System.
