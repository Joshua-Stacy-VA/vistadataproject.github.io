description
This file contains the body part index term for the images captured by VI. It also contain the corresponding values in ICD, LOINC, SNOMED CT, and DICOM standard. The information is stored in DICOM header during the conversion of image files in JPEG or TIFF formats to image files in DICOM format. The Body Part Term could be used to sort and retrieve group of images for educational and research uses when the corresponding information is stored in VistA Imaging data files.
