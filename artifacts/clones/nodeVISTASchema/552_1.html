description
This file is maintained at the Consolidated Mail Outpatient Pharmacy host facility and contains summary information for all data transmissions received from the medical centers. This information is used to manage the download of transmission data to the automated dispensing system and to control the purge of the CMOP DATABASE file (#552.2). Data in this file is maintained by the Consolidated Mail Outpatient Pharmacy software and must not be edited using VA FileMan.
