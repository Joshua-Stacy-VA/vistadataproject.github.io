description
This file holds information extracted nightly by a special pharmacy program to allow easier and quicker reporting of pharmacy activity the next day. When the daily information accumulates in excess of one month's information, it is condensed into a monthly total and the daily information is purged. This file serves as the principal database for the cost analysis programs.
