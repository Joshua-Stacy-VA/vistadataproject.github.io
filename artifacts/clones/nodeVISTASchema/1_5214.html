description
Strategies for converting base values to external values. In FileMan they are used to convert references to pointers to their text values. They are also used for the SET OF CODES type. SQLI projects pointer and set of codes as calls to $$GET1^DIQ, variable pointer into calls to $$EXTERNAL^DILFD. Vendors and other users of SQLI may choose to implement their own conversions to improve performance.
