description
This file is used in the calculation of the charge for an ambulatory surgery performed on any given date. This file contains time sensitive data with each entry defining a locality modifier and wage percentage for a division. There may be multiple entries for each division indicating changes effective on a particular date. Per VHA Directive 10-93-142, this file definition should not be modified.
