description
ScaleGroups are collections of MH SCALES file (#601.87) that are logically displayed together. Information needed to graph the scales are stored here. This includes the ordinate title, minimum and maximum values, graph increments and grid lines. All scales need to be associated with a MH SCALEGROUP to be displayed. MH SCALEGROUPS file (#601.86) and MH SCALES file (#601.87) allow multiple scoring possibilities within a single instrument.
