description
This file holds the rules that determine the conditions necessary for a notification or some other action to be performed. Rules consist of a set of Conditional Elements and a set of Element relationship expressions. When an element is found to be TRUE for a patient, it is compared to the other TRUE elements for the patient by evaluating all the Element relation expressions that contain it. If an expression is found to be TRUE then all of the actions defined for that relation are performed.
