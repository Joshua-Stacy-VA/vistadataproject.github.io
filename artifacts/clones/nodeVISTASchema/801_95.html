description
This file defines the resolution statuses that should be related to a particular health factor. The resolution status can be derived for most patient findings (visit file helps determine done and historical). In order to know the appropriate resolution statuses for a health factor, they must be defined in this file. This file is for local use. No health factor resolution statuses are distributed in this file.
