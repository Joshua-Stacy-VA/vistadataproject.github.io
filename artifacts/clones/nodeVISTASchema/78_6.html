description
This file contains all the camera/equip/rm that may be used to perform imaging examinations. The 'RADIOLOGY LOCATIONS' file uses this file to indicate which camera/eqip/rm are allowable choices when the technologist attaches a camera/equip/rm to an exam s/he has performed. In other words, when the procedure is performed on a patient at a location, the only valid choices are those cameras/equip/rms associated with that location. A camera/equip/rm can be associated with more than one location. Entries in this file should not be deleted. Adding new entries is always allowed. Data Storage ------------ The data for the 'CAMERA/EQUIP/RM' file is stored in the ^RA(78.6, global. At the present time this file is very static after day-one initialization is complete. However, it still should be journaled. It should also be translated if the operating system supports this function. Input Templates --------------- The rad/nuc med package does not use any input templates associated with this file. Print Templates --------------- The rad/nuc med package does not use any print templates associated with this file. Sort Templates -------------- The rad/nuc med package does not use any sort templates associated with this file.
