description
The RESOURCE USAGE MONITOR (RUM) file contains statistical data for options, protocols or RPCs that is gathered from the working system. The 'RUM Background Driver' [KMPR BACKGROUND DRIVER] compiles data from the ^KMPTMP("KMPR",DLY") global which is created by the %ZOSVKR routine. The 'RUM Background Driver', which is scheduled to run every day, consolidates this data into this file. NOTE: Data within this file should NOT be edited. Legend: PT - Prime Time Workday hours from 8:00 am to 5:00 pm NP - Non-Primetime Workday hours from midnight up to 8:00 am and after 5:00 pm up to midnight WD - Workday Monday through Friday excluding holidays as listed in the HOLIDAY file #40.5 NW - Non-Workday Saturday and Sunday including holidays as listed in the HOLIDAY file #40.5 A workday is defined by a call to the $$WORKDAY^XUWORKDY API.
