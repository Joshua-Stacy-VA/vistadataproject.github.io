description
Complete ICDO-2 Topography from the 1990 International Classification of Diseases for Oncology (2nd edition), World Health Organization. Contains site specific coding for Extent of Disease and AJCC Staging. This file comes with data, which overwrite the site's data. It is not recommended that sites modify this file.
