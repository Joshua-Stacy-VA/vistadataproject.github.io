.01filing error alert recipients0;1POINTER200BThese persons receive alerts from the upload filer process when a document of the given type cannot be filed/located, or has a missing field. If a document being uploaded has a missing/bad title, then alert recipients defined at the title level cannot be found. In this case, recipients named at the class level are alerted. For example, if a Progress Note is being uploaded and has a missing/bad title, then Progress Note-level recipients are alerted. If recipients are not specified, then alert recipients named in parameter UPLOAD ERROR ALERT RECIPIENTS in the TIU PARAMETER file are alerted as defaults.
