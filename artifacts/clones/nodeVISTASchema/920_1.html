Per VHA Directive, this file has been "locked down" by Data Standardization (DS). The file definition (i.e. data dictionary) shall not be modified. All additions, changes and deletions to entries in the file shall be done by Enterprise Reference Terminology (ERT) using the Master File Server (MFS), provided by the Common Services (CS). Creating and/or editing locally defined fields in the file are not permitted. Use of locally defined fields that were created prior to the VHA Directive shall not be supported. This file is a table of standard possible sources from which the information about a particular immunization event was obtained. The data in this file are derived from the CDC-defined Immunization Information Source table (NIP001).
