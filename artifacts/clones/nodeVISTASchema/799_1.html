THIS FILE MUST NOT BE MODIFIED LOCALLY! ONLY AUTHORIZED NATIONAL REGISTRY DEVELOPERS CAN EDIT THIS FILE! This file contains code sets used within different registries. Records of the file are uniquely identified by the type (it associates the set of values with the registry field), registry and item code. The "A" primary key and the "KEY" uniqueness index are used for this purpose. DO NOT DELETE RECORDS FROM THIS FILE IF THEY HAVE BEEN DISTRIBUTED TO THE SITES ALREADY!
