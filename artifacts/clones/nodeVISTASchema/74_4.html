This file points to the Rad/Nuc Med Reports file (#74). It contains only verified reports associated with the various active distribution queues. Input Templates --------------- The package does not use any input templates associated with this file. Print Templates --------------- The following is a list of print templates used by the package: Name Description; Option(s) ---- ---------------------- RA ALL UNPRINTED REPORTS Prints all unprinted reports for a distribution queue. RA PRINTED REPORTS Prints a list of previously printed reports for a distribution queue. RA UNPRINTED REPORTS Prints a list of unprinted reports in alpha- betical order by patient name.
