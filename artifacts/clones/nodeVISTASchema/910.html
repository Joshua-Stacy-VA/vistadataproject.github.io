description
The ESP MASTER NAME INDEX file contains demographic data (e.g., name, social security number, date of birth, age, AKA (also known as or alias) of persons have a record with the VA Police. This file is pointed to by the name fields in the ESP POLICE REGISTRATION LOG file (#910.2), ESP OFFENSE REPORT file (#912), ESP VIOLATION file (#914), and the ESP WANTS & WARRANTS file (#913).
