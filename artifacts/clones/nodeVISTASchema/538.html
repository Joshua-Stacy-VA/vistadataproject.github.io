description
This file is maintained at both the National Database and at the local sites. It contains all information needed for transmission of the National Rollup data to the National Database. Information is gathered into this file through IBQ Utilization Management menus extracting data from the IB Claims Tracking package. This file is maintained by menus in the IBQ Utilization Management Rollup package. Per VHA Directive 10-93-142, this file definition should not be modified.
