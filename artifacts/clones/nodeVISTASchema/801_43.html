description
This file is used to predefine a preferred dialog element or dialog group to represent a reminder finding item. Auto generation of a reminder dialog from the reminder definition uses the dialog in this file in preference to using the Finding Type Parameter's prefix and suffix to create a sentence. The finding items are restricted to finding types that can be used to resolve the reminder from the CPRS GUI. This file is for local use only. It does not contain any nationally distributed entries. Local entries in this file are not exchanged with other sites via the reminder exchange tool.
