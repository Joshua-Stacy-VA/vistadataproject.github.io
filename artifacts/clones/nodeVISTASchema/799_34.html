description
The ROR REPORT PARAMETERS file stores the report definitions that are used by the ROR REPORT SCHEDULE remote procedure to schedule the reports. Records of the file are uniquely identified by the report code (the "A" primary key and the uniqueness index "KEY" are used for that purpose).
