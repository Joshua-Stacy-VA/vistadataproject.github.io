description
This file was created as part of v1.5 of Integrated Billing, in conjunction with the automation of Means Test/Category C billing. This file contains a list of all continuous Hospital or Nursing Home Care patients since 7/1/86 who may be subject to Category C billing. These patients are exempt from the co-payment charges, but not the per diem charges. Patients who change their level of care must be manually unflagged. Per VHA Directive 10-93-142, this file definition should not be modified.
