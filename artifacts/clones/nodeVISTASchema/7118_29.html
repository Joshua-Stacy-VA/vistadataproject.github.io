description
This file contains system specific data. On the central server(s) this file contains data specific to the systems communicating to it/them and variable information required when processing facility data. On VistA systems this file contains data specific to the facility, such as the location(s) of the central server(s). The local system is typically the first entry in the file but there is no requirement for this to be true.
