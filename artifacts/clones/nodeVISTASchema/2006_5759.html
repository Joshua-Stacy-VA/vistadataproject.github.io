description
Entries in this file are locations identified in Radiology's IMAGING LOCATION file (#79.1) for entities that are non-credit locations. These entries will be used by the DICOM Gateways during the ordering process to assign a location to an 'unordered' Radiology study. The DICOM Gateways periodically download this file. Use menu option 'Build Outside Imaging Location File' [MAG BUILD OUT LOC] to populate this file.
