description
This file is a temporary file containing appointments for a given date range in which initialy populated by "VPS GETALL APPOINTMENTS" RPC. This temporary file represents appointment queue displayed in Vecna's VPS Patient Appointment Queue application. This temporary file will be updated periodically by Vecna's VPS Patient Appointment Queue by calling "VPS GET UPDATED APPOINTMENTS" RPC. The main purpose of this file is to minimize number of records returned to the Vecna application. The "VPS GET UPDATED APPOINTMENTS" RPC will use this file to filter out the non-modified appointments displayed in the Vecna's VPS Patient Appointment Queue.
