description
This file is used by the Consolidated Mail Outpatient Pharmacy host facility to maintain information on all participating medical centers which transmit outpatient prescription data for dispensing by the automated system. Only data received from medical centers designated as active in this file will be accepted by the host facility for processing. Data in this file is maintained by the Consolidated Mail Outpatient Pharmacy software and must not be edited using VA FileMan.
