description
Contains the Sub-headers and Procedures associated with each ambulatory surgery check-off sheet. Defines the relationship between check-off sheets, Sub-headers, and Procedures. Also defines the print order of each entry on the check-off sheet. Per VHA Directive 10-93-142, this file definition should not be modified.
