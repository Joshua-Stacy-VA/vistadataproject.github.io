.01procedure code0;1POINTER509850.4BThe PROCEDURE CODE field contains a CPT procedure code for this exam. QUASAR accepts multiple CPT-4 codes and displays a sequential count of entered procedures. QUASAR allows the same CPT-4 code to be entered more than once if the second and subsequent codes are enclosed in quotation marks (e.g., "92507"). If this is not done, QUASAR assumes that the entry is a mistake and displays the entered code followed by double slant bars (e.g., 92507//). This signals you that this code has already been entered for this visit.
.02*modifier0;2FREE TEXTThe MODIFIER field contains the letter modifiers for some CPT-4 procedure codes. The CPT-4 code set does not provide codes to describe the full scope and complexity of procedures performed by audiologists and speech-language pathologists. Recognizing this deficiency, modifiers are provided to more fully describe procedures or to provide procedures where none exist. The MODIFIER field has been marked for deletion. This field is not used by QUASAR V. 3.0. Eighteen months after the release of QUASAR 3.0, this field can be removed. Any Modifiers already defined for Procedure codes pre Version 3.0 will however be retained for the integrity of historic data.
