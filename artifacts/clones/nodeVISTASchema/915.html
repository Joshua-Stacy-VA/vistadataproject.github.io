description
The ESP OFFENSE CODES file contains appropriate offenses contained within VA Regulations 1.218, United State Codes, Titles 18 and 21. The entries contained within the released version of the program are those offenses considered to be standard at all VA facilities. The Chief of VA Police can add any local codes (e.g., VA facility, city, county or state) that might be enforced at the local site.
