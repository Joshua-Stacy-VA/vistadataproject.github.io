description
This is the Data file for the VA Point of Service (VPS) kiosk application. Data is stored using VPS remote procedure calls that are invoked by the Vetlink Kiosk (patient-facing) or staff-facing client interface. The Vetlink system is used by the patient or staff resource at the medical center or clinic. One of the functions of the Vetlink system is for the patient/provider review of the patient's medication history and allergy history. Each review is saved by Vetlink and then sent to VistA for storage using RPC Broker. The data stored in this file is sorted by PATIENT where each patient entry has one or more Medication Review Allergy Review (MRAR) session instances and these MRAR instances are sorted by transaction date/time. Each session represents a complete or incomplete MRAR.
