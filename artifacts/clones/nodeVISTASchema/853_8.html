This is the Clinical Survey Questionnaire data file for VA Point of Service (VPS) Kiosk application. VetLink (patient-facing or staff-facing) uses Remote Procedure Call (RPC) API to store the patient response as well as the question in the survey. The data stored in this file is sorted by patient and by survey. Each patient will have multiple surveys and each survey contains multiple questions and responses.
