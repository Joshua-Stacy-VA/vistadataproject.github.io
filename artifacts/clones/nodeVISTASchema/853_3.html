description
This file contains a list of the various indicators that Vecna will send to VistA for the accurate capture of discrepancies and non-discrepancies. A fundamental objective of MRAR is to identify allergy data omissions and medication adherence discrepancies. The Kiosk (patient facing) and the staff-facing interface allows a veteran, staff, or provider to change an allergy status (e.g., allergic to non-allergic).
