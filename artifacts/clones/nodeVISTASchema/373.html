description
This file is used to store charge data from various VistA applications so that a background process can create an HL7 P03 message from each record. The messages are batched and sent to the external medical billing system. Under no circumstances may records be created or modified directly via FileMan or other methods. Record creation and data update is allowed only through CHARGE^IBBAPI. Per VHA Directive 10-93-142, this file definition should not be modified.
