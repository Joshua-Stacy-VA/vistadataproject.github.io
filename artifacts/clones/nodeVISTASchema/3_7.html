description
This file holds pointers to the message file, according to users. Each mailbox entry corresponds to a user. POSTMASTER is a special user who controls the network communications queues. Each user is automatically given two baskets: IN and WASTE. All incoming messages go into the users IN basket. When the user deletes them, they are temporarily stored in the WASTE basket. Users may create new baskets as they wish.
