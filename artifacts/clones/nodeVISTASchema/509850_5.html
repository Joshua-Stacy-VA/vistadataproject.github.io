description
The A&SP Procedure Modifier file contains a list of the CPT and HCPCS Modifiers that are appropriate to Audiology and Speech Pathology. This file restricts the selection of Modifiers when entering CPT Procedures within the New Visit option and Edit Existing visit option. The file will be referenced when the user is either selecting a Modifier or requesting a Help list for Modifiers whilst within the New Visit or Edit an Existing Visit options. This file was introduced within Version 3.0 of QUASAR. Data in this file SHOULD NOT be altered through the use of VA FileMan; input should take place ONLY through the QUASAR menu options. Per VHA Directive 10-93-142, this file definition SHOULD NOT be modified.
