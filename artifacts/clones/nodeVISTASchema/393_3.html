#NameLocationTypeDetailsIndexDescription
.01name(+)0;1FREE TEXTBThis field contains the names of the reports or Items that will be tracked by the IRT package.
.02abbreviation0;2FREE TEXTThis field contains the three character abbreviation for the type of report. It is used on the outputs.
.03days for dictation0;3NUMERICThis field contains the number in days before the report being tracked is considered incomplete for dictation.
.04days for signature0;4NUMERICThis field contains the number in days before the report being tracked is considered incomplete for the physicians signature.
.05days for review0;5NUMERICThis field contains the number in days before the report being tracked is considered incomplete for review, if the facility reviews.
.06category(+)0;6POINTER393.41This field is a pointer to the TYPE OF CATEGORY file 393.41. It stores the category that the deficiency falls under.
.07track deficiency(+)0;7BOOLEAN0:NO
1:YES
This field turns the tracking of the deficiency on or off. If this field is set to 'Yes' or 1, the deficiency will be one that is tracked by the site. If this field is set to 'N0' or 0, the facility will not track this deficiency and this deficiency will not appear as a choice in any of the Enter/Edit options.
.08standard deficiency?(+)0;8BOOLEAN0:NO
1:YES
This field is checked by the IRT background jobs, Options: (IRT Update Std. Deficiencies & IRT Update Std. Def. Background Job). If the field is set to 1 (YES) this deficiency is considered a standard deficiency by the site. The background job will then check this field and create an entry for the patient by admission for the standard deficiency. If this field is set to 0 (NO), it is not considered a standard deficiency and therefore no IRT entry will be made for that deficiency.
.09uneditable deficiency0;9BOOLEAN1:YES
0:NO
This field is not seen by the user. It distinguishes the deficiencies that are transported with the TYPE OF DEFICIENCY file 393.3. Those deficiencies are highlighted on the screen under the Add a New/Edit Deficiency option. This is the field that is checked for highlighting purposes.
