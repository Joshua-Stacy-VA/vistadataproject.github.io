This file contains data which are passed into IB NCPDP APIs by outside applications - E CLAIMS MGMT ENGINE and OUTPATIENT PHARMACY (see IA # 4299). Data stored in this file are used for IB ECME EVENT report to provide details about E-Pharmacy claims processing and about communication details between IB and the applications listed above. The data in this file is populated internally by the IB application (data not directly entered by the user). Per VHA Directive 10-93-142, this file definition should not be modified.
