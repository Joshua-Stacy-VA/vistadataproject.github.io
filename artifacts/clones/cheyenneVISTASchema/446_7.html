This file contains report data needed by the Clinical Logistics Report Server (CLRS). The PRCPLO CLO GIP OPTION runs the report that clears and then populates this file. The Prosthetics and Clinical Logistics Office (P&CLO) requires this option to be run on the first day of each month.
