DSIR FINAL FOIA NUMBERS
number
19620.3
location
^DSIR(19620.3,
description
This file contains the computed data for FOIA reports along with offsets to the calculations and a locking mechanism to prevent changes after the FOIA report has been run. The fields on the zero node ofthis file identify which anual FOIA report is stored in the record and the status of the lock for this report. The remaining fields which end in the word REPORT are the values calculated during the FOIA extract process. The fields which end with the word MANUAL are the values from the offset screen in ROI which are then added to their associated REPORT fields to create the final FOIA numbers for the FOIA report.
