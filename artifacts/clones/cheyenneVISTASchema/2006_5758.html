description
The DICOM RADIOLOGY PROCEDURES file (#2006.5758) contains the list of orderable radiology procedures, along with related data that is needed for the Importer to place orders on VistA. This file resides on the DICOM Gateway and several of the fields will be redundantly storing the same information for pointer fields. This is required to continue image processing when a VistA connection is lost. Information in this file is a combination of data from files RAD/NUC MED PROCEDUES (#71) and OUTSIDE IMAGE LOCATION (#2006.5759). The data in this file will be periodically updated via a remote procedure file to VistA.
