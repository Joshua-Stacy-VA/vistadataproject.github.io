description
The ADP EQUIPMENT file (2056.31) is a master list of ADP EQUIPMENT for the VHS&RA ADP Tracking System. It is distributed to the field with entries and the associated data of centrally purchased DHCP ADP equipment as well as local purchases bought by facilities that have been added to the file. When a facility is creating or updating their individual inventories in the VHS&RA ADP TRACKING SYSTEM file (2056), an item is selected from this master file (2056.31) and the corresponding data for that model number is then "defined" into the VHS&RA ADP TRACKING SYSTEM file (2056) for that facility. It contains components, CRTs, printers, micom equipment, communication devices and "other" IO equipment. For each model number the following are defined: general category, type, manufacturer, description, average cost and VA System category. The site managers may add entries to this file.
