description
This file maps the CPT codes used in Dental to the ICD9 diagnosis file and to file 221. This file is used by DSS, Inc to interface the Dental package to the DSS Dental-GUI. The linkage here allows the creating of Dental Activity (file 221) records by selecting CPT codes. The ICD codes are used by the DSS-GUI to enable the creation of workload from the GUI. This file is controlled by VA Dental Central Office. It should not be modified except through use of the Dental DRM GUI. Supported DBA Integration Agreements 2815 - allows for pointers to the CPT file (#81) 10082 - allows for pointers to the DIAGNOSIS file (#80) allows for direct global reads of entire zeroth node
