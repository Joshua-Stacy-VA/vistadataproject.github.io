This file describes the parameters for controlling the Printing of Progress Notes.
Fields
#NameLocationTypeDetailsIndexDescription
.01hospital location(+)0;1POINTER44BThis is the Hospital Location to which the parameters apply.
1.01progress notes1;1BOOLEAN1:YES
This field should be set to 'YES' to enable batch printing of progress notes from the [TIU PRINT PN WARD] and [TIU PRINT PN OUPT LOC] options. These options are intended to be used by MAS personnel printing CHART copies.
1.02last progress note printed1;2FREE TEXTThis field is used to track the last progress note printed for this location. The option resets this field after each print. If the field is null the option will prompt to select a date.
1.03progress notes default printer1;3POINTER3.5This field is a pointer to the device file. The default the site would like to have presented when batch using the TIU progress notes batch print options [TIU PRINT PN OUTPT LOC] and [TIU PRINT PN WARD]. The user may select another printer at the time to job is queued.
3exclude from pn batch print3;1BOOLEAN1:YES
If you answer yes to this prompt, progress notes for this location will not be included in the progress notes outpatient batch print job [TIU PRINT PN BATCH]. Do this if you want to print the CHART copies of the notes for this location in the clinic and not in the file room.
