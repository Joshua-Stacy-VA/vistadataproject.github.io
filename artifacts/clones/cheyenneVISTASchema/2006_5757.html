description
The DICOM RADIOLOGY PROCEDURE MODIFIERS file (#2006.5757) resides on the DICOM Gateway. It contains the list of Radiology procedure modifiers that may be used when placing orders for VistA on the DICOM Gateway. This file is periodically updated on the Gateway by a remote procedure call to VistA to read file Radiology's PROCEDURE MODIFIERS (#71.2). This allows the Gateway to continue processing if a VistA connections is lost.
