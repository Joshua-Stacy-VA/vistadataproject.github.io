THIS FILE MUST NOT BE MODIFIED LOCALLY! ONLY AUTHORIZED NATIONAL REGISTRY DEVELOPERS CAN EDIT THIS FILE! The ROR AIDS INDICATOR DISEASE file contains definitions of the AIDS indicator diseases referenced by part VIII of the HIV CDC form. Also see the AIDS INDICATOR DISEASE multiple (10) of the ROR ICR STUDY file (#799.4).
