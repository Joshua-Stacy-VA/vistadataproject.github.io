description
This file contains the names of the distribution queues and the category of reports for each queue. Data Storage ------------ The data for 'REPORT DISTRIBUTION QUEUE' is stored under the ^RABTCH (74.3 global. Input Templates --------------- The following is a list of input templates used by the package and the entry in the OPTIONS file (#19) that uses the template. Name Description; Option(s) ---- --------------------- RA DISTRIBUTION EDIT Used to edit the report distribution queue. RA DISTRIBUTION LOG Edits the activity log of the distribution queue whenever reports are printed. This is done automatically by the system whenever reports are printed or re-printed from a queue. Print Templates --------------- The following is a list of print templates used by the package: Name Description; Option(s) ---- ---------------------- RA DISTRIBUTION Prints a list of names of the distribution queues and category of reports associated with each queue. Sort Templates --------------- The package does not use any sort templates associated with this file.
