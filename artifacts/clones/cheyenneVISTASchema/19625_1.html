description
This file contains ICB report data that is compiled by a background job. It currently only contains Exception Report data from the DSIV NIGHTLY REPORT option, but it could also contain Audit Report data compiled from file 19625 in the future. The Exception Report is a listing of patient appointments that were not acted on (placed in the buffer if needed) in a timely manner. Since it looks through all appointments for the day and processes clerk actions it can take more than a few minutes to run if all clinics are run at a site (there are inclusion/exclusion parameter lists). The nightly job should be scheduled during off peak hours and after the scheduling job has changed the appointment status of appointments that were not checked in-out for the day (automatic process).
