description
This ICU Outcome Improvement Project file stores selected laboratory results for critical care episodes identified by the project's extraction process. Data is acquired from the national LAB DATA File (#63) and stored in this file referencing the IPEC STUDY WINDOW File (#539394.2) scheme unique to each case. Results mapping relationships between the LAB DATA File (#63) and the LABORATORY TEST File (#60) are resolved by SVT2 LAB RESULTS MAP File (#539394.45) and the IPEC VISTA staff through the use of server options at remote sites and local, center options. Results are recorded for the following NATIONAL laboratory tests of interest: Glucose, Urea Nitrogen (BUN), Creatinine, Sodium, Cholesterol, Albumin, Bilirubin, SGOT (AST), CPK, WBC, HCT, PLT, Prothrombin Time (PTT), FIO2, pH, PCO2, PO2, CO2, Lactic Acid, HDL Cholesterol, C-Reactive Protein, TSH, Cortisol, T-4 Binding Globulin, Trimethadione, LDL Cholesterol, Capillary Glucose (Fingerstick), INR, D-Dimer, HGB A1C, B-Type Natriuretic Peptide. ipec-cvamc;st;v3.1;7.9.07
