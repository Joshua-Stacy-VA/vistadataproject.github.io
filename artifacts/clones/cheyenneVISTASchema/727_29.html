description
This file contains the set of laboratory LOINC codes tracked by the Decision Support System. It contains corresponding DSS test numbers, DSS test names, reporting units, and LOINC name. Entries in this file are determined by the DSS Program Office and should not be changed or deleted. New entries in this file should not be made.
