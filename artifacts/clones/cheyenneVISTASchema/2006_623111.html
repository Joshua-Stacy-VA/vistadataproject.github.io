This is a flag to indicate that the older Pixel Spacing field's value should be used when calculating distance measurements for this model/software version. This corrects for a bug in the Fuji CR image header in which the incorrect values were stored in the newer field (Imager Pixel Spacing--0018,1164), whereas the older field (Pixel Spacing--0028,0030) still carried the correct value; answering YES to this prompt enables the program to prefer the older field when calculating a measurement for images acquired with the Fuji bug.
