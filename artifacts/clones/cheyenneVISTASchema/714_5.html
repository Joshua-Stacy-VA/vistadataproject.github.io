EAS LTC MONTHLY MAX CALCULATIONS
number
714.5
location
^EASPFS(714.5,
description
This file stores the data relating to maximum Long Term Care (LTC) copayment calculations. When an HL7 message is received and an EAS protocol event is invoked the Enrollment Application System will calculate the LTC copayment amounts and store the data received and calculated in this file. The data is later on extracted and sent back to the requesting application via HL7 messaging.
