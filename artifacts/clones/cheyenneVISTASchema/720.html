description
This file contains the categories and procedures associated with DSS units in the Event Capture software. All entries in this file are procedures. If flagged using the CATEGORY (Y/N) field (2) these entries will be recognized as categories and procedures. Entries in this file are locally entered and edited. A synonym field is provided for local use in identifying these entries within various hospital departments. **NOTE** This file is being exported with Event Capture Version 2.0 for use in converting existing data, at the pilot sites, to current data formats. This file will be removed in a future release of the Event Capture software. ** Per patch EC*2*5, this file has been starred for deletion. **
