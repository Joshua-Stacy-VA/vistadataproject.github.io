description
This file captures flowsheet data in support of the the Anticoagulation Management Program (AMP), which is a Joint Commission National Patient Safety Goal initiative to use a standardized process for the management of their patients on extended anticoagulation therapy. AMP runs as a Windows Application from the CPRS Tools menu, and assists Clinicians with the expeditious documentation of visits to Anticoagulation Clinics.
