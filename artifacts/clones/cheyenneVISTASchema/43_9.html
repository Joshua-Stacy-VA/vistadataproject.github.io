This file contains the various movement types which a patient may be given while in inpatient status. This file should not be added to or altered in any way by the facility. Doing so will negatively impact virtually every output produced by the MAS module related to inpatient care. Every entry in the admission, transfer and discharge type file should point to a corresponding entry in this file.
