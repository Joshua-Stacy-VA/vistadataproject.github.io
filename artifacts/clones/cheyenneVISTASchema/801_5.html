description
This file contains a small amount of static data. Entries are entered and removed as Reminder Geriatric Extended Care Dialogs are processed by the CPRS GUI. Its main purpose is to keep track of and supply an Encounter Date/Time to the GUI interface so that the date/time can be added later to fields in the V HEALTH FACTOR file.
