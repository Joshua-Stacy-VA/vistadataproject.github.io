description
The Quality Improvement Checklist file contains station specific data for the semi-annual survey. The station's answer for each question on a selected survey is stored in this file. Also included are fields for approval/sign-off, sign-off date, and the date the data is transmitted to the centralized data base. Data in this file SHOULD NOT be altered through the use of VA FileMan; input should take place ONLY through the Quality Improvement Checklist menu options.
