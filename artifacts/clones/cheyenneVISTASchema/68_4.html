description
This file allows the lab to customize the short accession list. It will allow up to a 10-line heading of the accession. This heading area may be separated in a maximum of 4 columns. Each column allows an entry that may be used to define the supervisor of an accession area, or put in a space to allow the technologist to designate which shift, etc.
