This file contains a list of claim adjustment group codes. The codes in this file are used to identify the general category of payment adjustment. The source of the codes is the National Electronic Data Interchange Transaction Set Implementation Guide for the Health Care Claim Payment Advice (ASC X12N 835). It is available from Washington Publishing Company. The applicability for use by VA on fee claims is provided by the Fee Replacement Project Management Office in Denver which represents the VA Chief Business Office. This file is exported with data and the contents should not be locally modified.
