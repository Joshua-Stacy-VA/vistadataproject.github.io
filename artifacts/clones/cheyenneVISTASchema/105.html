description
This file contains Micom configurations to be automatically transferred to the Micom by the loader routine. Each configuration has commands and messages. Commands are stored in the order they are to be transmitted to the Micom. The system welcome message is a multi-line text. Individual class messages are 1 - 29 characters in length. The configuration identifier may be a number or name 1 - 8 characters.
