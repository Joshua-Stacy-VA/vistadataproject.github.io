description
The Quality Improvement Checklist Survey Date file contains all data necessary to define a specific survey. The survey is conducted semi- annually. The file contains the beginning and ending dates of the survey, the number of questions in the survey, and the survey status. This file arrives at the VAMC with data, and SHOULD NOT be altered locally.
