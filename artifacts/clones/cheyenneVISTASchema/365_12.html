description
This is a standard file exported by the IB package. It contains all payers which can be communicated with electronically for insurance identification and verification. Do not add, edit or delete these entries except through the provided edit options. At this time, WebMD is the only vendor VISTA is communicating with electronically and so this file will contain those payers with whom WebMD currently electronically communicates. Per VHA Directive 10-93-142, this file definition should not be modified.
