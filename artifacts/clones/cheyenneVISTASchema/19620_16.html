This file contains the default layout and text of the letters printed in the ROI Plus application. This file may be edited at the site. Letters contained in this file are copied from the DSIR LOCAL LETTERS file for modifications that contain site specific information. (Name, Address, etc.) This file may also contain letters defined by the site.
