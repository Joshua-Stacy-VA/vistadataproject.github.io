description
This file is used to define the shipping containers used by the Lab Shipping Manifest. Shipping containers can be of three types: Primary: specimen shipped in original collecting container. Aliquot: specimen transferred to this type of ontainer prior to shipping. Packaging: container used to hold and contain other specimen containers during shipping.
