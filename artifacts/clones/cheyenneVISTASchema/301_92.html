description
This file contains information pertaining to the demographic fields sent in HL7 segments from the IVM Center to the VAMCs. This table contains parameters which are used during the receipt and upload of demographics data from the IVM Center. This file should not be altered in any way. Per VHA Directive 10-93-142, this file definition should not be modified.
