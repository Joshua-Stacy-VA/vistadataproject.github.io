description
This is the main file for the Boise Cost Distribution Report (CDR) package. Data is stored for the following purposes: 1. Employee data for computation of %FTEE and %Personal Services dollars in selected subaccounts. 2. Control Point data for computation of All Other Costs (AOC) dollars in selected subaccounts. FILE MANAGER should NOT BE USED to edit this file. This file is used for data storage for the CDR package and all entries, edits and calculations are performed by the CDR package routines (AEUCDR*). If edits have been erroneously made using FileMan in this file, use the CDR manager menu option to re-calculate the effected Cost Center.
