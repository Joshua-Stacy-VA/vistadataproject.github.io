description
The TIU TEXT EVENTS file is used to store data to be used in searching for certain text within Progress Notes in order to alert treating specialty areas of monitored concerns when a patient is seen. Setting up entries in this file will cause alerts to be sent to providers and services to indicate a possible care opportunity.
