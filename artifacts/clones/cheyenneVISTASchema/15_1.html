Files > DUPLICATE RESOLUTION

name
DUPLICATE RESOLUTION
number
15.1
location
^VA(15.1,
description
This file is used to handle duplicate checking and merging of files that have entries in the Duplicate Record File. It is meant to provide the overall control information that would be used to first identify duplicates within a file, e.g. Patient File, and then to merge the entries.
