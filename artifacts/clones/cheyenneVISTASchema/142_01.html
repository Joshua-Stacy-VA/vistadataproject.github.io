3time limit0;4FREE TEXTThis is a code describing how far back the component should search for a given patient's data. If no limit is indicated, the component assumes an indefinite time interval. Exceptions are the Outpatient, Unit Dose and Intravenous Pharmacy components, which will show active orders only when no time limit is entered.
