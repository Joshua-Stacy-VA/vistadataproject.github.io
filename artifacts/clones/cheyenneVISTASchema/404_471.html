description
This file contains a log of all PCMM HL7 Transmissions that have occurred at the medical center. It is also used to track PCMM HL7 Transmission errors that have been transmitted back to the medical center from the Austin Automation Center (AAC). The data in this file is maintained by the PCMM software module and should not be altered.
