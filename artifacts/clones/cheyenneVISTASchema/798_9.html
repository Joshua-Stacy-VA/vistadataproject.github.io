description
Lab search criteria are stored in this file. These criteria are referenced by the selection rules and used in the search for Lab results. It is possible but not recommended to use the same criterion for several different registries. Records of the file are uniquely identified by the criterion name. The "A" primary key and the "B" uniqueness index are used for this purpose.
