This field contains the hardware conditions parameter that tells the system the type of operating conditions that are in effect. The available conditions are: 'N' for normal and 'E' for emergency. Under NORMAL conditions, the system uses the parameters set up in the files 195.1, 195.2, 195.9 to determine which input and printer devices to use. Under EMERGENCY conditions, the system ignores all the location parameters set up and the user is prompted for this information.
