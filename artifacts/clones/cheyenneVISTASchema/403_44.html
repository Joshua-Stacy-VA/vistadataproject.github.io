description
This table file contains the scheduling reasons. If an entry needs to be added, modified or deleted a patch will be issued instructing the site how to make the change. Otherwise, this table should not be edited in anyway by the site. Scheduling Reason event code 'HL-S15' entries in this file correspond to the entries in the Cancellation Reasons file (#409.2).
