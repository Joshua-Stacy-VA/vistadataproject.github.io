description
This file contains all DSS units defined for use in the Event Capture software. Entries cannot be deleted but may be inactivated using the Event Capture software. The fields representing service, medical specialty and cost center are required for each DSS unit. These fields can be edited using options on the Event Capture Management Menu but cannot be deleted. This file should NOT be modified directly using VA FileMan. The Event Capture Management Menu provides the options necessary to create, edit, and inactivate all DSS units.
