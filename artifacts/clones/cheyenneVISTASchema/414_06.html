description
This file is designed to hold the various types of transactions that support communication for the IFCAP and the Electronic Contract Management System (eCMS) interface. Reports from this file will be used by the IFCAP users to follow the interface activity to assure that communication with the eCMS vendor portal is timely and effective, and to provide contact information if needed.
