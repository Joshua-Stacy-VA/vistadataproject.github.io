description
Contains updates to the ambulatory surgery procedures that can be billed. The data comes from HCFA in a WP file that is uploaded and entered into this file, generally once a year. The new billing procedures or billing group updates are activated only after the data in this file is transferred to 350.4. This file should be empty except during the loading and transferring of the HCFA updates. Per VHA Directive 10-93-142, this file definition should not be modified. As of 10/1/96 this functionality has not been authorized. The data in this file is deleted by patch IB*2*52 and the file definition will be deleted in a future release.
