description
This file holds a list of Admission types which are used when setting ADT Parameters and admitting a patient. When initializing your ADT system it is important that you review these admission types and determine which types are to be used by your facility by setting the ACTIVE field to '0' if not used at your facility or '1' if used. The information in this file is planned to be incorporated into the FACILITY MOVEMENT file in a future release.

.07point to mas movement type0;7POINTER405.2This field is being provided prior to version 5.0 to allow entry of the new MAS MOVEMENT TYPE file pointer. Prior to version 5.0, the MOVEMENT TYPE (as defined in field .06 of this file) was a pointer to file 43.9 (the MOVEMENT TYPE file). It will now be a pointer to file 405.2 (the MAS MOVEMENT TYPE file). There are several additional entries in this new file, so you may wish to consider changing your present movement type pointer for version 5.0. Do not change the pointer in the .06 field. No changes should be made prior to version 5.0. However, upon installation of version 5.0, if you would like to utilize one of the new movement types, enter that type here. If there is nothing entered in this field, the conversion process will assume that you would like to leave the admission type unchanged. For example, AMBULATORY CARE (OPT-AC) is a new admission type. If you have a local admission type of OPT-AC, you may wish to point to this new admission type rather than the one you had previously chosen.
