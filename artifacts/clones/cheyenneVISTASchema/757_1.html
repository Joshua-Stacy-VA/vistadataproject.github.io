description
This file maps semantic classes and semantic types to the major concept, providing contextual meaning to both the term as well as the words contained in the term. This semantic meaning, when applied to the terms, is used as a filter during term file look-ups.
Fields
#NameLocationTypeDetailsIndexDescription
.01major concept(+)0;1POINTER757BThis is a pointer to the Major Concept, linking the semantic classification with a concept. Only Major Concepts are semantically classified. Synonyms and Lexical Variants adopt the semantic classification of the parent Major Concept.
1semantic class(+)0;2POINTER757.11This is a pointer to the general semantic class defined in the Semantic Class file #757.11 to which the Major Concept belongs. An example of a semantic class is ORG for concepts in the general class of Organisms.
2semantic type(+)0;3POINTER757.12This is a pointer to the specific semantic type defined in the Semantic Types file #757.12 to which the Major Concept belongs. Using the general semantic class ORG (Organism), examples of specific semantic types would be plants, alga, fungus, virus, rickettsia, bacterium, animal, intertebrate, vertebrate, amphibian, bird, fish, reptile, mammal, human, and organism.
