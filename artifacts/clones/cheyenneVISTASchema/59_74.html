name
VENDOR INTERFACE DATA
number
59.74
location

description
This file keeps track of when and for how long the Vendor interface is unavailable. A background process monitors the status of the interface and records in this file when the interface is down,when it becomes available again and the total time it was unavailable.
Fields

2total time not available0;3NUMERICThis field holds the total time in minutes that the Vendor interface was unavailable as determined by the monitoring background process.
