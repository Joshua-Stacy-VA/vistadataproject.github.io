description
This file on FORUM manages the followup automatic MailMan message function for the Satellite TV module. The Satellite TV module is designed so that a listing of all stations who have not input statistics for a program at a certain point in time is generated, and the stations are listed, as recipients, to an automatic MailMan message reminding them to do so.
