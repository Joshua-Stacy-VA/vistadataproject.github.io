description
The Master Name Index File is the center of all VAPD Files. All other files are triggering information into this file within the respective fields. In addition, as information is entered into other files, then the corresponding information from Master Name Index in automatically displayed in that field. For a more detailed explanation of the file, refer to VAPD HELP FILE.
