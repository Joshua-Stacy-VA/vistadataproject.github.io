description
This table file contains types of outpatient classifications. These include Service Connected, Agent Orange Exposure, Ionizing Radiation Exposure and Environmental Contaminants. If an entry needs to be added, modified or deleted a patch will be issued instructing the site how to make the change. Otherwise, this table should not be edited in anyway by the site.
