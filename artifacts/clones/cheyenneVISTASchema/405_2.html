description
This file holds all MAS accepted/approved movement types. All entries in you FACILITY MOVEMENT TYPE FILE (file 405.1) must point to an entry in this file. This file is distributed by MAS and should NOT be altered by any facility. Altering this file can severely corrupt the integrity of your bed control movement options. The internal entry numbers of this file are used by various packages to determine the processing necessary. These numbers and entries must remain as distributed by the MAS package.
