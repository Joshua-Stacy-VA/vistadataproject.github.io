203beginning patient date offset(+)LM;3NUMERICThis is the value that determines the starting date for the list by a patient in PCE Encounter Data Entry. The number is added to today to get the starting day. e.g. If the value is -30 then the starting day is T-30, or 30 days before today. If the valuse is 5 then the starting day is T+5, or 5 days after today. If it is 0 then the starting day will be today.
