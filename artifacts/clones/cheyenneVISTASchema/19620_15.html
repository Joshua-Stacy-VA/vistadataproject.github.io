This file contains the default layout and text of the letters printed in the ROI Plus application. This file is not to be edited at the site. Letters contained in this file will be copied into the DSIR LOCAL LETTERS file for modifications that contain site specific information. (Name, Address, etc.) All information in this file should only be edited through the ROI Plus Application and no other means.
