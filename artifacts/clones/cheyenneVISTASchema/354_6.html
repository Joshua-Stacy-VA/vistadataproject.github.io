description
This file contains the Header and Main body of letters that are generated by the IB package. Sites should edit the header of the letter to their own address. Sites may edit the main body of the letter to change the signer of the letter or add in contact persons and phone numbers. The text of the letters have been approved by MCCR VACO. Per VHA Directive 10-93-142, this file definition should not be modified.

.06job schedule0;6FREE TEXTThis field is used to store mornings (days) for which printed income test reminder letters may be picked up. If there is no value in this field, your income test reminder letters will not print.
.07reprint date0;7DATE-TIMEThis new field was introduced with patch IB*2*34. If the site wishes to reprint all of the letters which were printed on a certain date (due to printer jams, etc.) the date on which those letters were printed should be entered into this field. The next time that the background job prints letters, it will also print letters for exemptions which were marked as having had a reminder letter printed on this date. The background job will then delete the value in this field. This field may be entered through the option Edit Copay Exemption Letter.
