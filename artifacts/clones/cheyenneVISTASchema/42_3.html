This file contains the acceptable types of transfer as used when moving a patient within the medical facility. Each transfer type, with the exception of INTERWARD, must have an associated movement type otherwise it will be treated as an interward for G&L and statistical purposes. The data in this file will be incorporated into the FACILITY MOVEMENT file in a future release.
