description
The EQUIPMENT TYPE file (2056.2) contains a listing of the various equipment types. This file is pointed to by both the VHS&RA ADP TRACKING (2056) and the ADP EQUIPMENT (2056.31) files. The equipment type is defined when an entry is made in the ADP EQUIPMENT file. When an entry is made in the ADP TRACKING file, the equipment type is carried over from the ADP EQUIPMENT file and defined automatically into the appropriate field for that particular model number. ADP Equipment is divided into seven different categories. Some categories have only one type associated with them such as CRTs, printers, micoms and communication equipment. However, components can be of various types e.g. CPUs, disk drives, backplanes etc.
