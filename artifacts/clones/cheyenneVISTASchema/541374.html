description
This file (541374) is used to build the DIET PRINT MANAGER job stack, that prints out all the daily Dietetics Reports and Labels. The DIET PRINT MANAGER runs like Taskman. It does an initial build of the stack at one second after Midnight. It then checks the top of the stack for the next job to be done, working its way to the bottom (the end of the day). This file also points to 541374.1 and 541374.2.
