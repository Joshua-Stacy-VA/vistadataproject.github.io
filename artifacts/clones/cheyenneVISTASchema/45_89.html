45.89

description
This file contains all the ICD codes that require additional questions to be asked. There are three parts to the file 1) the ICD code, 2) the category where the code can be found and 3) an inactivation date. The category field points to the PTF EXPANDED CODE CATEGORY FILE (45.88).
