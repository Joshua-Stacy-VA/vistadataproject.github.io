description
Per VHA Directive 2004-038, this file should not be modified or edited with VA Fileman. This file stores Release of Information (ROI) data collected that relates to a specific patient/drug/insurance combination for the effective dates. A claim that includes a sensitive drug is checked against this file for a billing determination. Claims that do not pass the check are determined to be not ECME billable. Bills that do not pass the check cannot be authorized.
