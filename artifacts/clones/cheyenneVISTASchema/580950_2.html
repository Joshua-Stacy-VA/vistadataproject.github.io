This is the new, primary, transmission file which was implemented with the Version 2.0, rewrite. All bills selected for collection processing are entered into this file. Pretty much all collection processing starts with this file. The DATA DICTIONARY should not be modified NOR should the DATA ELEMENTS be edited or added to directly thru VA FileMan.
