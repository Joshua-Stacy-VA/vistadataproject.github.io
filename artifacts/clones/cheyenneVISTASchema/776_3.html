HL7 System Message Errors are errors discovered by the HL7 system while processing a particular message. Application Message Errors are errors discovered by an appication while processing a particular message. This type of error is defined by the application and usually involve application-specific business rules. Operational Errors are not related to a specific message, rather,they are related to a site. An common example is a logical link that fails. An Event Monitor Notification is any sort of status update message that may be posted by the Event Monitors.
