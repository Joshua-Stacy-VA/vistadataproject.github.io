description
The DICOM OBJECTS TO BE IMPORTED file (#2006.5751) resides on the DICOM Gateway and tracks DICOM objects that have been received and may be imported. Entries are made to this file by reading the DICOM Object files and recording the raw data elements from the file. The information is displayed in listing which are selectable for importing into the VistA database.
