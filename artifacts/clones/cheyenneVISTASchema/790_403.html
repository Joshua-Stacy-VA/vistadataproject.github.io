description
This file stores the notification types which decribe how test or procedure results and treatment needs are communicated to the patient. The notification type may be a letter, personal contact, an organization's name, or other term that describes the communication process. Entries in this file may be tagged to identify the number of attempts associated with the type of contact, e.g., first letter, second letter (certified), 3rd phone call. New Notification Types can be entered into this file through VA FileMan. Synonyms can be entered by the case manager(s) through the Edit Synonyms for Notification Types option. This information is stored in a patient's record through the Add a New Notification or Edit a Notification option.
