The value of this field is a code for the "value representation" of the DICOM element. Possible values are taken from part 5, section 6 of the DICOM standard: AE: Application Entity AS: Age String AT: Attribute Tag CS: Code String DA: Date DS: Decimal String DT: Date Time FL: Floating Point FD: Floating Point Double IS: Integer String LO: Long String LT: Long Text OB: Other Byte String OW: Other Word String PN: Person Name SH: Short String SL: Signed Long SQ: Sequence of Items SS: Signed Short ST: Short Text TM: Time UI: Unique Identifier (UID) UL: Unsigned Long UN: Unknown US: Unsigned Short UT: Unlimited Text

The value of this field provides information about the meaning of the list of possible values. When the value of this field is "D" (Defined list), an implementor may choose to add values to the list for their implementation. When the value of this field is "E" (Enumerated list), only those values specified in the DICOM standard may occur.
6status0;6SET OF CODESR:Retired
The value of this field is a code that indicates whether or not the current entity is "Retired" from the DICOM Standard. When the value of this field is blank, this entity is active. When the value of this field is equal to "R", this entity is retired.
