description
This file will be used to "automatically queue" the options on the QUIC "DHCP Extract Reports" menu. The file consists of the option menu text and a list of variables which must be "pre-answered" in order to queue each extract report to print. The file arrives at the VAMC with some data. Data in this file SHOULD NOT be altered through the use of VA FileMan; input should take place ONLY through the Quality Improvement Checklist menu options.
