description
This file will translate the patient's treating specialty to a DOM/PRRTP/SARRTP code (if applicable), an observation patient indicator, an inpatient/outpatient status indicator, and an observation stop code for use in all DSS extracts. This file was originally called DOM PRRTP SAARTP ETC and has been modified as part of the DSS FY2002 software release.
