description
This file contains one record for each provider id that an insurance company assigns to a facility for ALL billing providers at the facility. Each record can be for an insurance company and any combination of the patient status, form type and care unit. There must be only one record for each combination. This file contains the id numbers for the following levels: o Assigned by insurance company to all providers o Assigned by insurance company, to all providers, one id for each care unit
