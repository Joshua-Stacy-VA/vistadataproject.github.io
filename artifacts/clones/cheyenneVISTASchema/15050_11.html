description
This file contains the Maps (or templates) used to create Cache classes for FileMan files. Each map is created by the START^CASH call, which examines the ^DIC and ^DD structures for the file. This Map can be used to create many different Cache classes, with different options (FLAGS), by using the CREATE^CASH call.
