This file will contain an entry for any workstation that has connected to this site during an Imaging Session. The Workstation information is stored in this file. identified to automatically update its Imaging software. The Imaging System manager at the Medical Centers should periodically review this file to ensure that all the workstations have the latest code. Note: The information in this file should not be edited.
