description
This Inpatient Evaluation Center file addresses the relationship between selected fields in the national Lab Data File (#63) and the Laboratory Test File (#60). While the project has identified specific lab data results of interest that correspond to the nationally furnished lab data fields, it has been found that there are variances in actual lab data fields accumulating such data at many sites. Therefore, this necessitates a mapping facility that will properly direct the project's extraction software at each individual study site. This site specific file when properly initialized, reviewed, and edited by IPEC staff will provide that direction. Server options distributed to remote sites and local, center options provide the tools for thios work. ipec-cvamc;st;v3.1;7.9.07
