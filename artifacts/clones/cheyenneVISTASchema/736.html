description
The Quality Improvement Checklist Sort Data file contains the sort mechanisms necessary to generate the national reports. The file contains all VAMCs which will answer the survey questionnaire, their identifying station number, region, and other sort fields. This file arrives at the VAMC with data, and SHOULD NOT be altered locally.
