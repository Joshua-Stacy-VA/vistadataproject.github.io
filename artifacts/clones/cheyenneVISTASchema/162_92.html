description
Each unauthorized claim is assigned a status. This file contains the possible status' which an unauthorized claim may be assigned. Associated with a status is the status order. This is the order in which a claim may progress. Also associated with each status is the number of days before the claim expires, if applicable. *** ENTRIES SHOULD NOT BE ADDED TO THIS FILE *** Per VHA Directive 10-93-142, this file definition should not be modified.
