description
The ESP CRIME SUB-TYPES file contains subtypes of specific criminal activities which fall under specific crime types in ESP CRIME TYPES file (#912.8). These crime subtypes have been identified by the VA Police policy managers (VACO) and no additional crime subtypes are to be added by the local sites. You CANNOT add entries to this file.
