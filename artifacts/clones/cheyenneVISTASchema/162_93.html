description
Data contained in this file pertains to information needed in order to process an unauthorized claim. Sites have the ability to add local entries. The description of the requested information is printed as part of the text in the letter which is mailed to the submitter requesting the information. Per VHA Directive 10-93-142, this file definition should not be modified.
