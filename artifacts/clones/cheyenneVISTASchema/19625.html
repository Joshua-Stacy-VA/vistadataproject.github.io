description
This file contains audit information for ICB. When Buffer entries (#355.33) are created, audit file entries are created to capture fields for reporting. When Buffer entries are processed only a stub record is left so the audit file allows reporting/auditing for a longer, site-defined period of time. Audit entries are purged manually by the site.
