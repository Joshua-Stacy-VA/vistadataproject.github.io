description
This file contains a list of remittance remark codes. The codes in this file represent non-financial information critical to understanding the adjudication of a health insurance claim. The source of the codes is the National Standard Format Electronic Remittance Advice. It is available from Washington Publishing Company. The applicability for use by VA on fee claims is provided by the Fee Replacement Project Management Office in Denver which represents the VA Chief Business Office. This file is exported with data and the contents should not be locally modified.
