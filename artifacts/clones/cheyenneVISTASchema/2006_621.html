The data in the file can be used to determine what the Dicom-to-TGA processing parameters were used for a given device as of specified dates. This is needed when a TGA-to_Dicom conversion must be performed, or when performing Hounsfield calculations for these older files. The processing parameters are needed for CTs processed prior to Patch 50; after then, the parameters are stored in the individual image text files.
