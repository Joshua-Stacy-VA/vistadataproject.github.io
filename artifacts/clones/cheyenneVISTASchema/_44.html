description
This file stores the characteristics of various file export formats, which are used by the Export tool in building Export Templates to send data to non-M systems.

1field delimiter0;2FREE TEXTContents of the field delimiter is output between each field. Depending on the contents of the SEND LAST FIELD DELIMITER? field, the delimiter may be output after the last field, too. Identify the delimiter either by 1-15 characters not beginning with a number or by the ASCII value of the delimiter. When specifying the ASCII value, use 3 numbers (e.g., '009' for ASCII 9). Up to four ASCII-character values can be specified, separated by commas. If 'Ask' is entered, the user will be prompted for the field delimiter when creating the EXPORT template.
