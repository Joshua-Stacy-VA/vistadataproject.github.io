description
This file contains the responses to requests for the NAS Statistical Report and the NAS Monthly Report. The Statistical Report is stored on node 1. The Monthly Report has three possible formats. Types 1 (facility temporarily not available), 2 (professional capability temporarily not available), 3 (medically inappropriate), and 6 (facility or professional capability permanen tly not available) are stored on node 2. Type 4 (branch summary report) is stored on node 3. Type 5 (reason for issue summary report) is stored on node 4.
