This file tracks users that run canned reports and other options that allow them to see patient notes. At this time (17 April 2002) the only Note Type that has ever been supported is M - Patient Master Problem List and entry is triggered only when a user chooses option 3 (Patient Master Problem List) from the main menu in ADM.
