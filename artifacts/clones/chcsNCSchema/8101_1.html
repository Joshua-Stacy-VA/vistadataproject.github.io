description
This file contains Medical Treatment Facility (MTF) entries that are not part of the standard set of entries contained in File #4. It is classified as Group 3 (entries are built through user-interaction with the system). Lab uses this file to store the MTFs from referral testing. Also, the lab stores the names and addresses of reference laboratories used on the shipping labels.
