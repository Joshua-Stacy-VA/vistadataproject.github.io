.03nrbc test name0;3POINTER60Enter the name of the test used for Nucleated RBCs. This test will be counted on the diff, but the total cells will not advance when the NRBC test is counted.

.05diff test name(+)0;5POINTER60Enter the name of the Differential test as defined in your lab test file. The program needs this name in order to determine which tests are part of the differential when it builds the keyboard entry screens.
.06ua test name(+)0;6POINTER60Enter the name of the Urinalysis test as defined in your lab test file. The program needs this name in order to determine which tests are part of the urinalysis panel.
.07ua macro test name0;7POINTER60Enter the name of the panel defined for the macroscopic tests of the urinalysis. When the panel name is entered here the user can use the UA keyboard option to enter the results for a macroscopic only urine result.
.08ua micro test name0;8POINTER60Enter the name of the panel defined for microscopic tests. This panel is the lab test used by the lab for microscopic only urinalysis. These tests are probably ordered as secondary orders after a positive result on a urinalysis composed of macroscopic only tests. When this panel is added the user can select the UA keyboard option to continue entering the results.
