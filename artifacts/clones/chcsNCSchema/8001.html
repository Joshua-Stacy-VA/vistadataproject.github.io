description
This file stores the info about the SIDR tapes that are generated for biostatistics. In the future, the data will be electronically transmitted. There is an "ALD" node on the ^SIDTRAN global which is not defined in the data dictionary. This node tracks the tape number for the last 'D'-type tape transmitted and is maintained by the routine, ^SIDRDR.
