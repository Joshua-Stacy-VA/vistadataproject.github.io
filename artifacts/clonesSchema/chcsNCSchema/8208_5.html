description
The entries in this file point to transactions (prescriptions, refills, stock issues and inpatient orders) for which workload could not be counted because the MEPRS code is inactive or invalid, or because the combination of MEPRS/DMIS ID/Division is not valid. Users may correct the MEPRS code using the Pharmacy option Edit Workload Data, which updates the original transaction and adds the workload to the Pharmacy MEPRS file. Entries are purged after two years.
