description
This file contains the site-definable recipients of the Blood Type Bulletin (sent by routine LRBBABO when there is a discrepancy in the patient's blood type. The .01 field is a variable pointer. It points to 3 files: User, Mail Group, and Device. The .02 field and the cross reference on that field exist solely for the purpose of navigating from the Lab Host Platform Parameters file back to this file.
