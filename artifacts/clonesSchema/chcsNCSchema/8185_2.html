description
The WAM STARS/FL Master Data Elements File contains Cost Account Code (CAC)/ Job Order Number (JON) data set requirements that are to be used by the CHCS-WAM Subsystem. This information defines the workload data that is to be collected under the WAM option and transmitted to the STARS/FL system. The information is organized for an entity that hold the operating budget for a group of organizational units. This entity is identified by the OB-UIC, its organizational entities are identified by the WJON-SN and FCC, the work job order number serial number. The file is a user modifiable file (Class 2). Initial data and subsequent data updates will automatically be stored in the file based on input files received from STARS/FL system by DA software. Information stored in this file is to be maintained for 12 months after its expiration data so that historical reports can be generated. CAC/JON data definitions and format are defined by the DoD in the WAM Core File. That information and changes to that information will be provided by the government.
