description
This file will allow each site to keep a "log" of the various conversion routines which the site has already installed. This is to prevent the possibility of overwriting more valid data previously installed. (Software Release practices sometimes make this possible, particularly with Special Releases, e.g., DA's quarterly Zip Code updates). Conversion routines will (at installation time) check the site's Conversion Tracking file to see if that site does indeed need that update. This check will also help prevent the unnecessary downtime of re-installing data already in place.
