description
This file contains, for each radiology location entry, parameters that the radiology module uses during various stages of exam and report processing and inquiring. The parameter switches allow the 'customizing' of the module for each location by the radiology coordinator. Within a radiology division there may be a number of physical locations where a imaging procedure can be performed. The module is designed to handle multiple locations within a division. However, each location can only be associated with one division.
