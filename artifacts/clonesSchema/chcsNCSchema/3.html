description
This file is used to control the various users of the system. Any user with an access and verify code can use the computer. It is also used for assigning mailman mail boxes, controlling user privileges, security keys, termination dates, and user specific control of the conversation. It is controlled primarily with the ADD USER and EDIT user options.
