description
This file contains information about which of the non-narcotic items in the Inventory Record file are stocked by locations within this file. The information is basically static after the initial load of items which are routinely furnished to these locations as either Bulk or Clinic stock. After the initial load of data this file is used to construct a display-man screen of candidates which the user chooses from to create new issues of Bulk/Clinic stock. The display will build by assending numerical order according to the field "Display Order". If this field does not have value for all items under the location selected then the list will build by display order first and the remaining items will be displayed in alphabetic sequence. If there are no items with a display order the list will display in alphabetic sequence.
