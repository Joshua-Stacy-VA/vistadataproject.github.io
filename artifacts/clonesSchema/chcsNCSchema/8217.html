name
PDTS PROFILE COLLECTION FILE
number
8217
location
^PSM(8217,
description
This file is used to manage the activites associated with PDTS Patient Profile. It saves the users selection criteria as well as when the request was done. When a response is returned from the PDTS this file holds the results of that response. The entries in this file are purged based on the rules in the option PSO OVERNITE MAINTENANCE.
