name
DEERS PERFORMANCE DATA
number
8907.8
location
^XDW(8907.8,
description
Detailed performance data is written to this file for each DEERS transaction if the Performance Statistics field (.1) in the DEERS Site Parameters file (8907) is set to "yes". The intent of this file is for diagnostic purposes only. The DEERS Performance Data file also contains detailed data for the logging into and logging out from external DEERS process. Those fields contain login or logoff in the name.
