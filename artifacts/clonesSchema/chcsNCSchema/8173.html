description
The CMAC Provider Class file is a CLASS 1 file. CMAC Provider Class is the pricing cost based on the class of provider: Class 01 is the physician class, pertaining to MD and DO's (including clinical lab procedures that are not MD or DO related). Class 02 is the psychologist class, pertaining to PHD's and Psychologists. Class 03 is other mental health providers. Class 04 is extra medical provider (non-mental health only).
