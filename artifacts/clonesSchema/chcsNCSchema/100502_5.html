description
This file contains pointers to all of the E&M (Evaluation and Management) codes in the CPT/HCPCS file. It gives the clinics the ability to identify the E&M codes that their providers use most often and to create custom display text for each code. When providers are entering ADM encounter data, they can type one question mark for a list of commonly used (clinic defined) E&M codes or two question marks for a list of all available E&M codes.
