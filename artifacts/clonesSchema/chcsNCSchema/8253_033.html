.01hic30;1FREE TEXTTThe Hierarchical Ingredient Code Specific Therapeutic Class (HIC3) comprises the first three bytes of the Hierarchical Ingredient Code. Each ingredient has an associated specific therapeutic class, related to either the HICTBL (8250.1) or optionally to the HICSEQNOHIC (8250), but each product entity (GCNSEQNOTBL-8251) has an associated specific therapeutic class. This therapeutic class coding scheme (HIC3) is the most specific offered by First DataBank and is intended for users who need a very definitive therapeutic classification system. This data element is often used in FDB's Duplicate Therapy Checking module and FDB's Hierarchical Ingredient Code List (HICL). Contact FDB's customer service department for a copy of the specifications. For a complete listing of all of the HIC3 descriptions available check file HIC1TBL HIC2TBL HIC3TBL (8250.3). This file can be access using the HIC3 POINTER.
