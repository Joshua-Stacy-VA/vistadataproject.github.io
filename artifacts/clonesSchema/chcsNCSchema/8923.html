description
This file contains an entry for each logical record type which has been identified as eligible for archive/purge but did not meet the criteria so was not archived/purged. When the problem has been resolved, the record will be archived/purged and the entry in this file will be deleted. The data in this file will be generated by the system during an archive or purge run.

This field indicates the current resolution status of the Exception in question. Possible values: N = 'normal' exception. Exception is still to be resolved and will appear on all Exception List reports. I = 'ignore exception' This specific exception will be ignored. If there are no other exceptions regarding the same record, this record will be archived and purged in the next archive run. K = 'keep in production database' This specific exception will no longer be reported. The record in question, however, will remain in the production database and will not be archived or purged. A = 'archived' The record that is associated with this exception has been archived. This exception will no longer be reported in exception list reports.
