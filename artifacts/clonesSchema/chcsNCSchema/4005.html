name
INTERFACE DESTINATION
number
4005
location
^INRHD(
description
This file holds the information necessary to route a transaction to its appropriate location. For outgoing transaction either a Transceiver Routine or a Mail Recipient is defined. For incoming transactions a Transaction Type is defined.
