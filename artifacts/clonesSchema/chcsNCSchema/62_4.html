.08allow immediate certif.0;8BOOLEAN0:NO
1:YES
Enter yes into this field if the system should prompt for certification of the results during TAR. The user who is signed onto TAR must still review the results and enter a "Yes" to certify. If a timeout occurs and the user does not certify the result then the results are filed and the next accession appears. Immediate certification is useful for those results that are unlikely to be changed or added to after they are reviewed. In addition, certification in TAR may be useful for instruments that transfer results slowly enough so that the lab user can review the results and certify before the next set of results are transmitted. Instruments that transmit the results in a batch or delayed after the instrument has been running for awhile will probably not be best suited for immediate certificiation.

This field allows the users to match the instrument ID to the accession number without having to run the instrument by work document. This function will be useful for older instruments that do not have the capability to define accession numbers on the instrument. For instruments that are not pre-programmed, this field allows users to match the instrument ID to the accession number without having to run the instrument by work document.

5.01communications mode5;1SET OF CODES0:UNIDIRECTIONAL
1:BIDIRECTIONAL
2:BIDIRECTIONAL QUERY
3:AUTOMATIC DOWNLOAD
COMMUNICATIONS MODE This field determines if the instrument is to be run in unidirectional, bidirectional, or bidirectional query mode. Unidirectional mode indicates that the instrument can only transmit results to CHCS. Bidirectional mode indicates that CHCS can transmit a batch of test order information to the instrument and the instrument transmits results to CHCS. Bidirectional query mode indicates that CHCS can only transmit test order information when queried by the instrument and the instrument transmits results to CHCS. Automatic Download mode indicates that CHCS will automatically download a lab order message to the instrument upon completion of the accessioning process. When this selection is made, an Accession Area must be entered in the Auto Instrument file.
