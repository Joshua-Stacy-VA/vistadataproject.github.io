.01formal first name(+)0;1FREE TEXTBThis field represents the actual (top-level) name which is used as the base for which name variations can begin. For example, the formal first name of JOHN is the base for names such as JOHNNIE. The FORMAL FIRST NAME field will be used in duplicate patient lookups to identify other variations of the same name using a table with formal first name as the top-level reference point. If a user enters JOHN SMITH, the system will identify other names in the database which have the same formal first name such as SHAWN SMITH (SHAWN is a variation of the formal first name JOHN).
