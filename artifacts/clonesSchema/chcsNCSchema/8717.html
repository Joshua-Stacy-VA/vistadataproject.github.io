description
This file contains the CPT workload data accumulated over a period of 25 months. It is used for generating workload reports based on various sort criteria. The following cross-reference is created when the 1st test linked to an order task is certified. It is used to drive the nightly Itemized Billing postings to the Itemized Billing Suspense file. Purged when the entry is posted to the Itemized Billing Suspense file. ^LRO(8717,"ACPTO",LRORT)="" Where: LRORT = order task ien ^LRO(8717,"ACPTO",LRORT,INDEX) = "" where: INDEX = pointer to ^LRO(8717,n,0) where CPT Workload Data specifics are stored The following cross references will be used for report generation: ^LRO(8717,"ADSC",LRSWE,LRLBSEC,LRFDT,LRCPT,INDEX)="" ^LRO(8717,"ATPC",LRFDT,LRDIV,LRMPRD,LRPTN,LRCPT,INDEX)="" The "ADSC" cross reference is used for the Detail, Summary and Comparison reports. The "ATPC" cross reference is used for the TPC (Third Party Collection) report. LRFDT = file date/time LRDIV = performing division LRLBSEC = ien lab section LRCPT = ien CPT code file LRPTN = ien Patient file INDEX = ien CPT workload data file LRMPR = ien MEPRS file LRACP = ien accession file LRPTYPE = patient type LRPRI = processing priority LRMOD = CPT modifier LRMPRD = MEPRS code displayable name LRSWE = performing lab work element LRCOUNT = the number of repeats entered LRCNT = incremented index for this cross reference LRPATC = patient category of patient at time of workload capture ***The AACP xref may no longer be used - further research is needed *** AACP Cross reference - Non Fileman cross reference set in AMT^LRUTCPT4 to indicate an amt code was set for the accession. ^LRO(8717,"AACP",LRACP,LRTST)=# of AMT's^INDEX AACN Cross Reference - Non Fileman cross reference set to indicate that workload has been collected for Anatomic Pathology and microbiology accessions. It is killed when the test is certified. ^LRO(8717,"AACN",LRACP,LRTST)="" LRACP = ien to file #66 LRTST = ien to file #60 ***The AACPTA xref may no longer be used - further research is needed *** ACPTA Cross Reference - Non Fileman cross reference set to capture workload by accession before running LCP option. An exception will prevent workload from being captured and may be viewed via the Laboratory Workload Exception Report (WEX) option.
