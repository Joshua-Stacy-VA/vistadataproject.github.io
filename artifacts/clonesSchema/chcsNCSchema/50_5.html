description
This is the file of drug classifications that are used to 'group' drugs in a related class. Unfortunately, it was named 'FORMULARY' which has often led to confusion. It is not unique to each center, but should be standard throughout the VA system. Each DRUG file entry points to one of these classfications to define the 'class' it belongs in. This information is used to warn when a new Rx drug is in the same class as an existing Rx drug for a given patient.
