description
This file contains the statistics for HIV testing. It is updated automatically by the system when an HIV test is certified. Two cross references on this file are used to accumulate monthly and yearly totals. One cross reference is for 'Force Testing' and the other is for 'Clinical' (non-force testing). The monthly counter node will have a date format of 2941200 (Dec. 1994). The yearly counter node will have a date format of 2940000 (1994).
