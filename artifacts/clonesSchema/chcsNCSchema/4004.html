name
BACKGROUND PROCESS CONTROL
number
4004
location
^INTHPC(
description
This file contains the background processes that the GIS can run. This is a control file that is used in the operation of the GIS. The FORMAT CONTROLLER and OUTPUT CONTROLLER are setup at system initialization and cannot be modified. All other background processes are defined by the developers of specific interfaces as needed. The most common developer defined background process is the receiver. This is typically used to receive data from a remote system.
