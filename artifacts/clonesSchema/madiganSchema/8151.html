description
This file is a standard DOD (Group 1) file which is sent fully populated to the sites, with annual updates provided. It contains both CURRENT PROCEDURAL TERMINOLOGY (CPT) codes and the Health Care Administration's Level II Procedure Coding System (HCPCS) codes.
