description
This file contains the Health Insurance Carriers (HIC) which may be billed for healthcare services provided by the MTF. The file is organized by "Carrier Name." The minimum data set for each carrier record includes the "Carrier Name", "Carrier POC Name", and "Carrier POC Phone Number", "Carrier POC E-Mail Address"; and, at least, one "Coverage Type Code" and "Payer Type Code" combination The scope of coverage and type of payer is defined in the Coverage Type sub-file. There must be, at least one Coverage Type per policy and the following fields are required for each Coverage Type: "Payer Type", "Mailing Address Line 2, City, State or Country, and Telephone Number."
