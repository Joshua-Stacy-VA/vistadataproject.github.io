Remember the Review New Results option will electronically notify the HCP of all results and designate priority results.

.19snomed autoencoding(+)0;19SET OF CODESA:ACTIVE FOR ANATOMIC PATHOLOGY
M:ACTIVE FOR MICROBIOLOGY
B:ACTIVE FOR BOTH
I:INACTIVE
This field is used to indicate if SNOMED autoencoding should take place during result entry for Anatomic Pathology and/or Microbiology tests. If the field is set to ACTIVE FOR ANATOMIC PATHOLOGY or ACTIVE FOR BOTH, SNOMED CT Codes will be automatically assigned to text in the final diagnosis field for AP tests. If the field is set to ACTIVE FOR MICROBIOLOGY or ACTIVE FOR BOTH, SNOMED CT Codes will be automatically assigned to stain fields and result fields for Micro tests. If the field is set to INACTIVE, SNOMED CT Codes will not be automatically assigned to text fields. For AP tests, the user will be able to enter SNOMED CT Codes manually.
