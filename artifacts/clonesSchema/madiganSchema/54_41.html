description
This file holds transaction codes which are pointed to by files 54.4, 54.5 and 54.3. The numbering schema reflects a grouping by whether or not the transaction affects inventory and by what type of transaction generated the code (issues, prescriptions, adjustments and supply transactions). It should NOT be edited, either added to or deleted from.
