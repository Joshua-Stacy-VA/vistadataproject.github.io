description
The allergy definition file (8254.011) is a companion to the allergy selection file (8254.01). While allergy selection file contains all the allergy classes, brand names, generic names, and ingredients for the user to select, allergy definition file contains the actual definitions used by the system to do clinical screening. One to many entries in the allergy selection file point to one entry in the allergy definition file.
