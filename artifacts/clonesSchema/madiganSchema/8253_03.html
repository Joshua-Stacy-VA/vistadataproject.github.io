description
The DAC MONO INDEX CITE file (8253.03) consist of seven files: 1) The Main file relates HICs, DACs, HIC3s with Monographs and Citations 2) Subfile 1 8253.031 is a list of Hierarchical Ingredient Codes (HICS) 3) Subfile 2 8253.032 is a list of Drug Allergy Clases (DACTBL) 4) Subfile 3 8253.033 is a list of Therapeutic Class Codes (HIC3S) 5) Subfile 4 8253.034 is the general cautions monograph (CAUTIONS) 6) Subfile 5 8253.035 contain other Cautions monographs (SECTIONS) 7) Subfile 6 8253.036 contain citations referenced by the monographs (CITATIONS) A description of each file follows: 1. The Main file using a USAN Drug Title relates HICs, DACs, HIC3s with Caution Monographs and citations from the AHFS Drug information book. The Internal Entry Number (D0) is the AHFS Mono Code (AHFSMONO) associated with each monograph. (up to 6 digits) with leading zeros removed. 2. Subfile 1 8253.031 is a list of Hierarchical Ingredient Codes (HICS) The Internal Entry Number (D1) is the Pointer to the HICTBL (8250.1) Ingredient Code Number Sequence Table (ICNSETBL) (up to 4 digits) with leading zeros removed. 3. Subfile 2 8253.032 is a list of Drug Allergy Clases (DACTBL) The Internal Entry Number (D1) is the Drug Allergy Code Description Table (DACTBL) (8250.2) up to 2 digits) with leading zeros removed. If DACTBL=0 IGNORE. 4. Subfile 3 8253.033 is a list of Therapeutic Class Codes (HIC3S) The Internal Entry Number (D1) is the numeric ascii equivalent of the HIC3 entry. 5. Subfile 4 8253.034 is the general cautions monograph (CAUTIONS) This file contains general caution monograph from the CAUTIONS section of the AHFS Drug information book. The Internal Entry Number (D1) is a count of each Caution text as read. 6. Subfile 5 8253.035 contain other Cautions monographs (SECTIONS) This file contains specific caution monographs from the CAUTIONS section of the AHFS Drug information book. The Internal Entry Number (D1) is the AHFS full text monograph section-ID (AHFSSCTID) up to 2 digits, with leading zeros removed. The Internal Entry Number (D2) is a count of each section text as read. 7. Subfile 6 8253.036 contain citations referenced bythe monographs (CITATIONS) The Internal Entry Number (D1) is the AHFS full text monograph citation code (AHFSCITE) up to 3 digits with leading zeros removed. The Internal Entry Number (D2) is a count of each citation text as read. The Master File Primary Key (Unique ID) for this file: IEN\(.01)Name\99FDC
