description
An entry must be in this file for every non-CHCS performing laboratory to which specimens will be sent for testing. For each test sent to another laboratory, the performing laboratory test name must be in the file entry for that laboratory. In addition, the order code must be entered for those tests with a type of "INPUT" and "BOTH". The order code is not necessary for those tests with a type of "OUTPUT" or "NEITHER". Authorized users may add new entries or edit existing entries for laboratories, test names and order codes.
