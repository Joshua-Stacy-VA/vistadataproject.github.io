description
This file is a standard DOD (Group 1) file which is sent fully populated to the sites, with annual updates provided. The CMAC Rate file contains geographic-specific rate tables for billable services, based on CPT/HCPCS codes. The CMAC Rate file contains three types of table entries: 1. CMAC Rates - includes procedures and services that do not have component rates for technical and professional services. CMAC Rates apply to E&M, CPT, and HCPCS Level II codes with G, P, Q and W in the first position. 2. CMAC Component Rates - includes services that have both technical and professional component rates based on CPT/ HCPCS codes. 3. Non-CMAC Rates - includes services that are priced using the CHAMPUS allowable charge methodology that is computed on a statewide basis, based on pricing established by TRICARE Providers. The number of Non-CMAC rates varies by geographic locality.
