description
The PEMMONO PEMEMONO file consist of three files: 1) The Main file contains the Patient Education Monograph Code (PEMONO) with leading zeros removed. 2) Subfile 1 8253.0211 is the PEMMONO. 3) Subfile 2 8253.0212 is the PEMEMONO. A description of each file follows: The 4-byte PEMONO monograph code provide a more detailed text information that can be delivered to the patient on standard-sized 8 1/2" x 11" paper. The Internal entry number (D0) is the Patient Education Monograph Code (PEMONO) with leading zeros removed. PEMMONO - RPEMMO0 The Patient Education Short Form, or Standard Monograph Table uses the 4-byte PEMONO monograph key from the Patient Education Master Table to provide descriptive information that can be printed off on label-sized paper to be given to the patient. The information covers topics such as 'Why am I taking the drug?', 'Are there side effects?', etc. The Internal entry number (D1) is the Patient Education Text Seq.No.(Short) (PEMONOSN) with leading zeros removed. PEMEMONO - RPEMMOE0 The Patient Education Extended Monograph Table uses the 4-byte PEMONO monograph code from the Patient Education Master table to provide more detailed text information that can be delivered to the patient on standard-sized 8 1/2" x 11" paper. The Internal entry number (D1) is the Patient Education Text Seq.No.(Long) (PEMONOESN) with leading zeros removed.
