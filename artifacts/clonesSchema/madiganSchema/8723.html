description
This file contains the allowable fields per functionality that will be used in label printing. It includes the common name of the field and its FileMan Subfile number and field number, as defined in the Data Dictionary. It also includes the local variable name for the field within a given functionality, as well as sample data filled to the maximum limits of the field and a description of the field's location, meaning, and use. Fields may not be word-processing data types.
