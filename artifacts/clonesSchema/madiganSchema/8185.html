description
The WAM Core File contains Data Set (DS) data and related Performing Functional Cost Code (FCC), Requesting Functional Cost Code (FCC), and Cost Account Code (CAC) information for each service. The data is used to validate requests for workload information that is to be collected through the CHCS WAM sub-system. Additional business rules that are to be applied when generating data for the CHCS-WAM sub-system, are also stored in the file. The data is provided by the customer (TMA), on an annual or as needed basis. This is a class 1 file that is not modifiable by the user or site. The file will be updated via the CHCS special release process.
