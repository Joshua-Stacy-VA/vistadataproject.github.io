description
This file contains LOINC codes, which provide a set of universal names and identification codes for identifying laboratory and clinical test results. The purpose is to facilitate the exchange and pooling of clinical results for patient care, outcome management, statistical analysis, and research. The universal codes and names can be used to identify laboratory results to a level of detail that will fully specify the separately reported observations on a clinical laboratory report. A LOINC record requires a unique LOINC code, component/analyte name, property type, scale type, time aspect, system/sample, and where relevant, the method type. Fields #2-7 contain the six parts of the name. The fully specified name for a given LOINC code would be constructed by printing out the contents of these fields (#2-7), inserting a colon (:) between the contents of each of these fields.
