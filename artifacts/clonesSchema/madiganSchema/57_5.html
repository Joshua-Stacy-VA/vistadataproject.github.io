This file is used by the inpatient unit dose pharmacy to designate which wards are covered by a single unit dose cart. A Ward Group represents one cart.

This field defines which Ward Group for a Unit Dose site will contain the MED orders that are forwarded to it from another Unit Dose site. One Ward Group in a Unit Dose site must be defined as the group that contains the forwarded orders. If one is not defined, orders will not be allowed to be forwarded to the Unit Dose site. You must assign the Ward Group to a Unit Dose site before this field can be set to YES.
