description
The Etiology file describes the underlying cause of a disease process and an individual's method of acquiring the disease. Entries may be added or modified, but none of the supplied entries should be deleted. The file describes specific information about each etiologic agent, such as SNOMED CT code, gram stain and identifier. This file is mainly used in microbiology.
