description
This file is maintained at the Consolidated Mail Outpatient Pharmacy host facility and contains patient prescription orders received in data transmissions from the participating medical centers. Data from this file is downloaded to the automated system for dispensing. Once all data for a transmission is completed by the automated system, the file is purged. Data in this file is maintained and purged by the Consolidated Mail Outpatient Pharmacy software and must not be edited using VA FileMan.
