description
This file contains a list of encounters which were selected for review based on a comparison of the ICD diagnosis codes from the encounter and the VBA rated disabilities (VA) codes from the patient. Entries will be edited as necessary per the review criteria established for determining if an encounter is service-connected (SC) or non-service connected (NSC).
