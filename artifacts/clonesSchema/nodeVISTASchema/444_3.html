description
This file contains counters for getting the next available sequence number. There is a separate counter for each combination of Station # and Fiscal Year. This sequence number is padded on the left with zeros to form the sequence number portion (4th piece) of the RFQ number. (i.e. 688-96-RFQ-00014).
