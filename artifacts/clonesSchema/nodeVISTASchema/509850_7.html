description
The A&SP CLINIC WORKLOAD file (#509850.7) contains each month's capitation statistics. Data in this file are used to compile statistics which includes a display of total and unique visits by zip code, procedure, and disease code for any specified month. Data in this file SHOULD NOT be altered through the use of VA FileMan; input should take place ONLY through the QUASAR menu options. Per VHA Directive 10-93-142, this file definition SHOULD NOT be modified.
