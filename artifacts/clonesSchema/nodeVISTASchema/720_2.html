This file contains information that is used to screen medical procedures based on a unique location, DSS unit, category, and procedure. This file should NOT be modified directly using VA FileMan. **NOTE** This file is being exported with Event Capture Version 2.0 for use in converting existing data, at the pilot sites, to current data formats. This file will be removed in a future release of the Event Capture software. ** Per patch EC*2*5, this file has been starred for deletion. **
