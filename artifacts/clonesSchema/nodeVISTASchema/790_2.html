description
This file stores and defines the type of procedures that may be associated with patients within this application. The procedure type definitions includes the name, abbreviation, synonyms, last accession #, CPT and ICD codes, and colposcopy related information. There is no option to edit this file although new entries may be added through FileMan.

This field turns on and off the display of the clinical findings/tissue pathology screen that is used to document clinical exam and pathology information related to specific cervical procedures such as colposcopy, cone biopsy, laser cone and LEEP's.
