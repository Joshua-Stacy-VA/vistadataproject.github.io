description
This file contains vocabulary subsets of the main Expressions file (#757.01). Each sub-set is defined in file #757.2, and may contain [pointers to] terms which are exclusive to a discipline (i.e., Nursing), a device (i.e., Grid Pad), or an application (i.e., Order Checking, Encounter Form, etc).
