description
This file contains Pharmacy Division specific preferences for the ePharmacy Medication Profile. For divisions without specific preferences in this file, the Outpatient Pharmacy application uses the 'Factory Default', which has the following default values: EXP/CANCEL CUTOFF: 120 DAYS SORT BY: DRUG NAME SORT ORDER: ASCENDING DISPLAY SIG: OFF GROUP BY STATUS: OFF DISPLAY ORDER COUNT: ON

5group by status0;6SET OF CODES0:OFF
1:ON
This is the default value (YES/NO) that determine whether the list of prescription should be group by status (Active, Discontinued, Hold, etc.) or not in the ePharmacy Medication Profile for the users logged into the Outpatient Pharmacy application under this Pharmacy Division. When this parameter is set to 'NO', all the regular prescriptions (excludes: Non-VA Meds, Pending Orders and REFILL TOO SOON/DUR prescriptions) are listed under the 'CURRENT ORDERS' group. Note: This parameter can be overridden by user specific preferences.
