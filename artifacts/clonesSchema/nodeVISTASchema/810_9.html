description
This file is used to group a list of locations and give the group a name. The locations can be Hospital Locations and/or Clinic Stops (which resolve to a list of Hospital Locations). A Location List finding in Clinical Reminders is based on an entry from this file. A Location List finding will be true when a patient has an entry in the Visit file for at least one of the locations in the Location List.
