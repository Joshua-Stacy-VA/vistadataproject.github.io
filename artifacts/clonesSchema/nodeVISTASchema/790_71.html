description
This file contains information aggregated from data stored in the WV Procedure (790.1), WV Notification (790.4), and WV Refusals (790.3) files. Data storage is dependent upon running the Snapshot of the Program Today option and answering YES to the question, "Should today's Snapshot be stored for later retrieval and comparisons?" These records provide a historical perspective of the Woman Veterans Health Program at your facility and may be printed through the Retrieve/Print Earlier Snapshot option.
