description
This file keeps track of all editable parameters for Capacity Planning packages. It also stores Start, Stop and Delta times for background jobs, as well as CPU Node data. There is only one entry allowed in this file, and is a pointer to the Institution file taken from the first piece of the $$SITE^VASITE extrinsic function.
