description
This file holds VistALink Listener configurations. A configuration may contain multiple Listener ports, and each Listener port may be configured to start upon system startup. Note: This only applies to Cache NT systems. VMS/DSM systems need to use the TCP/IP (UCX) utility to have Listeners automatically started on reboot. This file is pointed to by the Foundations Site Parameters file (#18.01) and is used when obtaining the default Listener configuration for a specified BOX-VOLUME pair.
