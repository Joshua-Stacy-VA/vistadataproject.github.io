.01hl7 receiving application0;1POINTER771BPointer to the HL7 APPLICATIONS file (#771). If the Radiology HL7 Subscriber protocols (file #101) have been setup to use ROUTING LOGIC, then HL7 RECEIVING APPLICATIONS should be defined. This DIVISION will then send HL7 messages to the HL7 RECEIVING APPLICATIONS defined only. If no HL7 RECEIVING APPLICATIONS are set when ROUTING LOGIC is being used, no HL7 messages will be created for this DIVISION. If ROUTING LOGIC is not being used and no HL7 RECEIVING APPLICATIONS are set, then messages will be sent to all Radiology Subscriber protocols, as normal. If in doubt, do not put entries in this field.
