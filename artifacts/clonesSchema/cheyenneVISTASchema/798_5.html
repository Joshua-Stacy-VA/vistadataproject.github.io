description
This file stores all the ICD diagnostic codes used to identify patients for a given registry during the Registry Update process. The B cross reference for the ICD code is used in the EXPRESSION field of the ROR SELECTION RULES file (#798.2). The file design allows CCR to support an unlimited number of codes selected from the ICD DIAGNOSIS file (#80).
