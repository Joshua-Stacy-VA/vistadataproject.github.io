description
This file contains the types of coverages that an insurance company is generally associated with. If an insurer is identified with more than one type of coverage then it should be identified as HEALTH INSURANCE as this encompases all. Sites should not add, edit or delete entries from this file. Per VHA Directive 10-93-142, this file definition should not be modified.
