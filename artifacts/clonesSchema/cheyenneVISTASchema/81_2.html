description
This file contains the Copyright Notice which must be displayed to users of applications which utilize the Current Procedural Terminology (CPT) codes. The American Medical Association and the Department of Veterans Affairs have reached an agreement as to the specific wording and requirements of the Copyright Notice. If an entry needs to be added, modified or deleted a patch will be issued instructing the site how to make the change. Otherwise, this table file should not be edited in anyway by the site. Per VHA Directive 10-93-142, this file definition should not be modified.
