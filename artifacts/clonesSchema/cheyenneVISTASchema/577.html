description
This file describes VDEF events. Mostly, these events match the list of events in the HL7 2.4 standard, but there are some cases where there may be multiple entries in this file for a single HL7 standard event. ENTRIES IN THIS FILE SHOULD NOT BE EDITED IN PRODUCTION ACCOUNTS.
