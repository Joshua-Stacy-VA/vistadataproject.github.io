description
This file contains the name of the various constant data variable names used in the treatment plan PC software. For each data name there is an associated long integer value that is relevant to the PC software. Every data name must have a constant data type (pointer to file 228.31). There is a KEY on this file of the data name and data type. ********** ATTENTION ATTENTION ********** This data is not to be edited. The dental software is expecting specific entries to have a specific internal entry number. Any updates or changes to this file will be accomplished by KIDS distributions from the Dental Package.
