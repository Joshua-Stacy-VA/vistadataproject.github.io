description
This file defines specific scheduling report groups. Each report definition in the SCHEDULING REPORT DEFINITION (#404.92) is assoicated with a report group. The report group allows the easy display of related report definitions to the user for selection. These groups are pre-determined and must not be modified unless explicitly instructed as part of a patch or module upgrade.
