description
Information that is transmitted to the FMS System are sent via records called 'documents'. Each FMS document has a field called 'type' that describe the information or transaction being transmitted. When the Accounts Receivable system transmits documents, certain parameters are stored in this field that help define the layout of the document. Per VHA Directive 10-93-142, this file definition should not be modified.
