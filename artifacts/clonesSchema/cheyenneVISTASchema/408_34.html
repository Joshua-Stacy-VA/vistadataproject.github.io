This file contains a list of the various sources from which an income test may originate. This would include the VAMC if the means test was administered by a user or IVM if the income was verified by the Income Verification Match Center. This file, and the data contained within the file, is strictly maintained and should not be altered at the site level. Editing of this file or the data contained within may result in data corruption or may cause software to malfunction.
