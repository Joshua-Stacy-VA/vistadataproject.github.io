description
This file holds batch payments information. Payments no longer are posted directly to patient bills, but rather, entered into this file. Once the entries in this file are reconciled with cash on-hand and verified to be correct, they are then 'posted' to the patient bills during a background job. NOTICE: This file SHOULD NOT be edited directly via Fileman. Information in this file should only be modified via the AR package options.
