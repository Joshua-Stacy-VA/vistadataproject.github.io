description
File contains the results of the Diagnosis Option in the Mental Health System. "AC" Xref - DFN, Inverse Date, Dx, Status, IFN. Used to list dx as Active, Inactive, etc. "AE" Xref - Type (DSM or ICD9), DFN, System Dte, Dx, DFN. Used for reference. "AF" Xref - DFN, Inverse Date, DX, IFN. Used for historical listing. "AG" Xref - Type (DSM or ICD9), DFN, Dx, IFN. Used for listing duplicate diagnoses.
