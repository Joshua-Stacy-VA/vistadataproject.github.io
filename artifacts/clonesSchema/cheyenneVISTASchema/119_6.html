description
This file is a list of wards in the hospital and associated room-beds and contains various dietetic specific information such as the assigned clinician, bulk nourishments for the ward, and which Service Point, Communication Office, and Supplemental Feeding Site is assigned responsibility for the ward. This file will also be used to store outpatient locations necessary for the outpatient meals enhancement.
