description
The ESP ACTIVITY CODE file contains a list of work activities performed by VA Police Officers during the course of a routine daily shift. It contains the name of the work activity, a code number assigned to the work activity and the number of minutes the work activity should be performed. Local sub-activity codes may be added. Refer to the User's Manual for more information.
