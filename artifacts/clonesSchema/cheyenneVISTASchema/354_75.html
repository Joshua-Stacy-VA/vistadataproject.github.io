description
DO NOT delete entries in this file. DO NOT edit data in this file with VA File Manager. This file comes populated with data. The data in this file should not be edited, added, or deleted locally. The information stored here is the cap amounts for outpatient medication copayment. Once a patient has reached his cap, billing will stop for the remainder of the period indicated. Per VHA Directive 10-93-142, this file definition should not be modified.
