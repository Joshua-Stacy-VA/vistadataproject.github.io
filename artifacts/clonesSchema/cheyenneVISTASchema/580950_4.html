description
This file should never have data in it immediately following a transmission. The file contains only one field, which is a variable pointer to files 580950.2 (Transmital) and 580950.3 (Status Update). Records which have been flagged for retransmission, either by the system or user, are stored here for the transmission process to pick up (^AJK1UBT). New transmission strings ARE NOT sent, rather the originally transmitted string is resent to the vendor. Routines, options, etc., associated with this process are in the AJK1UB namespace.
