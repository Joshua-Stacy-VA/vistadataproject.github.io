description
The type of group defines what 'category' a group belongs to and the parameters that are associated with the specific type of group. A type of group, for example, is 'Billing Agency' or 'District Counsel'. Per VHA Directive 10-93-142, this file definition should not be modified.

1:YES
If an AR Group can be defined multiple times, then this field would be set to Yes. An example of a Group that would be defined multiple times is the Billing Agency since a site may have more than one billing agency. However, other groups such as the place the site sends Bank Deposits should not be defined more than once.
