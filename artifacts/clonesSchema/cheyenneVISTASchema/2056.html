description
The VHS&RA ADP TRACKING SYSTEM file (2056) is used to list and describe the ADP EQUIPMENT inventory for each facility. A model number is selected from the ADP EQUIPMENT file (2056.31), and the associated data items (e.g. manufacturer, type of equipment, average cost, and VA System category) are taken from the ADP EQUIPMENT file (2056.31) and defined into the VHS&RA ADP TRACKING SYSTEM file (2056). Additional fields relevant to the facility which complete the description of the model number are: serial number, type of purchase, P.O.number, quantity, date received and type of purchase. For components, two additional fields are described: the component's system, and the cabinet serial number in which the component resides. Three more fields are for field use only: service indentifier, location and comments.
