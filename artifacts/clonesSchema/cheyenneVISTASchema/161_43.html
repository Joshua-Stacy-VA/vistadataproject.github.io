description
This file contains a list of contracts for the local site. New outpatient and civil hospital authorizations and payments can be linked to an active contract. The contract number will be included with associated payments that are transmitted to Central Fee. Note that contracts for community nursing home rates are stored in the FEE BASIS CNH CONTRACT file instead of this file.
