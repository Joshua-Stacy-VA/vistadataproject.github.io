description
This file has been added for use by the 2507 REQUEST File (396.3). It contains information about C&P appointments linked to 2507 Requests. This file should not be edited via FileMan. The information in this file is crucial to proper calculation of the Average Processing Time on the AMIE AMIS 290. NOTICE: This file is part of an implementation of a Nationally Controlled Procedure. Local modifications to this file are prohibited per VHA Directive 10-93-142
